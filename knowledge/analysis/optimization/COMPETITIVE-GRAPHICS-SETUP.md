# Настройка графики для киберспорта (VALORANT-style)

## [OK] Выполненные изменения

### 1. Отключён Motion Blur
- `r.DefaultFeature.MotionBlur=False` в RendererSettings
- `r.MotionBlurQuality=0` в ConsoleVariables
- `r.MotionBlur.Amount=0` и `r.MotionBlur.Max=0`

**Результат**: Изображение не смазывается при движении, как в VALORANT.

### 2. Оптимизирован TAA для чёткости
- `r.TemporalAA.Quality=0` - минимальное качество
- `r.TemporalAACurrentFrameWeight=0.04` - меньше размытия
- `r.TemporalAASamples=2` - меньше сэмплов
- `r.TemporalAASharpness=0.5` - средняя резкость

**Результат**: Меньше размытия от антиалиасинга, более чёткое изображение.

### 3. Добавлена резкость
- `r.Tonemapper.Sharpen=1.0` - высокая резкость
- `r.Upscale.Sharpen=1.0` - резкость при масштабировании

**Результат**: Детали более заметны, изображение чётче.

### 4. Отключён VSync для низкой задержки
- `r.VSync=0` - отключён
- `r.FinishCurrentFrame=0` - не ждать завершения кадра
- `r.OneFrameThreadLag=0` - без задержки

**Результат**: Минимальная задержка ввода, как в киберспорте.

### 5. Отключены эффекты постобработки
- `r.DepthOfFieldQuality=0` - глубина резкости
- `r.BloomQuality=0` - свечение
- `r.LensFlareQuality=0` - блики
- `r.EyeAdaptationQuality=0` - адаптация яркости
- `r.FastBlurThreshold=0` - размытие

**Результат**: Чистое изображение без визуальных помех.

### 6. Настроено разрешение
- `r.ScreenPercentage=100` - 100% разрешение
- `r.ViewDistanceScale=1.0` - полная дальность прорисовки
- `r.Upscale.Quality=1` - качественное масштабирование

**Результат**: Максимальная чёткость изображения.

## [GAME] Дополнительные рекомендации

### В настройках игры:
1. **Разрешение**: Нативное разрешение монитора (1920x1080, 2560x1440 и т.д.)
2. **Частота обновления**: Максимальная частота монитора (144Hz, 240Hz)
3. **Режим отображения**: Полноэкранный (не оконный)
4. **Качество текстур**: Высокое (для чёткости)
5. **Тени**: Средние/Низкие (для производительности)

### Для записи демо:
1. **FPS**: Стабильные 60+ FPS
2. **Разрешение записи**: 1920x1080 или выше
3. **Битрейт**: Высокий (50+ Mbps)
4. **Формат**: H.264 или H.265

## [SYMBOL] Консольные команды для тонкой настройки

В игре можно использовать консоль (`) для дополнительной настройки:

```
r.Tonemapper.Sharpen 1.5    ; Увеличить резкость (0.0-2.0)
r.TemporalAA.Quality 0      ; Отключить TAA полностью
r.ScreenPercentage 100      ; 100% разрешение
r.VSync 0                   ; Отключить VSync
stat fps                     ; Показать FPS
stat unit                    ; Показать производительность
```

## [WARNING] Важные замечания

1. **Производительность**: Отключение эффектов повышает FPS на 20-30%
2. **Визуальное качество**: Изображение будет более "плоским", но чётким
3. **Задержка ввода**: Отключение VSync уменьшает задержку на 10-20 мс
4. **Тестирование**: Проверьте на разных мониторах и разрешениях
