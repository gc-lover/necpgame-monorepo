# Настройки графики в стиле VALORANT

## [TARGET] Цель

Сделать изображение чётким, резким и без смазывания, как в VALORANT, для киберспортивного геймплея.

## [OK] Выполненные настройки

### 1. Отключён Motion Blur
```ini
r.DefaultFeature.MotionBlur=False
r.MotionBlurQuality=0
r.MotionBlur.Amount=0
r.MotionBlur.Max=0
```
**Причина**: Motion blur смазывает изображение при движении, что мешает точному прицеливанию.

### 2. Оптимизирован TAA (Temporal Anti-Aliasing)
```ini
r.TemporalAA.Quality=0
r.TemporalAACurrentFrameWeight=0.04
r.TemporalAASamples=2
r.TemporalAASharpness=0.5
```
**Причина**: TAA может размывать изображение. Уменьшена интенсивность для большей чёткости.

### 3. Добавлена резкость (Sharpening)
```ini
r.Tonemapper.Sharpen=1.0
r.Upscale.Sharpen=1.0
```
**Причина**: Увеличивает чёткость изображения, делает детали более заметными.

### 4. Отключён VSync
```ini
r.VSync=0
r.FinishCurrentFrame=0
r.OneFrameThreadLag=0
```
**Причина**: VSync добавляет задержку ввода. Отключён для минимальной задержки.

### 5. Отключены эффекты постобработки
```ini
r.DepthOfFieldQuality=0
r.BloomQuality=0
r.LensFlareQuality=0
r.EyeAdaptationQuality=0
```
**Причина**: Эти эффекты могут размывать изображение и отвлекать от геймплея.

### 6. Настроено разрешение
```ini
r.ScreenPercentage=100
r.ViewDistanceScale=1.0
```
**Причина**: 100% разрешение для максимальной чёткости.

## [GAME] Дополнительные рекомендации

### В настройках игры:
1. **Разрешение**: Используйте нативное разрешение монитора
2. **Частота кадров**: Установите максимальную частоту обновления
3. **Полноэкранный режим**: Используйте полноэкранный режим для минимальной задержки
4. **Качество текстур**: Высокое для чёткости
5. **Тени**: Средние/Низкие для производительности

### Для записи демо:
1. **FPS**: Стабильные 60+ FPS
2. **Разрешение записи**: 1920x1080 или выше
3. **Битрейт**: Высокий для качества
4. **Формат**: H.264 или H.265

## [WARNING] Важные замечания

1. **Производительность**: Отключение эффектов повышает FPS
2. **Визуальное качество**: Изображение будет более "плоским", но чётким
3. **Задержка ввода**: Отключение VSync уменьшает задержку
4. **Тестирование**: Проверьте на разных мониторах и разрешениях

## [SYMBOL] Дополнительные консольные команды

Для дальнейшей настройки в игре можно использовать:
```
r.Tonemapper.Sharpen 1.5  ; Увеличить резкость
r.TemporalAA.Quality 0    ; Отключить TAA полностью
r.ScreenPercentage 100   ; 100% разрешение
stat fps                 ; Показать FPS
```
