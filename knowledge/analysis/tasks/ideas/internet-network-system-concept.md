# Система сети (интернет)

## Лор и атмосфера

После падения Blackwall в 2023 году глобальная сеть пережила катастрофический коллапс. Множество данных было потеряно, корпорации потеряли контроль над своими сетями, а киберпространство стало опасным местом, наполненным остатками AI и вирусов.

К 2077 году сеть частично восстановилась, но осталась фрагментированной и нестабильной. NetWatch пытается контролировать сеть, но их власть ограничена. Корпорации создали собственные защищенные сети, а криминальные организации используют темные сети для своих операций.

Сеть стала местом, где информация - это валюта, а доступ - это власть. Каждое взаимодействие через сеть оставляет цифровой след, который может быть использован для слежки или манипуляций. Корпорации используют сеть для контроля и мониторинга, взломщики используют ее для получения доступа к закрытой информации, а обычные граждане полагаются на нее для повседневной жизни.

В мире 2077 года сеть - это одновременно инструмент свободы и контроля, место, где можно найти все, но где каждое действие отслеживается и записывается. Это место, где реальность и виртуальность смешиваются, создавая новый уровень существования.

## Основные концепции

### Определение сети

Сеть (интернет) является основой для всех нефизических взаимодействий между персонажами в игровом мире. Все коммуникации, кроме прямого диалога, происходят через устройства, подключенные к сети. Сеть представляет собой глобальную инфраструктуру, связывающую все устройства и системы в единое цифровое пространство.

### Характеристики сети

**Фрагментированность:**
- Сеть после падения Blackwall стала фрагментированной
- Различные регионы имеют разный уровень доступа
- Корпорации создали собственные сети
- Криминальные организации используют темные сети

**Нестабильность:**
- Сеть подвержена сбоям и перебоям
- Качество соединения варьируется в зависимости от региона
- Постоянная угроза атак и вирусов
- Необходимость защиты от сетевых угроз

**Контроль:**
- NetWatch пытается контролировать сеть, но власть ограничена
- Корпорации контролируют свои сети
- Криминальные организации используют сеть для незаконных операций
- Обычные граждане имеют ограниченный контроль

## Механики системы

### Подключение устройств к сети

**Типы подключения:**
- Проводное подключение (надежное, но ограниченное физическими кабелями)
- Беспроводное подключение (гибкое, но уязвимое для перехвата)
- Спутниковое подключение (глобальное, но дорогое и с задержками)
- Оптоволоконное подключение (быстрое, но требует инфраструктуры)

**Требования для подключения:**
- Устройство с сетевым интерфейсом и поддержкой сетевых протоколов
- Доступ к сетевой инфраструктуре (кабели, вышки, спутники)
- Оплата сетевых услуг для некоторых типов подключения
- Авторизация и идентификация устройства в сети
- Настройка сетевых параметров и конфигурации

**Качество соединения:**
- Скорость передачи данных влияет на скорость взаимодействий
- Задержка (latency) влияет на отзывчивость системы
- Стабильность соединения определяет надежность доступа
- Пропускная способность ограничивает объем передаваемых данных

### Взаимодействие через сеть

**Коммуникации:**
- Текстовые сообщения (чат, email, форумы) для обмена информацией
- Голосовая связь (VoIP) для голосового общения
- Видеосвязь (видеозвонки, стриминг) для визуального общения
- Файловый обмен для передачи данных и файлов
- Удаленные сессии и доступ для управления удаленными системами

**Социальные взаимодействия:**
- Социальные сети и форумы для общения и обмена информацией
- Групповые чаты и каналы для координации действий
- Публикация контента для распространения информации
- Реакции и взаимодействия с контентом для выражения мнений
- Создание сообществ и групп по интересам

**Торговля и экономика:**
- Онлайн-магазины и маркетплейсы для покупки и продажи товаров
- Банковские операции для финансовых транзакций
- Криптовалютные транзакции для анонимных платежей
- Контракты и соглашения для юридических операций
- Аукционы и торговые площадки для обмена ценными предметами

**Игровые механики:**
- Кооперативные задания для совместного выполнения задач
- PvP взаимодействия для соревновательных режимов
- Торговля между игроками для обмена предметами
- Групповые активности для совместных действий
- Системы репутации и рейтингов для оценки игроков

## Типы сетей

### Глобальная сеть (Интернет)
- Публичная сеть, доступная всем пользователям
- Фрагментированная и нестабильная после падения Blackwall
- Контролируется NetWatch частично, но власть ограничена
- Используется для большинства взаимодействий между пользователями
- Постоянная угроза атак и вирусов
- Различное качество доступа в зависимости от региона

### Корпоративные сети
- Защищенные сети крупных корпораций для защиты данных
- Высокий уровень безопасности с множественными уровнями защиты
- Ограниченный доступ только для авторизованных пользователей
- Интеграция с глобальной сетью через защищенные шлюзы
- Использование передовых технологий защиты
- Регулярное обновление защиты и мониторинг

### Криминальные сети
- Темные сети для незаконных операций и координации
- Анонимность и скрытность для защиты от правоохранительных органов
- Используются преступными организациями для различных целей
- Сложны для отслеживания из-за использования шифрования и анонимности
- Постоянная угроза взлома со стороны конкурентов
- Использование устаревших методов защиты

### Локальные сети (LAN)
- Изолированные сети для групп устройств
- Повышенная безопасность за счет изоляции от глобальной сети
- Ограниченный функционал без доступа к глобальной сети
- Возможность подключения к глобальной сети через шлюзы при необходимости
- Использование для защиты критически важных данных
- Управление устройствами в пределах локальной сети
