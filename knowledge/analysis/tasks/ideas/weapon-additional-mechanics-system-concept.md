# Дополнительные механики оружия: альтернативные режимы, управляемые снаряды, оружие для передвижения - Концепция

## Обзор

Система дополнительных механик оружия расширяет возможности оружия через альтернативные режимы стрельбы, управляемые снаряды и оружие для передвижения. В мире Cyberpunk 2077 это отражает развитие технологий и их применение в боевых системах.

## Лор в мире Cyberpunk 2077

### Контекст дополнительных механик

В мире NECPGAME дополнительные механики оружия стали стандартом для тактических операций, особенно популярного среди Solo, Netrunner и корпоративных солдат. Альтернативные режимы, управляемые снаряды и оружие для передвижения создают уникальные тактические возможности.

**Как работают дополнительные механики:**
- Оружие может иметь различные режимы стрельбы
- Снаряды могут управляться через кибердеки
- Оружие может использоваться для передвижения
- Различные механики эффективны в разных ситуациях

### Технологическое обоснование

**Альтернативные режимы:**
- Используют электронные системы для переключения режимов
- Разработано корпорациями Arasaka, Militech и Kang Tao
- Популярно среди Solo для тактических преимуществ
- Эффективно для адаптации под ситуацию

**Управляемые снаряды:**
- Используют системы наведения и кибердеки для управления
- Разработано корпорациями Arasaka и Kang Tao
- Популярно среди Netrunner для тактических манёвров
- Эффективно для обхода укрытий

**Оружие для передвижения:**
- Использует гравитационные и реактивные технологии
- Разработано корпорациями Arasaka и Militech
- Популярно среди Solo для мобильности
- Эффективно для тактических манёвров

## Альтернативные режимы стрельбы

### Одиночные выстрелы

**Характеристики:**
- Высокая точность (+30-50% точность)
- Высокий урон (+20-40% урон)
- Низкая скорость стрельбы (10-30 выстрелов в минуту)
- Низкое потребление боезапаса

**Тактические применения:**
- Дальние дистанции для максимальной точности
- Ситуации, требующие высокой точности
- Когда точность важнее скорости

**Типы оружия:**
- Пистолеты, винтовки, снайперские винтовки

### Очередь

**Характеристики:**
- Средняя точность (базовая точность)
- Средний урон (базовый урон)
- Средняя скорость стрельбы (300-600 выстрелов в минуту)
- Среднее потребление боезапаса

**Тактические применения:**
- Средние дистанции для баланса урона и точности
- Ситуации, требующие баланса урона и точности
- Когда нужен баланс между уроном и точностью

**Типы оружия:**
- Автоматы, винтовки, пистолеты-пулеметы

### Автоматический огонь

**Характеристики:**
- Низкая точность (-20-40% точность)
- Низкий урон за пулю (-10-20% урон за пулю)
- Высокая скорость стрельбы (600-900 выстрелов в минуту)
- Высокое потребление боезапаса

**Тактические применения:**
- Ближние дистанции для максимального урона
- Ситуации, требующие максимального урона
- Когда урон важнее точности

**Типы оружия:**
- Автоматы, пистолеты-пулеметы, пусковые установки

### Заряженные выстрелы

**Характеристики:**
- Накопление энергии для мощного удара
- Увеличение урона при полном заряде (+100-200% урон)
- Время зарядки (1-3 секунды)
- Ограниченное количество заряженных выстрелов

**Тактические применения:**
- Ситуации, требующие мощного удара
- Когда нужен максимальный урон за один выстрел
- Создание тактических преимуществ через мощный удар

**Типы оружия:**
- Лазерное оружие, энергетическое оружие, пусковые установки

### Залп

**Характеристики:**
- Несколько снарядов одновременно (3-10 снарядов)
- Высокий урон за залп (+50-100% урон за залп)
- Ограниченное количество залпов
- Высокое потребление боезапаса

**Тактические применения:**
- Ситуации, требующие максимального урона за короткое время
- Атака по группе целей
- Создание тактических преимуществ через залп

**Типы оружия:**
- Пусковые установки, дробовики, энергетическое оружие

## Управляемые снаряды

### Ракеты с наведением

**Характеристики:**
- Самонаведение на цели через системы наведения
- Обход укрытий через наведение
- Увеличение точности на 50-100%
- Ограниченное количество ракет

**Тактические применения:**
- Обход укрытий для гарантированных попаданий
- Атака по движущимся целям
- Создание тактических преимуществ через наведение

**Типы оружия:**
- Пусковые установки, smart-оружие

### Дроны для атаки

**Характеристики:**
- Призыв дронов для атаки через оружие
- Автоматическая атака по целям
- Ограниченное время жизни дронов (30-120 секунд)
- Ограниченное количество дронов

**Тактические применения:**
- Контроль территории через дронов
- Поддержка в групповых боях
- Отвлечение внимания врагов

**Типы оружия:**
- Призывное оружие, размещаемое оружие

### Снаряды с управляемой траекторией

**Характеристики:**
- Управление траекторией через кибердеку
- Обход укрытий через управляемую траекторию
- Увеличение точности на 30-50%
- Ограниченное количество управляемых снарядов

**Тактические применения:**
- Обход укрытий для гарантированных попаданий
- Атака по движущимся целям
- Создание тактических преимуществ через управление

**Типы оружия:**
- Smart-оружие, энергетическое оружие

### Управление через кибердеку

**Характеристики:**
- Управление снарядами через кибердеку
- Увеличение эффективности на 20-40%
- Доступ к уникальным способностям
- Требует навыков Netrunner

**Тактические применения:**
- Максимальная эффективность управляемых снарядов
- Доступ к уникальным способностям
- Создание тактических преимуществ через управление

**Типы оружия:**
- Smart-оружие, управляемые снаряды

## Оружие для передвижения

**Характеристики:**
- Притягивание/отталкивание целей и объектов
- Использование для передвижения (притягивание к объектам)
- Создание гравитационных аномалий
- Ограниченное количество использований

**Тактические применения:**
- Притягивание к объектам для быстрого передвижения
- Отталкивание от опасных зон
- Создание тактических преимуществ через гравитацию

**Типы оружия:**
- Гравитационное оружие, экспериментальное оружие

**Характеристики:**

**Тактические применения:**
- Быстрое передвижение для тактических манёвров
- Уклонение от опасных зон
- Создание тактических преимуществ через скорость

**Типы оружия:**

- Ограниченное количество телепортаций

**Тактические применения:**
- Мгновенное перемещение для тактических манёвров
- Обход препятствий для тактических преимуществ
- Создание тактических возможностей через телепортацию
