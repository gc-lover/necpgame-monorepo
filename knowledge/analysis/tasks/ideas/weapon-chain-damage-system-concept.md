# Система цепного урона (молнии, EMP волны) - Концепция

## Обзор

Система цепного урона создаёт динамичные боевые сценарии, где урон от одного выстрела распространяется между несколькими целями через электрические или электромагнитные эффекты. В мире Cyberpunk 2077 это отражает развитие энергетического оружия и его взаимодействие с кибернетическими имплантами и металлическими поверхностями.

## Лор в мире Cyberpunk 2077

### Контекст цепного урона

В мире NECPGAME цепной урон стал стандартом для энергетического оружия, особенно популярного среди Netrunner и корпоративных солдат. Молнии и EMP волны эффективно распространяются между целями, создавая тактические преимущества в групповых боях.

**Как работает цепной урон:**
- Первичная цель получает полный урон от выстрела
- Молния или EMP волна прыгает на ближайшие подходящие цели
- Каждый прыжок снижает урон на 20-30% для балансировки
- Максимум 3-5 прыжков в зависимости от типа оружия
- Металлические объекты и импланты увеличивают шанс быть целью

### Технологическое обоснование

**Молнии:**
- Используют технологию направленной энергии для создания электрических разрядов
- Автоматически находят ближайшие проводящие цели (металл, импланты, вода)
- Популярны среди Solo и Netrunner для групповых боёв
- Разработаны корпорациями Kang Tao и Arasaka

**EMP волны:**
- Электромагнитные импульсы распространяются волной от точки попадания
- Отключают импланты и кибер-части на время
- Наносят урон только по кибернетическим компонентам
- Распространяются через металлические поверхности
- Используются для создания "мёртвых зон" без имплантов

## Типы цепного урона

### Молнии / Электричество

**Характеристики:**
- Первичная цель получает 100% урона
- Молния прыгает на ближайшие металлические объекты/импланты
- Каждый прыжок снижает урон на 20-30%
- Максимум 3-5 прыжков (зависит от типа оружия)
- Импланты увеличивают шанс быть целью на 50%
- Металлические части окружения проводят молнию

**Тактические применения:**
- Групповые бои против нескольких целей
- Бои в металлических помещениях
- Против врагов с множеством имплантов
- Создание зон контроля через цепные реакции

**Типы молний:**
- **Стандартные:** Базовый цепной урон
- **Усиленные:** Больше прыжков, меньше снижение урона
- **Фокусные:** Меньше прыжков, но больше урон на цель
- **Распределённые:** Равномерное распределение урона между целями

### EMP волны

**Характеристики:**
- Электромагнитный импульс распространяется волной от точки попадания
- Отключает импланты на 3-10 секунд (зависит от силы)
- Урон только по кибер-частям (не наносит урон органике)
- Распространяется через металлические поверхности
- Создаёт "мёртвые зоны" без работающих имплантов

**Тактические применения:**
- Отключение имплантов врагов
- Создание тактических преимуществ
- Бои против кибернетически усиленных врагов
- Контроль территории через мёртвые зоны

**Типы EMP:**
- **Стандартные:** Базовое отключение имплантов
- **Усиленные:** Дольше отключение, больше урон по кибер-частям
- **Фокусные:** Меньше радиус, но сильнее эффект
- **Распределённые:** Больше радиус, но слабее эффект

## Механики взаимодействия

### Проводники для молний

**Металлические объекты:**
- Стены, полы, потолки из металла
- Трубы, кабели, арматура
- Транспортные средства
- Увеличивают радиус прыжков молний

**Импланты:**
- Кибер-руки, ноги, торс
- Нервные импланты
- Кибердеки
- Увеличивают шанс быть целью на 50%

**Вода:**
- Лужи, водоёмы, дождь
- Распространение молний по воде
- Увеличение радиуса действия
- Создание опасных зон

### Распространение EMP волн

**Металлические поверхности:**
- Проводят EMP волны дальше
- Увеличивают радиус действия
- Создают зоны отключения имплантов

**Кибер-части:**
- Притягивают EMP волны
- Получают дополнительный урон
- Отключаются на дольше

**Органические части:**
- Не получают урон от EMP
- Импланты отключаются, но тело не страдает
- Можно использовать для нелетального воздействия

### Комбо-эффекты

**Множественные молнии:**
- Каждая молния может создать свою цепь
- Пересекающиеся цепи усиливают урон
- Создание сетей молний между целями

**Молнии + EMP:**
- Молнии наносят урон, EMP отключает импланты
- Комбинированный эффект для максимального контроля
- Создание тактических преимуществ

**Молнии + Вода:**
- Распространение молний по воде
- Увеличение радиуса действия
- Создание опасных зон

## Интеграция с игровыми системами

### Combat System
- Интеграция с системой урона и цепных реакций
- Отслеживание цепных прыжков
- Расчет урона для каждого прыжка

### Implant System
- Определение имплантов для цепных прыжков
- Отключение имплантов от EMP
- Урон по кибер-частям

### Environment System
- Определение металлических объектов
- Проводящие свойства материалов
- Взаимодействие с водой и другими элементами

### Visual System
- Визуализация молний между целями
- Визуализация EMP волн
- Эффекты отключения имплантов
