# Система связи классов персонажей и оружия (синергии и бонусы) - Концепция

## Обзор

Система связи классов и оружия создаёт уникальные игровые стили для каждого класса, где каждый класс имеет естественную синергию с определёнными типами оружия, выражающуюся через бонусы, способности и перки. В мире Cyberpunk 2077 это отражает специализацию различных классов и их предпочтения в боевых системах.

## Лор в мире Cyberpunk 2077

### Контекст классовых синергий

В мире NECPGAME различные классы персонажей имеют естественную склонность к определённым типам оружия, основанную на их навыках, имплантах и боевом стиле. Эти синергии создают уникальные игровые стили для каждого класса.

**Как работают классовые синергии:**
- Каждый класс имеет приоритетные типы оружия
- Классовые бонусы усиливают определённые типы оружия
- Эксклюзивные способности доступны только для определённых классов
- Специальные перки усиливают синергии с оружием

### Технологическое обоснование

**Специализация классов:**
- Solo специализируются на боевых системах
- Netrunner используют кибероружие и smart-оружие
- Techie модифицируют и создают оружие
- Medtech используют лечебное и нелетальное оружие
- Другие классы имеют свои специализации

## Классы и их приоритетное оружие

### Solo

**Приоритетное оружие:**
- Автоматы, винтовки, снайперские винтовки
- Пистолеты, дробовики, ближний бой
- Стандартное боевое оружие

**Классовые бонусы:**
- +10-20% урон для автоматов и винтовок
- +15-25% точность для снайперских винтовок
- +10-15% скорость стрельбы для пистолетов
- +20-30% урон для ближнего боя

**Эксклюзивные способности:**
- Ган-ката: комбо-атаки с пистолетами
- Точный выстрел: увеличение шанса крита
- Боевая ярость: увеличение урона при низком здоровье

**Специальные перки:**
- Мастер стрельбы: увеличение точности
- Боевой опыт: снижение времени перезарядки
- Ближний бой: увеличение урона в ближнем бою

### Netrunner

**Приоритетное оружие:**
- Smart-оружие, кибероружие
- EMP оружие, лазерное оружие
- Электрическое оружие

**Классовые бонусы:**
- +15-25% урон для smart-оружия
- +20-30% эффективность для кибероружия
- +10-15% урон для EMP оружия
- +15-20% урон для лазерного оружия

**Эксклюзивные способности:**
- Взлом оружия: контроль smart-оружия врагов
- Кибер-усиление: увеличение эффективности кибероружия
- Электрический разряд: увеличение урона от электричества

**Специальные перки:**
- Хакер оружия: взлом smart-оружия
- Кибер-мастер: увеличение эффективности кибероружия
- Электрический мастер: увеличение урона от электричества

### Techie

**Приоритетное оружие:**
- Модифицированное оружие, размещаемое оружие
- Ловушки, крафтовое оружие
- Модульное оружие

**Классовые бонусы:**
- +20-30% эффективность для модифицированного оружия
- +25-35% урон для размещаемого оружия
- +15-20% эффективность для ловушек
- +10-15% урон для крафтового оружия

**Эксклюзивные способности:**
- Модификация оружия: улучшение характеристик оружия
- Размещение оружия: быстрое размещение турелей и ловушек
- Крафт оружия: создание уникального оружия

**Специальные перки:**
- Мастер модификаций: улучшение модификаций
- Инженер оружия: увеличение эффективности размещаемого оружия
- Крафтер: создание уникального оружия

## Синергии с имплантами

### Кибер-руки + Когти
- Увеличение урона ближнего боя на 30-50%
- Увеличение скорости атак на 20-30%
- Доступ к уникальным комбо-атакам

### Нервные ускорители + Ближний бой
- Увеличение скорости атак на 40-60%
- Увеличение шанса крита на 15-25%
- Доступ к уникальным комбо-атакам

### Импланты-проводники + Молнии
- Увеличение урона от молний на 25-35%
- Увеличение радиуса цепного урона на 30-50%
- Доступ к уникальным электрическим атакам
