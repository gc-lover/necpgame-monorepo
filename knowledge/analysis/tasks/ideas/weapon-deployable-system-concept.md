# Система размещаемого оружия (мины, ловушки, турели) - Концепция

## Обзор

Система размещаемого оружия позволяет игрокам устанавливать мины, ловушки и турели на карте для создания тактических преимуществ и контроля территории. В мире Cyberpunk 2077 это отражает тактический подход к боевым операциям и использование технологий для контроля пространства.

## Лор в мире Cyberpunk 2077

### Контекст размещаемого оружия

В мире NECPGAME размещаемое оружие стало стандартом для тактических операций, особенно популярного среди Techie, Solo и корпоративных солдат. Мины, ловушки и турели позволяют контролировать территорию и создавать тактические преимущества в бою.

**Как работает размещаемое оружие:**
- Игрок может размещать объекты на карте в определённых местах
- Объекты активируются при приближении целей или по команде
- Можно взломать или обезвредить размещённые объекты
- Турели и дроны автоматически стреляют по целям
- Ограниченный боезапас и время жизни

### Технологическое обоснование

**Мины:**
- Компактные взрывные устройства с различными типами активации
- Разработаны корпорациями Militech и Arasaka
- Популярны среди Solo и Techie для контроля территории
- Различные типы для разных ситуаций

**Ловушки:**
- Тактические устройства для фиксации и контроля целей
- Используют различные технологии (электричество, сети, клей)
- Популярны среди Techie и Netrunner
- Можно комбинировать с другими тактиками

**Турели и дроны:**
- Автоматические системы стрельбы с AI наведением
- Разработаны корпорациями Kang Tao и Arasaka
- Популярны среди Techie и Solo для поддержки
- Можно взломать для контроля

## Типы размещаемого оружия

### Мины

**Противопехотные:**
- Урон при наступлении на мину
- Различные типы взрывов (фрагментационные, направленные)
- Можно обнаружить с помощью сканеров
- Популярны для контроля проходов

**EMP:**
- Отключение имплантов при активации
- Урон только по кибер-частям
- Создание мёртвых зон без имплантов
- Эффективны против кибернетически усиленных врагов

**Химические:**
- Облако яда при активации
- DoT урон в радиусе
- Замедление движения
- Эффективны в закрытых помещениях

**Магнитные:**
- Притягивают металлические объекты
- Активация при приближении металла
- Можно использовать для контроля транспорта
- Эффективны против врагов с металлической броней

**Сетевые:**
- Ловят и фиксируют цели
- Ограничивают движение
- Можно использовать для захвата
- Эффективны для контроля территории

### Ловушки

**Электрические:**
- Урон + оглушение при активации
- Распространение по воде
- Можно использовать для создания опасных зон
- Эффективны в металлических помещениях

**Взрывные:**
- Урон + отбрасывание при активации
- Разрушение окружения
- Можно использовать для создания проходов
- Эффективны для контроля территории

**Сетевые:**
- Фиксация целей на месте
- Ограничение движения
- Можно использовать для захвата
- Эффективны для контроля проходов

**Лазерные:**
- Разрезание целей при прохождении
- Высокий урон по незащищённым частям
- Можно использовать для создания барьеров
- Эффективны для контроля территории

**Клейкие:**
- Замедление движения при активации
- Фиксация целей на месте
- Можно использовать для контроля
- Эффективны для создания ловушек

### Турели и дроны

**Турели:**
- Автоматическая стрельба по целям
- AI наведение с различной точностью
- Ограниченный боезапас
- Можно взломать для контроля

**Дроны:**
- Мобильные системы стрельбы
- Автоматическое наведение
- Ограниченное время жизни
- Можно взломать для контроля

**Типы турелей:**
- **Стандартные:** Базовое наведение и урон
- **Усиленные:** Больше урон, лучше наведение
- **EMP:** Отключение имплантов
- **Лазерные:** Высокий урон, перегрев

## Механики размещения и активации

### Размещение

**Ограничения:**
- Можно размещать только в определённых местах
- Ограничение количества одновременно размещённых объектов
- Требует времени для установки
- Можно прервать размещение

**Визуальные индикаторы:**
- Показ доступных мест для размещения
- Предпросмотр размещения
- Индикаторы активации

### Активация

**При приближении:**
- Автоматическая активация при приближении целей
- Различные радиусы активации
- Можно настроить чувствительность

**По команде:**
- Ручная активация по команде игрока
- Можно использовать для синхронизации
- Тактические преимущества

**Взлом:**
- Netrunner может взломать размещённые объекты
- Взломанные объекты работают на игрока
- Требует времени и навыков

### Обезвреживание

**Вручную:**
- Можно обезвредить вручную
- Требует времени и навыков
- Риск активации при ошибке

**С помощью имплантов:**
- Специальные импланты для обезвреживания
- Быстрее и безопаснее
- Требует специальных навыков
