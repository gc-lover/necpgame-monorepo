# Система стихийных эффектов оружия (огонь, лед, яд, кислота) - Концепция

## Обзор

Система стихийных эффектов создаёт разнообразие в боевых механиках, где оружие может наносить различные стихийные эффекты, создавая тактические возможности и синергии. В мире Cyberpunk 2077 это отражает развитие химических и энергетических технологий и их применение в боевых системах.

## Лор в мире Cyberpunk 2077

### Контекст стихийных эффектов

В мире NECPGAME стихийные эффекты стали стандартом для тактических операций, особенно популярного среди Techie, Solo и корпоративных солдат. Огонь, лёд, яд и кислота создают уникальные тактические возможности и синергии.

**Как работают стихийные эффекты:**
- Оружие наносит стихийный урон при попадании
- DoT эффекты наносят урон со временем
- Накопление эффектов усиливает урон
- Взаимодействие между стихиями создаёт синергии
- Различные стихии эффективны против разных типов целей

### Технологическое обоснование

**Огненное оружие:**
- Использует термические модули для создания огня
- Разработано корпорациями Militech и Kang Tao
- Популярно среди Solo для контроля территории
- Эффективно против органических целей

**Ледяное оружие:**
- Использует криогенные модули для создания льда
- Разработано корпорациями Biotechnica и Arasaka
- Популярно среди Techie для тактических преимуществ
- Эффективно для замедления и контроля

**Ядовитое оружие:**
- Использует химические инжекторы для создания яда
- Разработано корпорациями Biotechnica
- Популярно среди Netrunner для скрытых атак
- Эффективно для долгосрочного урона

**Кислотное оружие:**
- Использует химические инжекторы для создания кислоты
- Разработано корпорациями Biotechnica
- Популярно среди Techie для разрушения брони
- Эффективно против бронированных целей

## Типы стихийных эффектов

### Огненное оружие

**Характеристики:**
- DoT урон от горения (5-15 урона каждые 0.5 секунды)
- Распространение огня на ближайшие объекты
- Взаимодействие с химическими веществами (взрывы)
- Взаимодействие с водой (пар, снижение урона)

**Тактические применения:**
- Контроль территории через распространение огня
- DoT урон для долгосрочного контроля
- Взрывы при взаимодействии с химией
- Создание опасных зон

**Типы огненного оружия:**
- **Стандартные:** Базовый DoT и распространение
- **Усиленные:** Больше DoT, быстрее распространение
- **Фокусные:** Меньше распространение, больше DoT
- **Взрывные:** Взрывы при взаимодействии с химией

### Ледяное оружие

**Характеристики:**
- Замедление движения и атак на 20-50%
- Полная заморозка при накоплении эффекта
- Хрупкость замороженных целей (+50% урон)
- Взаимодействие с водой (замерзание)
- Взаимодействие с огнём (тает, но наносит урон)

**Тактические применения:**
- Замедление целей для тактических преимуществ
- Заморозка для контроля
- Увеличение урона по замороженным целям
- Создание ледяных поверхностей

**Типы ледяного оружия:**
- **Стандартные:** Базовое замедление и заморозка
- **Усиленные:** Больше замедление, быстрее заморозка
- **Фокусные:** Меньше радиус, больше эффект
- **Распределённые:** Больше радиус, меньше эффект

### Ядовитое оружие

**Характеристики:**
- DoT урон от яда (3-10 урона каждые 0.5 секунды)
- Снижение регенерации на 50-100%
- Накопление яда (больше попаданий = больше урон)
- Взаимодействие с огнём (взрыв ядовитого облака)
- Взаимодействие с электричеством (электрификация яда)

**Тактические применения:**
- DoT урон для долгосрочного контроля
- Снижение регенерации для ослабления целей
- Накопление урона через множественные попадания
- Взрывы при взаимодействии с огнём

**Типы ядовитого оружия:**
- **Стандартные:** Базовый DoT и накопление
- **Усиленные:** Больше DoT, быстрее накопление
- **Фокусные:** Меньше радиус, больше DoT
- **Взрывные:** Взрывы при взаимодействии с огнём

### Кислотное оружие

**Характеристики:**
- Снижение брони цели на 10-30% за попадание
- DoT урон (2-8 урона каждые 0.5 секунды)
- Разрушение укрытий (металл, пластик)
- Взаимодействие с водой (распространение)

**Тактические применения:**
- Разрушение брони для последующих атак
- DoT урон для долгосрочного контроля
- Разрушение укрытий для тактических преимуществ
- Распространение по воде для контроля территории

**Типы кислотного оружия:**
- **Стандартные:** Базовое снижение брони и DoT
- **Усиленные:** Больше снижение брони, больше DoT
- **Фокусные:** Меньше радиус, больше эффект
- **Распределённые:** Больше радиус, меньше эффект

### Огонь + Химия

- Создание ледяных поверхностей
- Увеличение радиуса действия

### Лёд + Огонь
- Таяние льда
- Урон при таянии
- Создание пара

### Яд + Огонь

### Яд + Электричество

- Цепной урон через ядовитое облако
- Увеличение эффективности

### Кислота + Вода
- Распространение кислоты по воде
- Увеличение радиуса действия
- Создание опасных зон
