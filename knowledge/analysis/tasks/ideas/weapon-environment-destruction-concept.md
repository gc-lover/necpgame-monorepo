# Система разрушения окружения и частей тела взрывным оружием - Концепция

## Обзор

Система разрушения окружения и частей тела взрывным оружием создаёт динамичные боевые сценарии, где взрывы физически изменяют окружение и могут разрушать части тел, создавая тактические возможности. В мире Cyberpunk 2077 это отражает мощь взрывного оружия и его влияние на боевую тактику.

## Лор в мире Cyberpunk 2077

### Контекст разрушения окружения

В мире NECPGAME взрывное оружие стало стандартом для тактических операций, особенно популярного среди Solo и корпоративных солдат. Взрывы не только наносят урон, но и физически изменяют окружение, создавая новые проходы и разрушая укрытия.

**Как работает разрушение:**
- Взрывы разрушают укрытия (стены, баррикады, препятствия)
- Создаются новые проходы для тактических манёвров
- Части тел могут быть оторваны взрывами
- Осколки наносят дополнительный урон в радиусе
- Оглушение в радиусе взрыва дезориентирует цели

### Технологическое обоснование

**Взрывное оружие:**
- Использует различные типы взрывчатых веществ (C4, пластид, нанобомбы)
- Разработано корпорациями Militech и Arasaka для военных операций
- Популярно среди Solo для тактических преимуществ
- Различные типы взрывов для разных ситуаций

**Гранаты:**
- Компактные взрывные устройства для тактического использования
- Различные типы для разных целей (урон, EMP, дым, химия)
- Популярны среди всех классов для контроля территории

## Типы взрывного оружия

### Взрывное оружие

**Характеристики:**
- Разрушение укрытий (стены, баррикады, препятствия)
- Создание новых проходов для тактических манёвров
- Разрушение частей тела (отрывание конечностей)
- Осколки наносят дополнительный урон в радиусе
- Оглушение в радиусе взрыва на 2-5 секунд

**Тактические применения:**
- Разрушение укрытий врагов
- Создание новых путей для атаки
- Контроль территории через разрушение
- Тактические преимущества в групповых боях

**Типы взрывов:**
- **Стандартные:** Базовое разрушение и урон
- **Фокусные:** Больше разрушение, меньше радиус
- **Распределённые:** Больше радиус, меньше разрушение
- **Проникающие:** Разрушение через несколько слоёв

### Гранаты

**Фрагментационные:**
- Урон + осколки в радиусе
- Разрушение окружения
- Оглушение целей

**EMP:**
- Отключение имплантов на время
- Урон только по кибер-частям
- Не разрушает окружение

**Дымовые:**
- Закрытие видимости в радиусе
- Тактическое прикрытие
- Не наносит урон

**Химические:**
- DoT урон + замедление
- Облако яда в радиусе
- Разрушение органических материалов

**Клейкие:**
- Фиксация целей на месте
- Замедление движения
- Можно использовать для контроля

## Механики разрушения

### Разрушение окружения

**Укрытия:**
- Стены из различных материалов (дерево, металл, бетон)
- Баррикады и препятствия
- Двери и окна
- Разрушение создаёт новые проходы

**Материалы:**
- **Дерево:** Легко разрушается, создаёт осколки
- **Металл:** Средняя прочность, создаёт опасные осколки
- **Бетон:** Высокая прочность, требует мощных взрывов
- **Пластик:** Легко разрушается, плавится от огня

**Создание проходов:**
- Взрывы создают новые пути для атаки
- Можно использовать для обхода укрытий
- Тактические преимущества в бою

### Разрушение частей тела

**Органические части:**
- Отрывание конечностей при мощных взрывах
- Критический урон при попадании в части тела
- Оглушение и дезориентация

**Кибер-части:**
- Отключение кибер-конечностей
- Повреждение имплантов
- Дополнительный урон от осколков

**Визуальные эффекты:**
- Реалистичное отображение разрушения
- Кровь и осколки
- Анимации падения частей тела

### Осколки и дополнительный урон

**Осколки:**
- Наносят дополнительный урон в радиусе
- Различные типы осколков (металл, стекло, камень)
- Могут застрять в теле (персистентные эффекты)

**Радиус действия:**
- Зависит от типа взрыва
- Уменьшается с расстоянием
- Можно усилить модификаторами

### Оглушение

**Эффекты:**
- Дезориентация на 2-5 секунд
- Снижение точности стрельбы
- Замедление движения
- Визуальные искажения

**Защита:**
- Импланты могут снижать эффект
- Броня может поглощать часть урона
- Можно использовать укрытия
