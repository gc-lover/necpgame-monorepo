# Дополнительные механики оружия: взрыв при смерти, поглощение урона, фазирование, мутация - Концепция

## Обзор

Система дополнительных механик оружия создаёт уникальные тактические возможности через взрывы при смерти, поглощение урона, фазирование и мутацию. В мире Cyberpunk 2077 это отражает развитие экспериментальных технологий и их применение в боевых системах.

## Лор в мире Cyberpunk 2077

### Контекст экспериментальных механик

В мире NECPGAME экспериментальные механики оружия стали доступны для опытных бойцов, особенно популярного среди Solo и корпоративных солдат. Взрывы при смерти, поглощение урона, фазирование и мутация создают уникальные тактические возможности.

**Как работают экспериментальные механики:**
- Цели могут взрываться при смерти
- Оружие может поглощать урон для усиления
- Снаряды могут проходить через препятствия
- Оружие может мутировать в бою

### Технологическое обоснование

**Взрывы и поглощение:**
- Используют биотехнологии и энергетические технологии
- Разработано корпорациями Biotechnica и Arasaka
- Популярно среди Solo для тактических преимуществ
- Эффективно для создания хаоса в бою

**Фазирование и мутация:**
- Используют квантовые и биотехнологии
- Разработано корпорациями Arasaka и Biotechnica
- Популярно среди Netrunner для тактических преимуществ
- Эффективно для обхода препятствий и адаптации

## Взрыв при смерти

### Цели взрываются при смерти

**Характеристики:**
- Цели взрываются при смерти, нанося урон окружающим целям
- Урон в радиусе 5-15 метров (50-100% от урона цели)
- Различные типы взрывов для различных типов оружия
- Визуальные эффекты взрывов

**Тактические применения:**
- Создание урона по группе целей через взрывы
- Создание тактических преимуществ через взрывы
- Увеличение эффективности через взрывы
- Создание хаоса в групповых боях

**Типы взрывов:**
- **Огненные:** Взрывы огня, распространение огня
- **Ядовитые:** Взрывы яда, создание ядовитых облаков
- **Электрические:** Взрывы электричества, цепные разряды
- **Энергетические:** Взрывы энергии, ударные волны

### Различные типы взрывов

**Огненные взрывы:**
- Взрывы огня при смерти цели
- Распространение огня на ближайшие объекты
- Урон от огня в радиусе взрыва
- Визуальные эффекты пламени

**Ядовитые взрывы:**
- Взрывы яда при смерти цели
- Создание ядовитых облаков в радиусе взрыва
- DoT урон от яда в радиусе взрыва
- Визуальные эффекты ядовитых облаков

**Электрические взрывы:**
- Взрывы электричества при смерти цели
- Цепные разряды между ближайшими целями
- Урон от электричества в радиусе взрыва
- Визуальные эффекты молний

**Энергетические взрывы:**
- Взрывы энергии при смерти цели
- Ударные волны в радиусе взрыва
- Урон от энергии в радиусе взрыва
- Визуальные эффекты энергетических волн

### Урон по окружающим целям

**Характеристики:**
- Урон по окружающим целям в радиусе взрыва
- Урон составляет 50-100% от урона цели
- Различные типы урона для различных типов взрывов
- Ограничения на максимальный радиус взрыва

**Тактические применения:**

## Поглощение урона

### Оружие поглощает урон для усиления

**Характеристики:**
- Оружие поглощает урон, нанесённый игроку, для усиления
- Накопление поглощённого урона увеличивает эффективность оружия
- Увеличение урона на 10-20% за каждые 100 единиц поглощённого урона
- Максимальное увеличение урона на 100-200% при максимальном накоплении

**Тактические применения:**
- Увеличение эффективности через поглощение урона
- Создание мощных атак при максимальном накоплении
- Создание тактических преимуществ через поглощение
- Увеличение эффективности через управление накоплением

**Типы поглощения:**
- **Стандартное:** Поглощение стандартного урона
- **Элементальное:** Поглощение элементального урона
- **Энергетическое:** Поглощение энергетического урона
- **Комбинированное:** Поглощение различных типов урона

### Накопление поглощённого урона

**Характеристики:**
- Накопление поглощённого урона при получении урона
- Увеличение эффективности оружия при накоплении
- Максимальное накопление увеличивает урон на 100-200%
- Ограничения на максимальное накопление

**Тактические применения:**
- Увеличение эффективности через накопление
- Создание мощных атак при максимальном накоплении
- Создание тактических преимуществ через накопление
- Увеличение эффективности через управление накоплением

### Усиление оружия при накоплении

**Характеристики:**
- Усиление оружия при накоплении поглощённого урона
- Увеличение урона на 10-20% за каждые 100 единиц поглощённого урона
- Максимальное увеличение урона на 100-200%

## Фазирование

### Прохождение снарядов через препятствия

**Характеристики:**
- Снаряды проходят через препятствия (стены, баррикады, укрытия)
- Прохождение через препятствия не снижает урон
- Различные типы фазирования для различных типов оружия
- Ограничения на количество препятствий для прохождения

**Тактические применения:**
- Обход укрытий через фазирование
- Создание тактических преимуществ через фазирование
- Увеличение эффективности через фазирование
- Создание возможностей для атак через препятствия

**Типы фазирования:**
- **Базовое:** Простое прохождение через препятствия
- **Продвинутое:** Прохождение через несколько препятствий
- **Квантовое:** Прохождение через любые препятствия
- **Экспериментальное:** Уникальные типы фазирования

### Прохождение через стены

**Характеристики:**
- Снаряды проходят через стены без снижения урона
- Прохождение через стены создаёт тактические возможности
- Различные типы стен для различных типов фазирования
- Ограничения на толщину стен для прохождения

**Тактические применения:**
- Обход укрытий через прохождение через стены
- Создание тактических преимуществ через прохождение
- Увеличение эффективности через прохождение
- Создание возможностей для атак через стены

### Прохождение через врагов

**Характеристики:**

- Прохождение через врагов создаёт множественные попадания
- Различные типы врагов для различных типов фазирования
- Ограничения на количество врагов для прохождения

**Тактические применения:**
- Атака по группе целей через прохождение через врагов
- Создание тактических преимуществ через прохождение
- Увеличение эффективности через прохождение
- Создание возможностей для множественных попаданий

### Квантовое фазирование

**Характеристики:**
- Квантовое фазирование позволяет проходить через любые препятствия
- Прохождение через препятствия не снижает урон
- Ограничения на использование квантового фазирования
- Высокая стоимость и редкость

**Тактические применения:**
