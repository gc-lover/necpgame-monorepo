# Концепция: Система игровых механик оружия

## Описание
Система управления ресурсами оружия, прогрессии улучшений и балансировки для различных режимов игры.

## 1. Управление ресурсами

### Боезапас
- Ограниченный запас специальных снарядов
- Пополнение: лут, покупка, крафт

### Перегрев
- Накопление тепла при стрельбе (лазерное оружие)
- Автоматическое охлаждение
- Перегрев = блокировка стрельбы на 3-5 сек

### Кулдауны
- Для размещаемого оружия (мины, турели)
- Cooldown: 10-30 сек в зависимости от типа

### Энергия
- Для энергетического оружия
- Регенерация: 10% в секунду
- Максимум: 100 единиц

## 2. Прогрессия

### Улучшения оружия
- Уровни улучшения: 0-10
- Каждый уровень: +5% урон, +2% точность
- Стоимость улучшения растет экспоненциально

### Перки
- Специальные перки для каждого типа оружия
- Открываются через уровень персонажа
- Примеры: "Быстрая перезарядка", "Критический урон +50%"

### Модификации
- Обвес, чипы, прошивки (см. Issue #1575)
- Изменение характеристик

## 3. Балансировка

### PvE
- Урон x1.5
- Больше эффектов
- Разрушения окружения

### PvP
- Урон x1.0
- Ограниченные эффекты
- Минимум разрушений

### Арены
- Стандартизированные правила
- Пресеты оружия
- Fair play баланс
