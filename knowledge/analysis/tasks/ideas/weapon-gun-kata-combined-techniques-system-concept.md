# Система ган-каты и комбинированных техник - Концепция

## Обзор

Система ган-каты и комбинированных техник объединяет стрельбу и ближний бой для создания уникальных боевых техник. В мире Cyberpunk 2077 это отражает развитие боевых искусств и их интеграцию с современными технологиями.

## Лор в мире Cyberpunk 2077

### Контекст ган-каты

В мире NECPGAME ган-ката стала стандартом для опытных бойцов, особенно популярного среди Solo и корпоративных солдат. Ган-ката объединяет стрельбу и ближний бой для создания уникальных боевых техник, основанных на боевых искусствах и кибернетических технологиях.

**Как работает ган-ката:**
- Комбинация стрельбы и ближнего боя
- Блокирование выстрелов оружием ближнего боя
- Синхронизированные атаки
- Уникальные комбо-атаки

### Технологическое обоснование

**Боевые искусства:**
- Использует развитие боевых искусств для координации стрельбы и ближнего боя
- Разработано корпорациями Arasaka и независимыми мастерами
- Популярно среди Solo для тактических преимуществ
- Эффективно для создания уникальных боевых стилей

**Кибернетические технологии:**
- Использует кибернетические импланты для координации
- Разработано корпорациями Arasaka и Biotechnica
- Популярно среди Solo и Netrunner для тактических преимуществ
- Эффективно для снижения штрафов и увеличения бонусов
