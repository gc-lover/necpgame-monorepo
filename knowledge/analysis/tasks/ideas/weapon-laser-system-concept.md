# Система лазерного оружия (непрерывные лучи, импульсы, рельсы) - Концепция

## Обзор

Система лазерного оружия создаёт уникальные механики энергетического боя, где лазеры могут резать через несколько целей, пробивать укрытия и создавать направляющие пути для других снарядов. В мире Cyberpunk 2077 это отражает развитие энергетических технологий и их применение в боевых системах.

## Лор в мире Cyberpunk 2077

### Контекст лазерного оружия

В мире NECPGAME лазерное оружие стало символом технологического превосходства, особенно популярного среди корпоративных солдат и Netrunner. Лазеры эффективны против брони и могут создавать тактические преимущества через прожжение укрытий и отражение.

**Как работает лазерное оружие:**
- Лазеры наносят урон мгновенно при попадании
- Могут резать через несколько целей подряд
- Пробивают укрытия из дерева и пластика
- Отражаются от зеркальных поверхностей
- Создают направляющие пути для других снарядов

### Технологическое обоснование

**Непрерывные лучи:**
- Используют технологию направленной энергии для создания постоянного луча
- Накопление тепла при длительной стрельбе требует охлаждения
- Разработаны корпорациями Kang Tao и Arasaka
- Популярны среди Solo и Netrunner для точных атак

**Импульсные лазеры:**
- Высокоэнергетические импульсы для мощных ударов
- Можно заряжать для более мощных импульсов
- Эффективны против брони и укрытий
- Популярны среди корпоративных солдат

**Лазерные рельсы:**
- Создают направляющие пути для других снарядов
- Увеличивают скорость и урон снарядов
- Можно изгибать траекторию для обхода укрытий
- Разработаны корпорациями Kang Tao для тактических преимуществ

## Типы лазерного оружия

### Непрерывные лучи

**Характеристики:**
- Урон наносится пока луч на цели
- Перегрев при длительной стрельбе (накопление тепла)
- Можно "резать" через несколько целей подряд
- Прожжение укрытий (дерево, пластик, тонкий металл)
- Отражение от зеркальных поверхностей

**Тактические применения:**
- Точные атаки по нескольким целям
- Прожжение укрытий для создания проходов
- Использование отражений для обхода укрытий
- Контроль территории через постоянный луч

**Типы непрерывных лучей:**
- **Стандартные:** Базовый урон и перегрев
- **Усиленные:** Больше урон, быстрее перегрев
- **Охлаждаемые:** Медленнее перегрев, меньше урон
- **Фокусные:** Меньше радиус, больше урон

### Импульсные лазеры

**Характеристики:**
- Высокий урон за импульс
- Можно заряжать для более мощного импульса
- Прожжение укрытий
- Отражение от зеркальных поверхностей
- Нет перегрева, но есть кулдаун между импульсами

**Тактические применения:**
- Мощные удары по бронированным целям
- Прожжение укрытий одним импульсом
- Использование отражений для тактических преимуществ
- Контроль территории через мощные импульсы

**Типы импульсных лазеров:**
- **Стандартные:** Базовый урон и зарядка
- **Усиленные:** Больше урон, дольше зарядка
- **Быстрые:** Меньше урон, быстрее зарядка
- **Фокусные:** Меньше радиус, больше урон

### Лазерные рельсы

**Характеристики:**
- Лазер создает "рельсы" для других снарядов
- Снаряды следуют по лазерному пути
- Можно изгибать траекторию для обхода укрытий
- Увеличение скорости снарядов на 50-100%
- Увеличение урона снарядов на 20-50%

**Тактические применения:**
- Направление снарядов по сложным траекториям
- Обход укрытий через изгиб траектории
- Усиление урона других типов оружия
- Создание тактических преимуществ

**Типы лазерных рельсов:**
- **Стандартные:** Базовое направление и усиление
- **Усиленные:** Больше усиление, меньше радиус
- **Гибкие:** Больше радиус изгиба, меньше усиление
- **Фокусные:** Меньше радиус, больше усиление

## Механики взаимодействия

### Прожжение укрытий

**Материалы:**
- **Дерево:** Легко прожгается, создаёт дым
- **Пластик:** Легко прожгается, плавится
- **Тонкий металл:** Средняя прочность, требует времени
- **Бетон:** Высокая прочность, требует мощных лазеров

**Создание проходов:**
- Лазеры создают новые пути для атаки
- Можно использовать для обхода укрытий
- Тактические преимущества в бою

### Отражение от зеркальных поверхностей

**Зеркальные поверхности:**
- Зеркала, полированный металл, отражающие поверхности
- Лазеры отражаются под углом попадания
- Можно использовать для обхода укрытий
- Создание тактических преимуществ

**Механика отражения:**
- Угол отражения равен углу попадания
- Урон сохраняется при отражении
- Можно создать цепь отражений
- Требует точного расчёта траектории

### Резание через несколько целей

**Механика:**
- Лазер проходит через несколько целей подряд
- Урон снижается на 10-20% для каждой последующей цели
- Максимум 3-5 целей в зависимости от типа лазера
- Эффективно для групповых боёв

**Тактические применения:**
- Атака нескольких целей одновременно
- Создание тактических преимуществ
- Контроль территории через резание
