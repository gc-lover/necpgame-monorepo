# Система ближнего боя (катана, когти, сюрикены) - Концепция

## Обзор

Система ближнего боя создаёт уникальные механики рукопашного боя, где различные типы оружия имеют уникальные характеристики и синергии с имплантами. В мире Cyberpunk 2077 это отражает традиции боевых искусств и их интеграцию с кибернетическими технологиями.

## Лор в мире Cyberpunk 2077

### Контекст ближнего боя

В мире NECPGAME ближний бой остаётся актуальным, особенно среди Solo и корпоративных солдат, которые ценят точность и контроль. Катана, когти и сюрикены стали символами мастерства и технологической интеграции.

**Как работает ближний бой:**
- Различные типы оружия имеют уникальные механики
- Синергии с имплантами усиливают эффективность
- Комбо-атаки создают тактические преимущества
- Блокирование выстрелов требует специальных имплантов
- Критические удары при правильном тайминге

### Технологическое обоснование

**Катана:**
- Традиционное японское оружие с кибернетическими улучшениями
- Энергетические клинки для усиления урона
- Популярны среди Solo и корпоративных солдат
- Символ мастерства и традиций

**Когти (импланты):**
- Кибернетические импланты для ближнего боя
- Разрывание брони и нанесение DoT урона
- Можно цепляться за поверхности для мобильности
- Популярны среди Solo и Netrunner

**Сюрикены:**
- Метательное оружие для ближнего и дальнего боя
- Застревают в теле (персистентные эффекты)
- Можно комбинировать с другими типами оружия
- Популярны среди Solo и Corpo

## Типы оружия ближнего боя

### Катана

**Характеристики:**
- Быстрые режущие атаки
- Высокий урон по незащищённым частям тела
- Можно блокировать выстрелы (требует специальных имплантов)
- Комбо-атаки с увеличением урона
- Критические удары при правильном тайминге

**Тактические применения:**
- Быстрые атаки по незащищённым целям
- Блокирование выстрелов для защиты
- Комбо-атаки для максимального урона
- Контроль территории через быстрые атаки

**Типы катаны:**
- **Стандартные:** Базовый урон и скорость
- **Энергетические:** Усиление урона через энергию
- **Быстрые:** Больше скорость, меньше урон
- **Мощные:** Больше урон, меньше скорость

### Когти (импланты)

**Характеристики:**
- Быстрые атаки с разрыванием брони
- Можно цепляться за поверхности для мобильности
- DoT урон от ран
- Синергии с другими имплантами

**Тактические применения:**
- Разрывание брони для последующих атак
- Мобильность через цепляние за поверхности
- DoT урон для долгосрочного контроля
- Комбо с другими типами оружия

**Типы когтей:**
- **Стандартные:** Базовый урон и разрывание
- **Усиленные:** Больше урон, лучше разрывание
- **Мобильные:** Лучше цепляние, меньше урон
- **Токсичные:** Ядовитый DoT урон

### Сюрикены

**Характеристики:**
- Быстрые броски для ближнего и дальнего боя
- Застревают в теле (персистентные эффекты)
- Можно комбинировать с другими типами оружия
- Множественные сюрикены создают комбо-эффекты

**Тактические применения:**
- Ближний и дальний бой в одном оружии
- Накопление урона через персистентные эффекты
- Комбо с другими типами оружия
- Контроль территории через множественные попадания

**Типы сюрикенов:**
- **Стандартные:** Базовый урон и персистентные эффекты
- **Ядовитые:** Ядовитый DoT урон
- **Взрывные:** Взрыв при детонации
- **Электрические:** Электрический DoT урон

## Механики взаимодействия

### Комбо-атаки

**Система комбо:**
- Последовательные атаки увеличивают урон
- Каждое попадание увеличивает множитель урона на 10-20%
- Максимум 5-10 комбо в зависимости от типа оружия
- Сброс комбо при промахе или блокировании

**Типы комбо:**
- **Быстрые:** Много быстрых атак с небольшим уроном
- **Мощные:** Медленные мощные атаки с большим уроном
- **Смешанные:** Комбинация быстрых и мощных атак

### Блокирование выстрелов

**Механика:**
- Можно блокировать выстрелы оружием ближнего боя
- Требует специальных имплантов (рефлексы, кибер-руки)
- Успешное блокирование отражает снаряд обратно
- Требует точного тайминга

**Тактические применения:**
- Защита от дальнобойных атак
- Отражение снарядов обратно во врагов
- Создание тактических преимуществ
- Демонстрация мастерства

### Критические удары

**Механика:**
- Критические удары при правильном тайминге
- Увеличение урона на 50-100%
- Требует точного тайминга и позиционирования
- Различные типы критических ударов

**Типы критических ударов:**
- **Тайминг:** Правильный момент атаки
- **Позиционирование:** Атака с правильной позиции
- **Комбо:** Критический удар в конце комбо
- **Синергия:** Критический удар с имплантами

### Синергии с имплантами

**Кибер-руки:**
- Увеличение урона и скорости атак
- Улучшенное блокирование выстрелов
- Специальные комбо-атаки

**Нервные ускорители:**
- Увеличение скорости атак
- Улучшенный тайминг для критических ударов
- Быстрее восстановление между атаками

**Кибердеки:**
- Улучшенное наведение для сюрикенов
- Специальные способности для ближнего боя
- Интеграция с другими системами
