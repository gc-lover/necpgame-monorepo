# Система модификаторов оружия (обвес, чипы, прошивки кибер-деки) - Концепция

## Обзор

Система модификаторов оружия позволяет изменять характеристики и добавлять эффекты через обвес, чипы и прошивки кибер-деки. В мире Cyberpunk 2077 это отражает развитие технологий модификации оружия и персонализацию боевых систем.

## Лор в мире Cyberpunk 2077

### Контекст модификаторов

В мире NECPGAME модификация оружия стала стандартом для персонализации боевых систем, особенно популярного среди Solo, Techie и корпоративных солдат. Обвес, чипы и прошивки создают уникальные комбинации эффектов и характеристик.

**Как работают модификаторы:**
- Обвес изменяет физические характеристики оружия
- Чипы добавляют электронные эффекты
- Прошивки изменяют программное обеспечение кибер-деки
- Комбинации модификаторов создают синергии

### Технологическое обоснование

**Обвес:**
- Использует физические модификации для изменения характеристик
- Разработано корпорациями Militech и независимыми производителями
- Популярно среди Solo для тактических преимуществ
- Эффективно для базовых улучшений

**Чипы:**
- Использует электронные технологии для добавления эффектов
- Разработано корпорациями Arasaka и Kang Tao
- Популярно среди Netrunner для тактических манёвров
- Эффективно для добавления специальных эффектов

**Прошивки:**
- Использует программное обеспечение для изменения поведения
- Разработано корпорациями Arasaka
- Популярно среди Netrunner для максимальной эффективности
- Эффективно для усиления эффектов и создания синергий

## Обвес на оружие

### Термический модуль

**Характеристики:**
- Добавляет огненные эффекты к оружию
- Увеличение урона от огня на 20-40%
- Распространение огня на ближайшие объекты
- Взаимодействие с химическими веществами (взрывы)

**Тактические применения:**
- Создание огненных эффектов для контроля территории
- Увеличение урона через распространение огня
- Создание тактических преимуществ через взрывы

### Криогенный модуль

**Характеристики:**

- Хрупкость замороженных целей (+50% урон)

**Тактические применения:**
- Замедление целей для тактических преимуществ
- Увеличение урона через хрупкость
- Создание тактических возможностей через заморозку

### Химический инжектор

**Характеристики:**

**Тактические применения:**
