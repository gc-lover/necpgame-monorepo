# Система персистентных эффектов для снарядов - Концепция

## Обзор

Система персистентных эффектов для снарядов создаёт уникальные механики взаимодействия, где болты, стрелы и сюрикены остаются в теле цели, нанося постоянный урон и создавая тактические возможности. В мире Cyberpunk 2077 это отражает жестокость боевых технологий и необходимость адаптации к новым типам оружия.

## Лор в мире Cyberpunk 2077

### Контекст персистентных снарядов

В мире NECPGAME персистентные снаряды стали популярны среди наёмников и корпоративных солдат как способ создания долгосрочных тактических преимуществ. Болты арбалетов, стрелы луков и сюрикены остаются в теле, нанося постоянный урон и ограничивая мобильность противника.

**Как работают персистентные снаряды:**
- Снаряды физически остаются в теле цели после попадания
- Каждый снаряд наносит DoT урон каждые 0.1 секунды
- Снаряды можно выдернуть вручную или с помощью имплантов
- Множественные снаряды создают комбо-эффекты
- Снаряды можно использовать для фиксации и подвешивания целей

### Технологическое обоснование

**Болты арбалетов:**
- Созданы с зазубренными наконечниками для максимального удержания в теле
- Используют специальные материалы, которые не ржавеют и не ломаются
- Могут быть оснащены наноботами для усиления урона
- Популярны среди Solo и Netrunner для скрытых атак

**Стрелы луков/арбалетов:**
- Различные типы наконечников (ядовитые, взрывные, EMP)
- Можно поджечь перед выстрелом для дополнительного урона
- Используются для создания ловушек и засад
- Популярны среди Nomad и Fixer для тактических операций

**Сюрикены:**
- Множественные сюрикены могут застревать в теле одновременно
- Каждый сюрикен наносит отдельный DoT урон
- Можно выдернуть все сразу с помощью специальных способностей
- Комбо: множество сюрикенов создают взрыв при детонации
- Популярны среди Solo и Corpo для ближнего боя

## Типы персистентных снарядов

### Болты арбалетов

**Характеристики:**
- Высокий базовый урон при попадании
- DoT урон: 5-15 урона каждые 0.1 секунды
- Длительность: 10-30 секунд (зависит от типа болта)
- Можно выдернуть вручную (3 секунды) или имплантом (1 секунда)
- Критический урон при попадании в уже встроенный болт (+50% урона)

**Тактические применения:**
- Фиксация целей для последующих атак
- Подвешивание целей на стенах или препятствиях
- Создание зон контроля через множественные болты
- Комбо с другими типами оружия

**Типы болтов:**
- **Стандартные:** Базовый DoT урон
- **Ядовитые:** Дополнительный ядовитый DoT
- **Взрывные:** Взрыв при выдергивании или детонации
- **EMP:** Отключение имплантов на время
- **Наноботовые:** Усиление урона со временем

### Стрелы луков/арбалетов

**Характеристики:**
- Средний базовый урон при попадании
- DoT урон: 3-10 урона каждые 0.1 секунды
- Длительность: 8-25 секунд
- Различные типы наконечников с уникальными эффектами
- Можно поджечь перед выстрелом для огненного DoT

**Тактические применения:**
- Создание ловушек через размещение стрел
- Поджог целей для распространения огня
- EMP стрелы для отключения имплантов
- Ядовитые стрелы для долгосрочного урона

**Типы наконечников:**
- **Стандартные:** Базовый DoT урон
- **Ядовитые:** Ядовитый DoT + снижение регенерации
- **Взрывные:** Взрыв при попадании или детонации
- **EMP:** Отключение имплантов + урон по кибер-частям
- **Огненные:** Огненный DoT + распространение огня
- **Ледяные:** Замедление + заморозка при накоплении

### Сюрикены

**Характеристики:**
- Низкий базовый урон при попадании
- DoT урон: 2-8 урона каждые 0.1 секунды
- Длительность: 5-20 секунд
- Можно застрять множественные сюрикены одновременно
- Комбо: множество сюрикенов создают взрыв при детонации

**Тактические применения:**
- Быстрые атаки с множественными попаданиями
- Накопление урона через множество сюрикенов
- Взрыв при детонации всех сюрикенов сразу
- Ближний и дальний бой в одном оружии

**Типы сюрикенов:**
- **Стандартные:** Базовый DoT урон
- **Ядовитые:** Ядовитый DoT
- **Взрывные:** Взрыв при детонации
- **Электрические:** Электрический DoT + цепной урон
- **Ледяные:** Замедление + заморозка

## Механики взаимодействия

### Выдергивание снарядов

**Вручную:**
- Время: 3 секунды
- Требует близкого контакта с целью
- Можно прервать атакой
- Восстанавливает часть здоровья цели

**С помощью имплантов:**
- Время: 1 секунда
- Требует специальных имплантов (магнитные захваты, кибер-руки)
- Нельзя прервать
- Восстанавливает меньше здоровья цели

**Способности:**
- Массовое выдергивание всех снарядов сразу
- Взрыв при выдергивании (для взрывных снарядов)
- Усиление урона при выдергивании

### Комбо-эффекты

**Множественные болты:**
- Каждый дополнительный болт увеличивает DoT урон на 10%
- Критический урон при попадании в уже встроенный болт
- Фиксация цели при 3+ болтах

**Множественные стрелы:**
- Огненные стрелы создают зону огня
- Ядовитые стрелы усиливают ядовитый DoT
- EMP стрелы создают зону отключения имплантов

**Множественные сюрикены:**
- 5+ сюрикенов создают взрыв при детонации
- Каждый дополнительный сюрикен увеличивает урон взрыва
- Массовое выдергивание всех сюрикенов создает цепную реакцию

### Фиксация и подвешивание

**Фиксация целей:**
- 3+ болта фиксируют цель на месте
- Цель не может двигаться, пока болты не выдернуты
- Можно использовать для создания засад

**Подвешивание:**
- Болты можно использовать для подвешивания целей на стенах
- Требует специальных болтов с крюками
- Создает тактические преимущества в бою
