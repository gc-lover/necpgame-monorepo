# Дополнительные механики оружия: разделение, рикошеты, замедление, маркировка - Концепция

## Обзор

Система дополнительных механик оружия создаёт тактические возможности через разделение снарядов, рикошеты, замедление и маркировку. В мире Cyberpunk 2077 это отражает развитие технологий и их применение в боевых системах для создания тактических преимуществ.

## Лор в мире Cyberpunk 2077

### Контекст тактических механик

В мире NECPGAME тактические механики оружия стали стандартом для опытных бойцов, особенно популярного среди Solo и корпоративных солдат. Разделение снарядов, рикошеты, замедление и маркировка создают уникальные тактические возможности.

**Как работают тактические механики:**
- Снаряды могут разделяться на части
- Снаряды могут рикошетить между целями
- Попадания могут замедлять цели
- Цели могут быть помечены для увеличения урона

### Технологическое обоснование

**Разделение и рикошеты:**
- Используют баллистические и энергетические технологии
- Разработано корпорациями Militech и Kang Tao
- Популярно среди Solo для тактических преимуществ
- Эффективно для обхода укрытий

**Замедление и маркировка:**
- Используют кибернетические и программные технологии
- Разработано корпорациями Arasaka и NetWatch
- Популярно среди Netrunner для тактических преимуществ
- Эффективно для контроля и координации

## Разделение снарядов

### Снаряды разделяются на несколько частей

**Характеристики:**
- Снаряды разделяются на 3-10 частей при активации
- Разделение происходит при приближении к цели или при попадании
- Каждая часть наносит урон (30-50% от исходного урона)
- Различные типы разделения для различных типов оружия

**Тактические применения:**
- Атака по группе целей через разделение
- Обход укрытий через разделение
- Увеличение шанса попадания через разделение
- Создание тактических преимуществ через разделение

**Типы разделения:**
- **Базовое:** Простое разделение на части
- **Направленное:** Разделение с направлением частей
- **Умное:** Разделение с наведением на цели
- **Экспериментальное:** Уникальные типы разделения

### Разделение при приближении к цели

**Характеристики:**
- Разделение происходит при приближении к цели на определённое расстояние
- Разделение создаёт веерное рассеивание частей
- Увеличение шанса попадания через разделение
- Различные расстояния разделения для различных типов оружия

**Тактические применения:**
- Увеличение шанса попадания через разделение
- Атака по группе целей через разделение
- Обход укрытий через разделение
- Создание тактических преимуществ через разделение

### Разделение при попадании

**Характеристики:**
- Разделение происходит при попадании в цель
- Разделение создаёт осколки, наносящие урон окружающим целям
- Увеличение урона через разделение
- Различные типы разделения для различных типов оружия

**Тактические применения:**
- Увеличение урона через разделение
- Атака по группе целей через разделение
- Создание тактических преимуществ через разделение
- Увеличение эффективности через разделение

## Рикошеты (расширенная версия)

### Рикошеты между целями

**Характеристики:**
- Снаряды рикошетят между целями (2-5 целей)
- Увеличение урона при рикошетах (+10-20% урон за рикошет)
- Умные рикошеты (smart-рикошеты) находят цели автоматически
- Ограничение количества рикошетов (максимум 3-5 рикошетов)

**Тактические применения:**
- Атака по группе целей через рикошеты
- Обход укрытий через рикошеты
- Увеличение эффективности через рикошеты
- Создание тактических преимуществ через рикошеты

**Типы рикошетов:**
- **Базовые:** Простые рикошеты между целями
- **Умные:** Рикошеты с автоматическим наведением
- **Энергетические:** Рикошеты для энергетического оружия
- **Экспериментальные:** Уникальные типы рикошетов

### Увеличение урона при рикошетах

**Характеристики:**
- Увеличение урона на 10-20% за каждый рикошет
- Максимальное увеличение урона на 50-100% при множественных рикошетах
- Различные увеличения для различных типов оружия
- Ограничения на максимальное увеличение

**Тактические применения:**
- Увеличение эффективности через рикошеты
- Создание максимального урона через рикошеты
- Создание тактических преимуществ через рикошеты
- Увеличение эффективности через управление рикошетами

### Умные рикошеты (smart-рикошеты)

**Характеристики:**
- Умные рикошеты находят цели автоматически через системы наведения
- Увеличение точности рикошетов на 50-100%
- Интеграция с кибердеками для максимальной эффективности
- Ограничения на количество умных рикошетов

**Тактические применения:**
- Максимальная эффективность через умные рикошеты
- Увеличение точности через умные рикошеты
- Создание тактических преимуществ через умные рикошеты
- Увеличение эффективности через управление умными рикошетами
