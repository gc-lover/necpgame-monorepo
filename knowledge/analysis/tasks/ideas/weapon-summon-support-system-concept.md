# Система призыва и поддержки оружием (призыв союзников, лечение) - Концепция

## Обзор

Система призыва и поддержки создаёт командные преимущества, где оружие может призывать союзников или лечить команду. В мире Cyberpunk 2077 это отражает развитие наноботовых и дроновых технологий и их применение в боевых системах.

## Лор в мире Cyberpunk 2077

### Контекст призыва и поддержки

В мире NECPGAME призыв и поддержка стали стандартом для командных операций, особенно популярного среди Techie, Medtech и корпоративных солдат. Дроны, наноботы и лечебные системы создают уникальные командные возможности.

**Как работает призыв и поддержка:**
- Оружие может призывать дронов и союзников для поддержки
- Лечебные системы восстанавливают здоровье команды
- Наноботы обеспечивают долгосрочную поддержку
- Ограниченное время жизни и количество призывов

### Технологическое обоснование

**Призывное оружие:**
- Использует наноботовые технологии для создания дронов
- Разработано корпорациями Kang Tao и Arasaka
- Популярно среди Techie для тактической поддержки
- Эффективно для контроля территории

**Лечебное оружие:**
- Использует медицинские наноботы для лечения
- Разработано корпорациями Biotechnica и Arasaka
- Популярно среди Medtech для поддержки команды
- Эффективно для долгосрочного выживания

## Типы призыва и поддержки

### Призывное оружие

**Характеристики:**
- Призыв дронов для атаки
- Призыв союзников для поддержки
- Ограниченное время жизни призванных существ (30-120 секунд)
- Ограниченное количество призывов (1-3 одновременно)

**Тактические применения:**
- Контроль территории через дронов
- Поддержка в групповых боях
- Отвлечение внимания врагов
- Создание тактических преимуществ

**Типы призывов:**
- **Боевые дроны:** Атака врагов
- **Поддерживающие дроны:** Лечение и поддержка
- **Разведывательные дроны:** Разведка и маркировка
- **Защитные дроны:** Защита и блокирование

### Лечебное оружие

**Характеристики:**
- Лечение союзников через оружие
- Восстановление здоровья в радиусе 5-15 метров
- Медицинские наноботы для долгосрочного лечения
- Восстановление через вампиризм

**Тактические применения:**
- Поддержка команды в бою
- Долгосрочное выживание
- Восстановление здоровья без медикаментов
- Создание командных преимуществ

**Типы лечебного оружия:**
- **Стандартное:** Базовое восстановление здоровья
- **Усиленное:** Больше восстановление, больше радиус
- **Наноботовое:** Долгосрочное лечение через наноботов
- **Вампирическое:** Восстановление через поглощение энергии

## Механики взаимодействия

### Призыв дронов

**Боевые дроны:**
- Автоматическая атака по врагам
- Ограниченный боезапас
- Можно уничтожить врагами
- Время жизни 30-60 секунд

**Поддерживающие дроны:**
- Лечение союзников в радиусе
- Восстановление здоровья каждые 1-2 секунды
- Время жизни 60-120 секунд
- Можно уничтожить врагами

**Разведывательные дроны:**
- Маркировка врагов для команды
- Разведка территории
- Время жизни 90-180 секунд
- Можно уничтожить врагами

**Защитные дроны:**
- Блокирование выстрелов
- Защита союзников
- Время жизни 30-90 секунд
- Можно уничтожить врагами

### Лечение команды

**Радиусное лечение:**
- Восстановление здоровья в радиусе
- Эффект на всех союзников в радиусе
- Постоянное восстановление пока оружие активно
- Визуальные эффекты лечения

**Наноботовое лечение:**
- Наноботы остаются в теле для долгосрочного лечения
- DoT восстановление здоровья
- Накопление наноботов для усиления
- Время жизни наноботов 30-120 секунд

**Вампирическое лечение:**
- Восстановление здоровья через поглощение энергии
- Процент от нанесённого урона
- Накопление энергии для мощного лечения
- Визуальные эффекты поглощения

### Комбо-эффекты

**Призыв + Лечение:**
- Дроны лечат союзников во время боя
- Максимальная поддержка команды
- Создание тактических преимуществ

**Множественные призывы:**
- Несколько дронов одновременно
- Различные типы дронов для разных целей
- Максимальный контроль территории
