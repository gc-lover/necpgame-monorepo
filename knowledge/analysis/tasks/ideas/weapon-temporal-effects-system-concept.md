# Система временных эффектов оружия (вампиризм, ускорение, замедление времени) - Концепция

## Обзор

Система временных эффектов создаёт тактические преимущества, где оружие может временно изменять характеристики игрока или цели. В мире Cyberpunk 2077 это отражает развитие кибернетических технологий и их влияние на боевые возможности.

## Лор в мире Cyberpunk 2077

### Контекст временных эффектов

В мире NECPGAME временные эффекты стали стандартом для тактических операций, особенно популярного среди Solo и корпоративных солдат. Вампиризм, ускорение и замедление времени создают уникальные тактические возможности.

**Как работают временные эффекты:**
- Оружие наносит урон и применяет временные эффекты
- Эффекты изменяют характеристики на ограниченное время
- Накопление эффектов при убийствах усиливает их
- Различные эффекты эффективны в разных ситуациях

### Технологическое обоснование

**Вампиризм:**
- Использует наноботы для извлечения биоматериала
- Восстановление здоровья через поглощение энергии
- Разработано корпорациями Biotechnica и Arasaka
- Популярно среди Solo для выживания

**Ускорение:**
- Использует нервные ускорители для увеличения скорости
- Временное усиление рефлексов и скорости реакции
- Разработано корпорациями Arasaka и Kang Tao
- Популярно среди Solo и Netrunner

**Замедление времени:**
- Использует квантовые технологии для замедления времени
- Временное искажение восприятия времени
- Разработано корпорациями Arasaka
- Популярно среди Solo для тактических преимуществ

## Типы временных эффектов

### Вампиризм

**Характеристики:**
- Восстановление 5-20% от нанесённого урона
- Увеличение регенерации при убийстве на 50-100%
- Накопление "крови" для мощной атаки
- Визуальные эффекты поглощения энергии

**Тактические применения:**
- Выживание в длительных боях
- Восстановление здоровья без медикаментов
- Накопление энергии для мощных атак
- Контроль территории через выживание

**Типы вампиризма:**
- **Стандартные:** Базовое восстановление здоровья
- **Усиленные:** Больше восстановление, быстрее накопление
- **Фокусные:** Меньше восстановление, больше накопление
- **Распределённые:** Восстановление для команды

### Ускорение

**Характеристики:**
- Увеличение скорости стрельбы на 50-100%
- Снижение времени перезарядки на 30-50%
- Увеличение скорости движения на 20-40%
- Накопление эффекта при убийствах

**Тактические применения:**
- Быстрые атаки для максимального урона
- Снижение времени перезарядки для непрерывного огня
- Увеличение мобильности для тактических манёвров
- Накопление эффекта для усиления

**Типы ускорения:**
- **Стандартные:** Базовое увеличение скорости
- **Усиленные:** Больше увеличение, дольше длительность
- **Фокусные:** Меньше радиус, больше эффект
- **Распределённые:** Ускорение для команды

### Замедление времени

**Характеристики:**
- Замедление движения цели на 30-70%
- Увеличение шанса крита на 20-50%
- Накопление эффекта при попаданиях
- Визуальные искажения времени

**Тактические применения:**
- Замедление целей для тактических преимуществ
- Увеличение точности через замедление
- Накопление эффекта для максимального контроля
- Создание тактических возможностей

**Типы замедления:**
- **Стандартные:** Базовое замедление движения
- **Усиленные:** Больше замедление, дольше длительность
- **Фокусные:** Меньше радиус, больше эффект
- **Распределённые:** Замедление для нескольких целей

## Механики накопления эффектов

### Накопление при убийствах

**Вампиризм:**
- Каждое убийство увеличивает накопление "крови"
- Накопленная "кровь" может быть использована для мощной атаки
- Максимальное накопление увеличивает эффективность
- Сброс накопления при смерти или перезарядке

**Ускорение:**
- Каждое убийство увеличивает скорость на 5-10%
- Накопление эффекта до максимума
- Максимальное накопление создаёт временное безумие скорости
- Сброс накопления при смерти или окончании эффекта

**Замедление:**
- Каждое попадание увеличивает замедление на 5-10%
- Накопление эффекта до полной остановки
- Максимальное накопление создаёт временную заморозку
- Сброс накопления при окончании эффекта

### Комбо-эффекты

**Вампиризм + Ускорение:**
- Быстрые атаки с восстановлением здоровья
- Максимальная эффективность в ближнем бою
- Создание тактических преимуществ

**Ускорение + Замедление:**
- Игрок ускоряется, враги замедляются
- Максимальное тактическое преимущество
- Создание временного доминирования

**Вампиризм + Замедление:**
- Медленные цели легче попадать
- Восстановление здоровья при каждом попадании
- Долгосрочный контроль территории
