# Специальные механики по типам оружия - Концепция

## Обзор

Система специальных механик по типам оружия создаёт уникальные игровые стили для каждого типа оружия. В мире Cyberpunk 2077 это отражает разнообразие технологий производства оружия и их применение в различных боевых ситуациях.

## Лор в мире Cyberpunk 2077

### Контекст специальных механик

В мире NECPGAME различные типы оружия имеют уникальные специальные механики, основанные на их конструктивных особенностях и технологиях производства. Эти механики создают разнообразие в боевых стилях и тактических возможностях.

**Как работают специальные механики:**
- Каждый тип оружия имеет уникальные характеристики
- Различные типы эффективны в разных ситуациях
- Специальные механики создают уникальные игровые стили
- Развитие механик по временным периодам (2020-2093)

### Технологическое обоснование

**Ближний бой:**
- Использует физические характеристики оружия
- Разработано корпорациями Arasaka и независимыми производителями
- Популярно среди Solo для тактических преимуществ
- Эффективно для ближнего боя

**Дальний бой:**
- Использует баллистические и энергетические технологии
- Разработано корпорациями Militech, Arasaka и Kang Tao
- Популярно среди Solo и Netrunner для тактических преимуществ
- Эффективно для дальнего боя

## Режущее оружие

### Катана

**Характеристики:**
- Высокий урон по незащищенным частям (+50-100%)
- Разрывание брони (снижение брони на 20-40%)
- Комбо-атаки (увеличение урона при последовательных атаках)
- Блокирование выстрелов (с имплантом, снижение урона на 50-80%)

**Тактические применения:**
- Быстрые режущие атаки для максимального урона
- Разрывание брони для увеличения эффективности
- Блокирование выстрелов для защиты
- Комбо-атаки для максимального урона

**Типы катаны:**
- **Стандартная:** Базовые характеристики
- **Энергетическая:** Энергетические эффекты, увеличение урона
- **Усиленная:** Увеличение урона и скорости
- **Легендарная:** Уникальные способности

### Ножи

**Характеристики:**
- Быстрые атаки (высокая скорость атак)
- Высокий урон по незащищенным частям (+30-60%)
- Можно метать (дальний бой)
- Застревание в теле (DoT урон)

**Тактические применения:**
- Быстрые атаки для максимального урона
- Метание для дальнего боя
- Застревание для DoT урона
- Комбо-атаки для максимальной эффективности

### Когти (импланты)

**Характеристики:**
- Быстрые атаки (очень высокая скорость атак)
- Разрывание брони (снижение брони на 30-50%)
- Можно цепляться за поверхности
- DoT от ран (кровотечение)

**Тактические применения:**
- Быстрые атаки для максимального урона
- Разрывание брони для увеличения эффективности
- Цепляние за поверхности для мобильности
- DoT от ран для долгосрочного контроля

## Автоматы

**Характеристики:**
- Высокая скорость стрельбы (600-900 выстрелов в минуту)
- Средний урон (баланс между уроном и скоростью)
- Контроль отдачи (снижение точности при длительной стрельбе)
- Различные режимы стрельбы (одиночные, очередь, автоматический)

**Тактические применения:**
- Высокая скорость стрельбы для максимального урона
- Контроль отдачи для точности
- Различные режимы для различных ситуаций
- Эффективность против множественных целей

**Типы автоматов:**
- **Стандартные:** Базовые характеристики
