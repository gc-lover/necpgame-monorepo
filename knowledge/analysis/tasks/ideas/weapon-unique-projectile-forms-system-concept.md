# Система уникальных форм проджектайлов для оружия - Концепция

## Обзор

Система уникальных форм проджектайлов создаёт разнообразие в боевых механиках, где оружие может стрелять различными формами снарядов, создавая уникальные тактические возможности для каждого типа оружия. В мире Cyberpunk 2077 это отражает развитие технологий баллистики и энергетических систем.

## Лор в мире Cyberpunk 2077

### Контекст форм проджектайлов

В мире NECPGAME различные формы проджектайлов стали стандартом для тактических операций, особенно популярного среди Solo, Netrunner и корпоративных солдат. Развитие технологий позволило создавать уникальные формы снарядов для различных тактических ситуаций.

**Как работают формы проджектайлов:**
- Оружие может стрелять различными формами снарядов
- Каждая форма имеет уникальные характеристики и тактические применения
- Различные формы эффективны в разных ситуациях
- Развитие форм по временным периодам (2020-2093)

### Технологическое обоснование

**Баллистические формы:**
- Используют баллистические технологии для создания различных форм
- Разработано корпорациями Militech и Arasaka
- Популярно среди Solo для тактических преимуществ
- Эффективно для контроля территории

**Энергетические формы:**
- Используют энергетические технологии для создания различных форм
- Разработано корпорациями Arasaka и Kang Tao
- Популярно среди Netrunner для тактических манёвров
- Эффективно для обхода укрытий

## Типы форм проджектайлов

### Точка (стандартная пуля)

**Характеристики:**
- Стандартная форма для большинства оружия
- Высокая точность и скорость
- Прямолинейная траектория
- Базовый урон

**Тактические применения:**
- Стандартная атака для большинства ситуаций
- Высокая точность для дальних дистанций
- Эффективна против одиночных целей

**Типы оружия:**
- Пистолеты, автоматы, винтовки, снайперские винтовки

### Линия (горизонтальная, вертикальная, диагональная)

**Характеристики:**
- Множественные снаряды в линию
- Увеличение шанса попадания
- Различные направления (горизонтальная, вертикальная, диагональная)
- Средний урон на снаряд

**Тактические применения:**
- Атака по группе врагов
- Увеличение шанса попадания
- Контроль территории через линию огня

**Типы оружия:**
- Пистолеты, автоматы, винтовки, ПП

### Веер (горизонтальный, вертикальный, круговой)

**Характеристики:**
- Множественные снаряды в веер
- Широкое рассеивание
- Различные направления (горизонтальный, вертикальный, круговой)
- Низкий урон на снаряд

**Тактические применения:**
- Атака по группе врагов
- Широкое рассеивание для контроля территории
- Эффективна в ближнем бою

**Типы оружия:**
- Дробовики, пистолеты, автоматы, ПП

### Спираль (спиральная траектория)

**Характеристики:**
- Снаряды движутся по спиральной траектории
- Увеличение шанса попадания
- Обход укрытий через спиральную траекторию
- Средний урон

**Тактические применения:**
- Обход укрытий через спиральную траекторию
- Увеличение шанса попадания
- Создание тактических преимуществ

**Типы оружия:**
- Пистолеты, автоматы, ПП, энергетическое оружие

### Волна (волнообразная траектория)

**Характеристики:**
- Снаряды движутся по волнообразной траектории
- Увеличение шанса попадания
- Обход укрытий через волнообразную траекторию
- Средний урон

**Тактические применения:**
- Обход укрытий через волнообразную траекторию
- Увеличение шанса попадания
- Создание тактических преимуществ

**Типы оружия:**
- Автоматы, энергетическое оружие

### Круг (круговое рассеивание)

**Характеристики:**
- Снаряды рассеиваются по кругу
- Широкое рассеивание
- Атака по всем направлениям
- Низкий урон на снаряд

**Тактические применения:**
- Атака по всем направлениям
- Защита от окружения
- Контроль территории через круговое рассеивание

**Типы оружия:**
- Дробовики, энергетическое оружие

### Квадрат (квадратный паттерн)

**Характеристики:**
- Снаряды рассеиваются по квадратному паттерну
- Контролируемое рассеивание
- Атака по квадратной области
- Средний урон на снаряд

**Тактические применения:**
- Атака по квадратной области
- Контролируемое рассеивание
- Контроль территории через квадратный паттерн

**Типы оружия:**
- Дробовики, энергетическое оружие

### Цепь (цепная реакция)

**Характеристики:**
- Снаряды создают цепную реакцию
- Урон распространяется между целями
- Увеличение урона при множественных целях
- Высокий урон

**Тактические применения:**
- Атака по группе врагов
- Увеличение урона при множественных целях
- Создание тактических преимуществ

**Типы оружия:**
- Энергетическое оружие, электрическое оружие

### Разделение (разделение на части)

**Характеристики:**
- Снаряды разделяются на части при попадании
- Увеличение урона при разделении

- Высокий урон

**Тактические применения:**
- Атака по нескольким целям
- Увеличение урона при разделении
- Создание тактических преимуществ

**Типы оружия:**
- Пусковые установки, энергетическое оружие

### Рикошеты (рикошетирующие снаряды)

**Характеристики:**
- Снаряды рикошетят от поверхностей
- Обход укрытий через рикошеты
- Увеличение шанса попадания
- Средний урон
