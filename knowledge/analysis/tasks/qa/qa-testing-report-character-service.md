### 1. API Endpoints

✅ **Accounts:**
- `GET /api/v1/accounts/{accountId}` - Получение аккаунта по ID
- `POST /api/v1/accounts` - Создание нового аккаунта

✅ **Characters:**
- `GET /api/v1/characters?account_id={accountId}` - Получение списка персонажей
- `GET /api/v1/characters/{characterId}` - Получение персонажа по ID
- `POST /api/v1/characters` - Создание нового персонажа
- `PUT /api/v1/characters/{characterId}` - Обновление персонажа
- `DELETE /api/v1/characters/{characterId}` - Удаление персонажа
- `GET /api/v1/characters/{characterId}/validate` - Проверка валидности персонажа
