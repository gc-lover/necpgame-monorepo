# Скрипты автоматизации

## autocommit.ps1 / autocommit.sh

Скрипт для автоматического коммита и отправки изменений в GitHub.

### Использование

#### Windows (PowerShell):
```powershell
.\scripts\autocommit.ps1 [сообщение коммита]
```

#### Linux/Mac (Bash):
```bash
chmod +x scripts/autocommit.sh
./scripts/autocommit.sh [сообщение коммита]
```

### Функции

1. **Автоматическое определение изменений** - проверяет наличие изменений в репозитории
2. **Умное сообщение коммита** - генерирует осмысленное сообщение на основе измененных файлов (если не указано явно)
3. **Автоматический commit и push** - добавляет все изменения, коммитит и отправляет в GitHub

### Примеры

```powershell
# Автоматическое сообщение коммита
.\scripts\autocommit.ps1

# С указанием сообщения
.\scripts\autocommit.ps1 "Добавлена новая API спецификация для боевой системы"
```

### Определение типа изменений

Скрипт автоматически определяет тип изменений:
- **Документация** - если изменены .md файлы
- **API спецификация** - если изменены .yaml/.yml файлы
- **Реализация** - если изменены .go/.java/.js/.ts/.py файлы
- **Обновление правил** - если изменены rules.mdc файлы
- **Обновление** - для остальных случаев

### Установка в репозитории

Скопируйте скрипт в каждый репозиторий:
- `.BRAIN/scripts/autocommit.ps1`
- `API-SWAGGER/scripts/autocommit.ps1`
- `BACK-JAVA/scripts/autocommit.ps1`
- `FRONT-WEB/scripts/autocommit.ps1`
- `GAME-CPP-FOR-UE5/scripts/autocommit.ps1`

Или используйте скрипт из корневой директории, указав путь к репозиторию.
