========================================
ПРАВИЛЬНОЕ ДОБАВЛЕНИЕ ВИДЖЕТОВ
========================================

ПРОБЛЕМА:
Не получается добавить несколько Text виджетов

РЕШЕНИЕ:
========================================

ВАРИАНТ 1: Использовать Canvas Panel (рекомендуется)
-----------------------------------------------------

1. Откройте WBP_TechDemoHUD

2. Проверьте Hierarchy (слева):
   - Должен быть "Canvas Panel" (или "Canvas")
   - Если его нет - добавьте из палитры:
     * Палитра -> ПАНЕЛЬ (Panel) -> Canvas Panel
     * Перетащите на Canvas

3. Добавляйте Text виджеты:
   - Выделите Canvas Panel в Hierarchy
   - Или кликните на Canvas (центральная область)
   - Из палитры перетащите Text виджет
   - Повторите 4 раза

4. Установите имена:
   - FPSText
   - PingText
   - ConnectionStatusText
   - PlayerCountText

ВАРИАНТ 2: Использовать Vertical Box
--------------------------------------

1. Добавьте Vertical Box из палитры:
   - Палитра -> ПАНЕЛЬ (Panel) -> Vertical Box
   - Перетащите на Canvas

2. Выделите Vertical Box

3. Добавляйте Text виджеты:
   - Перетащите Text виджеты НА Vertical Box
   - Они автоматически расположатся вертикально

4. Установите имена (как выше)

ВАРИАНТ 3: Добавлять через Hierarchy
--------------------------------------

1. В Hierarchy (слева) найдите Canvas Panel
2. Правый клик на Canvas Panel
3. Add Widget -> Text
4. Повторите 4 раза
5. Установите имена в Details

========================================
ВАЖНО:
========================================

- Виджеты должны быть ДОЧЕРНИМИ элементами Canvas Panel
- Не пытайтесь добавлять виджеты на Root напрямую
- Canvas Panel - это контейнер для виджетов
