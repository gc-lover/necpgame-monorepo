========================================
ДИАГНОСТИКА: HUD НЕ ПОЯВЛЯЕТСЯ
========================================

ВОЗМОЖНЫЕ ПРИЧИНЫ:
========================================

1. HUDWidgetClass не установлен в BP_TechDemoHUD
-------------------------------------------------
Проверьте:
- Откройте BP_TechDemoHUD
- В Details панели найдите "HUD Widget Class"
- Должно быть установлено: WBP_TechDemoHUD
- Если пусто - установите и нажмите Compile

2. PlayerController не установлен
----------------------------------
Проверьте:
- Откройте BP_TechDemoGameMode
- В Details панели найдите "Player Controller Class"
- Должно быть: TechDemoPlayerController
- Если пусто - установите и нажмите Compile

3. GameMode не установлен в World Settings
------------------------------------------
Проверьте:
- World Settings -> Game Mode -> GameMode Override
- Должно быть: BP_TechDemoGameMode

4. Виджет не виден (позиция/размер)
------------------------------------
Проверьте в WBP_TechDemoHUD:
- Text виджеты должны быть на Canvas
- Позиция должна быть в пределах экрана (например, X=50, Y=50)
- Размер текста должен быть достаточным (например, Font Size = 20)

5. Виджет прозрачный
--------------------
Проверьте в WBP_TechDemoHUD:
- Text виджеты -> Details -> Color and Opacity
- Должно быть не прозрачным (Alpha = 1.0)

========================================
ПОШАГОВАЯ ПРОВЕРКА:
========================================

ШАГ 1: Проверьте BP_TechDemoHUD
--------------------------------
1. Откройте BP_TechDemoHUD
2. В Details (справа) найдите "HUD Widget Class"
3. Должно быть: WBP_TechDemoHUD
4. Если пусто - выберите WBP_TechDemoHUD
5. Compile -> Save

ШАГ 2: Проверьте BP_TechDemoGameMode
-------------------------------------
1. Откройте BP_TechDemoGameMode
2. В Details найдите "Player Controller Class"
3. Должно быть: TechDemoPlayerController
4. Если пусто - выберите TechDemoPlayerController
5. Compile -> Save

ШАГ 3: Проверьте World Settings
--------------------------------
1. Edit -> Project Settings -> Maps & Modes
2. Game Mode -> GameMode Override: BP_TechDemoGameMode
3. Сохраните

ШАГ 4: Проверьте позицию виджетов
----------------------------------
1. Откройте WBP_TechDemoHUD
2. Выделите каждый Text виджет
3. В Details -> Slot (Canvas Panel Slot):
   - Position X: 50 (или другое значение > 0)
   - Position Y: 50, 100, 150, 200 (для каждого виджета)
4. Size -> Auto: включено
5. Сохраните

ШАГ 5: Проверьте цвет текста
-----------------------------
1. В WBP_TechDemoHUD выделите Text виджет
2. Details -> Appearance -> Color and Opacity
3. Установите белый цвет (R=1, G=1, B=1, A=1)
4. Font -> Size: 20 или больше
5. Сохраните

========================================
БЫСТРАЯ ПРОВЕРКА В OUTPUT LOG:
========================================

При запуске Play проверьте Output Log:
- Window -> Developer Tools -> Output Log
- Ищите ошибки связанные с HUD, Widget, BindWidget
