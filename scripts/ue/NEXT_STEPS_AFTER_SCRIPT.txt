OK Blueprint файлы созданы успешно:
   - WBP_TechDemoHUD
   - BP_TechDemoHUD

ШАГ 1: Настроить WBP_TechDemoHUD (Widget Blueprint)
----------------------------------------------------
1. В Content Browser найдите: Content/UI/WBP_TechDemoHUD
2. Двойной клик для открытия
3. В редакторе Widget:
   a) Class Settings (иконка шестерёнки вверху) → Parent Class → TechDemoHUDWidget
   b) Из Palette (слева) перетащите 4 Text виджета на Canvas
   c) Расположите их в левом верхнем углу (вертикально)
   d) Для каждого Text виджета:
      - Выделите виджет
      - В Details панели (справа) → Name:
        • Первый: FPSText
        • Второй: PingText
        • Третий: ConnectionStatusText
        • Четвёртый: PlayerCountText
4. Сохраните (Ctrl+S)
5. Закройте редактор Widget

ШАГ 2: Настроить BP_TechDemoHUD (HUD Blueprint)
------------------------------------------------
1. В Content Browser найдите: Content/Blueprints/BP_TechDemoHUD
2. Двойной клик для открытия
3. В редакторе Blueprint:
   a) Class Settings → Parent Class → TechDemoHUD
   b) В Details панели (справа) → HUD Widget Class → WBP_TechDemoHUD
4. Нажмите Compile (зелёная кнопка вверху)
5. Сохраните (Ctrl+S)
6. Закройте редактор Blueprint

ШАГ 3: Настроить BP_TechDemoGameMode (GameMode Blueprint)
-----------------------------------------------------------
1. В Content Browser найдите: Content/Blueprints/BP_TechDemoGameMode
2. Двойной клик для открытия
3. В редакторе Blueprint:
   a) Class Settings → Parent Class → MyGameMode
   b) В Details панели (справа) → HUD Class → BP_TechDemoHUD
4. Нажмите Compile (зелёная кнопка вверху)
5. Сохраните (Ctrl+S)
6. Закройте редактор Blueprint

ШАГ 4: Установить GameMode в World Settings
---------------------------------------------
1. В редакторе уровня:
   - Edit → Project Settings → Maps & Modes
   ИЛИ
   - В World Outliner → правый клик на уровень → World Settings
2. В World Settings панели (справа):
   - Game Mode → GameMode Override → BP_TechDemoGameMode
3. Сохраните уровень (Ctrl+S)

ШАГ 5: ТЕСТИРОВАНИЕ
--------------------
1. Нажмите Play (зелёная кнопка Play вверху)
2. В левом верхнем углу должны появиться:
   - FPS: [число]
   - Ping: [число] ms
   - Status: Connected/Disconnected
   - Players: [число]

========================================
ВАЖНО: Проверьте что все Blueprint'ы скомпилированы
========================================
В Content Browser должны быть зелёные галочки на всех Blueprint'ах.
Если есть жёлтые предупреждения - откройте Blueprint и нажмите Compile.
