# Очереди реализации (Frontend)

Единый пул статусов для frontend: `not_started`, `in_progress`, `completed`, `failed`.

- Очереди расположены в каталоге `frontend/` (по одному файлу на статус).  
- Лимит каждого файла — 500 строк; при переполнении создавайте продолжения (`<status>_0001.md`, `<status>_0002.md`).  
- При смене статуса переносите запись между файлами, избегая дублирования.  
- Следуйте примерам, указанным внутри файлов.
