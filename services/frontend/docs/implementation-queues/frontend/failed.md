# Frontend очередь — статус `failed`

- Лимит файла: ≤ 500 строк. При превышении создайте `failed_0001.md`, `failed_0002.md`.
- Формат записи:

```markdown
- **API:** api/v1/gameplay/combat/combat-session.yaml (API-TASK-206)  
  **Остановлено:** 2025-11-09 01:00 — Frontend Agent  
  **Причина:** Требуется доработка WebSocket контракта.  
  **Следующий шаг:** Вернуть в `not-started.md` после обновления API.
```

- После восстановления работы удалите запись и перенесите в актуальный файл статуса.
