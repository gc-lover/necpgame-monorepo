# Frontend очередь — статус `in_progress`

- Лимит файла: ≤ 500 строк. При превышении создайте `in-progress_0001.md`, `in-progress_0002.md`.
- Формат записи:

```markdown
- **API:** api/v1/gameplay/progression/progression-core.yaml (API-TASK-205)  
  **Старт:** 2025-11-09 00:53 — Frontend Agent  
  **Примечание:** Настраиваем Zustand store и страницы прогрессии.

- **API:** api/v1/economy/auction-house/auction-mechanics.yaml (API-TASK-109)  
  **Старт:** 2025-11-09 17:32 — Frontend Agent  
  **Примечание:** Генерируем Orval-клиент и создаём UI для лотов, ставок и buyout.
```

- После завершения работы перенесите запись в `completed.md` или `failed.md`.
