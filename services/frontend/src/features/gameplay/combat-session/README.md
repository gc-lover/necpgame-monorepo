## Combat Session

Интерактивная страница управления текстовой боевой сессией. Использует API `api/v1/gameplay/combat/combat-session.yaml`:

- создание новой сессии с динамическим набором участников;
- просмотр состояния сессии, списка участников и текущего хода;
- лента событий с автообновлением;
- действия: нанесение урона, переход к следующему ходу, завершение боя.

Компоненты построены на MUI, данные получаются через React Query хуки, сгенерированные Orval (`gameplay/combat-session/*`). Размер файлов < 400 строк, состояние локальное, сетевые ошибки отображаются через `Alert`.
