# Contracts Feature
Контракты между игроками: escrow, collateral, шаблоны и арбитраж.

**OpenAPI:** contracts.yaml | **Роут:** /game/contracts

**⭐ UI на shared библиотеке (`GameLayout`, `CompactCard`, `CyberpunkButton`).**

## Функционал
- Просмотр активных контрактов (тип, статус, collateral, escrow)
- Фильтры по типу и статусу
- Шаблоны контрактов (auto-execution, теги)
- Escrow мониторинг (released, disputed, условия)
- Споры и арбитры, количество доказательств
- Действия: создать контракт, заполнить шаблон, открыть спор

## Компоненты
- **ContractCard** — сводка контракта (участники, залог, срок)
- **ContractTemplateCard** — описание шаблона с тегами
- **EscrowStatusCard** — прогресс выплат и спорных сумм
- **DisputeCard** — статус спора и назначенный арбитр
- **ContractsPage** — SPA-страница (сеткой 380px | flex | 320px)

## Нотации
- Шрифты 0.65–0.875rem, киберпанк цвета, компактная верстка
- Тесты на отображение ключевых данных (не запускались по инструкции)

**Соответствие:** SPA, 1 экран, MUI, React Router, shared/ui.
