# D&D Checks Feature
Система D&D проверок - кубики, модификаторы, критические (Baldur's Gate 3 / D&D 5e).

**OpenAPI:** dnd-checks.yaml | **Роут:** /game/dnd-checks

## Функционал
- **D20 система:** d20 + Атрибутный Мод + Навыковый Бонус + Ситуационные ≥ DC
- **Критические:** 1 = автофейл, 20 = автоуспех
- **Advantage/Disadvantage:** 2d20, берем лучший/худший
- **Атрибутный мод:** floor((ATTR - 10) / 2)
- **Навыковые бонусы:** Novice (+1) → Expert (+3) → Legendary (+5)
- **DC шкала:** Легкая (10), Средняя (15), Сложная (20), Очень сложная (25), Экстремальная (30+)

## Компоненты
- **DiceRollDisplay** - визуальное отображение результата броска с breakdown
- **DnDChecksPage** - интерфейс для проверок с настройками

## Механики
- Формула: `d20 + АтрибутныйМод + НавыковыйБонус + Ситуационные >= DC`
- Критические успехи/провалы
- Advantage: бросок 2d20, берем лучший
- Disadvantage: бросок 2d20, берем худший

**Соответствие:** Полная реализация на основе OpenAPI, React Query хуки, MUI компоненты.
