# Events 2020-2040 Feature
Мировые события эпохи 2020-2040 "Разрушение и восстановление" (война Arasaka vs Militech).

**OpenAPI:** events-2020-2040.yaml | **Роут:** /game/events-2020-2040

## Функционал
- **Эпоха:** 2020-2040 (война Arasaka vs Militech, ядерные удары)
- **D&D генератор событий (d100):**
  - 01-15: Рад-зоны
  - 16-30: Спасработы
  - 31-45: Разбор руин
  - 46-60: Очаги боёв
  - 61-75: Сетевые "дыры"
  - 76-90: Переговоры
  - 91-00: Чёрный заслон
- **DC скейлинг:**
  - Социальные: 14/18/22
  - Тех/Хак: 18/22/25
  - Боевые: 16/20/24
- **AI слайдеры фракций:**
  - Агрессия: 4
  - Экспансия: 3
  - Дипломатия: 2
  - Шпионаж: 4
- **Экономика:** Ресурсы ×0.7, волатильность высокая, пошлины +1..+2
- **Технологии:** Импланты развивающиеся, сеть фрагментированная

## Компоненты
- **WorldEventCard** - карточка мирового события эпохи
- **Events2020_2040Page** - генератор событий, активные события, параметры эпохи

## Механики
- Генерация случайных событий (d100)
- Активные события эпохи
- Настройки эпохи (DC, AI, экономика)
- Последствия и квестовые хуки

**Соответствие:** Полная реализация на основе OpenAPI, React Query хуки, MUI компоненты.
