# Events 2040-2060 Feature
Мировые события эпохи 2040-2060 "Time of the Red" (восстановление, новые технологии, киберпсихоз).

**OpenAPI:** events-2040-2060.yaml | **Роут:** /game/events-2040-2060

## Функционал
- **Эпоха:** 2040-2060 (Time of the Red - восстановление, рост технологий)
- **D&D генератор событий (d100):**
  - 01-15: Срыв психостабильности (киберпсихоз)
  - 16-30: Брейндэнс-утечки
  - 31-45: Уличные бунты
  - 46-60: Подпольные клиники
  - 61-75: Сетевой прорыв
  - 76-90: Реставрация районов
  - 91-00: Новые субкультуры
- **DC скейлинг:**
  - Социальные: 15/19/22
  - Тех/Хак: 18/22/24
  - Боевые: 14/18/22
- **AI слайдеры фракций:** Агрессия/Экспансия/Дипломатия/Шпионаж: 3 (сбалансированно)
- **Экономика:** Ресурсы ×0.85, волатильность средняя/высокая, пошлины +1
- **Технологии:** Импланты продвинутые (рост киберпсихоза), сеть восстанавливается

## Компоненты
- **RedEraEventCard** - карточка события Time of the Red (тёмно-красный дизайн)
- **Events2040_2060Page** - генератор событий, активные события, параметры эпохи

## Механики
- Генерация случайных событий (d100)
- Активные события эпохи
- Настройки эпохи (DC, AI, экономика)
- Специфика: киберпсихоз, брейндэнс, субкультуры

**Соответствие:** Полная реализация на основе OpenAPI, React Query хуки, MUI компоненты.
