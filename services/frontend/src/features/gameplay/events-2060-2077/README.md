# Events 2060-2077 Feature
Мировые события эпохи 2060-2077 "Преддверие современности" (пик корпоративного контроля, Night City независимость).

**OpenAPI:** events-2060-2077.yaml | **Роут:** /game/events-2060-2077

## Функционал
- **Эпоха:** 2060-2077 (пик корпораций, Night City становится независимым)
- **D&D генератор событий (d100):**
  - 01-15: Корпоративный аудит
  - 16-30: Подполье сопротивления
  - 31-45: Нейроинтерфейсные сбои
  - 46-60: Высокий протокол безопасности
  - 61-75: Политическая интрига
  - 76-90: Космическая логистика
  - 91-00: Независимость Night City
- **DC скейлинг (топовый):**
  - Социальные: 16/20/24
  - Тех/Хак: 20/24/28
  - Боевые: 16/20/24
- **AI слайдеры фракций:**
  - Агрессия: 3
  - Экспансия: 4
  - Дипломатия: 4
  - Шпионаж: 5 (шпионаж на пике!)
- **Экономика:** Ресурсы ×1.1, волатильность средняя, пошлины +2 (корп.контроль)
- **Технологии:** Импланты топовые, сеть глобальная, безопасность очень высокая

## Компоненты
- **ModernEraEventCard** - карточка события современности (золотой дизайн)
- **Events2060_2077Page** - генератор событий, активные события, параметры эпохи

## Механики
- Генерация случайных событий (d100)
- Активные события эпохи
- Настройки эпохи (DC, AI, экономика)
- Специфика: корп.контроль, космос, Night City независимость

**Соответствие:** Полная реализация на основе OpenAPI, React Query хуки, MUI компоненты.
