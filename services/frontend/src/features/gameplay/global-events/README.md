# Global Events Feature
Глобальные события мира - timeline 2020-2093 (Kenshi / RimWorld style живой мир).

**OpenAPI:** global-events.yaml | **Роут:** /game/global-events

## Функционал
- **Timeline:** 2020-2030, 2030-2045, 2045-2060, 2060-2077, 2077, 2078-2093
- **Типы:** Political, Economic, Technological, Environmental, Social
- **Активные события:** зависят от текущего игрового времени в лиге
- **Влияние:**
  - Региональные модификаторы
  - Реакции фракций
  - Связанные квесты
  - Модификаторы геймплея
- **Детали события:** лор Cyberpunk, описание, влияние, триггеры

## Компоненты
- **GlobalEventCard** - карточка события с типом, эпохой, годами, статусом (активно/неактивно)
- **GlobalEventsPage** - timeline событий, фильтры по эпохам и типам

## Механики
- Мир живет сам по себе (Kenshi style)
- События происходят независимо от игрока
- Влияние на экономику, фракции, регионы
- Связь с квестами и репутацией

**Соответствие:** Полная реализация на основе OpenAPI, React Query хуки, MUI компоненты.
