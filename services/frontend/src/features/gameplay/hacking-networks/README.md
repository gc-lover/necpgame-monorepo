# Hacking Networks Feature
Система взлома сетей - локальные/корпоративные/городские сети (CP2077 / Watch Dogs).

**OpenAPI:** hacking-networks.yaml | **Роут:** /game/hacking-networks

## Функционал
- **Структура сетей:**
  - Простые: устройства связаны, взлом одного = доступ к сети
  - Сложные: иерархическая/графовая структура, узлы и связи
  - Изолированные: требуют физического подключения
- **Типы сетей:** локальные, корпоративные, городские, персональные
- **Узлы:** камеры, двери, роутеры, серверы
- **Методы взлома:** brute force, stealth, daemon
- **Защита:** ICE (Intrusion Countermeasures Electronics), алерты, trace risk
- **Уровни доступа:** read, control, admin

## Компоненты
- **NetworkNodeCard** - карточка узла сети (камера/дверь/роутер/сервер)
- **HackingNetworksPage** - визуализация сети, узлы, методы взлома, защита

## Механики
- Граф сети (узлы и связи)
- Security level (1-5)
- ICE активация при взломе
- Trace risk (риск отследить)
- Время взлома
- Compromised nodes

**Соответствие:** Полная реализация на основе OpenAPI, React Query хуки, MUI компоненты.
