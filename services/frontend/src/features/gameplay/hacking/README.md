# Hacking Feature
Система хакерства - типы и применение (Cyberpunk 2077 / Watch Dogs style).

**OpenAPI:** hacking-types.yaml | **Роут:** /game/hacking

## Функционал
- **Взлом врагов:** отключение имплантов, контроль, урон, перегрев
- **Взлом устройств:** камеры, двери, роботы, транспорт (доступно всем классам)
- **Взлом инфраструктуры:** городские системы, сети (требует Netrunner)
- **Система перегрева:** сложные демоны разогревают кибердек
- **Quickhacks (демоны):** 5 тиров, RAM cost, heat generation, кулдауны

## Компоненты
- **DaemonCard** - карточка quickhack с тиром, RAM, heat
- **HeatMeter** - индикатор перегрева кибердека с охлаждением
- **HackingPage** - 3 вкладки (враги/устройства/инфраструктура)

## Механики
- Перегрев блокирует использование quickhacks
- Автоматическое охлаждение (cooling_rate)
- Зависимость от класса: Netrunner - все типы, другие - базовые
- RAM cost и heat generation для баланса

**Соответствие:** Полная реализация на основе OpenAPI, React Query хуки, MUI компоненты.
