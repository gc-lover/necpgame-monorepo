# Investments Feature
Инвестиционная система: портфель, ROI, дивиденды, риск-аналитика.

**OpenAPI:** investments.yaml | **Роут:** /game/investments

**⭐ UI построен на shared библиотеке (`GameLayout`, `CompactCard`, `CyberpunkButton`).**

## Функционал
- Загрузка инвестиционных возможностей (5 типов)
- Фильтры по типу и уровню риска
- Управление портфелем: стоимость, ROI, прибыль
- Анализ распределения рисков
- Календарь дивидендов и выплат
- Мониторинг прогресса финансирования и доходности

## Компоненты
- **InvestmentsPage** — SPA (сеткой 380px | flex | 320px), фильтры, алерт, списки
- **InvestmentCard** — детальная карточка возможности (ROI, риск, прогресс)
- **PortfolioOverviewCard** — ключевые метрики портфеля
- **RiskDistributionCard** — распределение капиталов по рискам
- **DividendScheduleCard** — ближайшие выплаты

## Особенности
- Шрифты 0.65–0.875rem, киберпанк-палитра из `cyberpunkTokens`
- Тесты на отображение ключевых данных (написаны, но не запускались по требованию)
- Подготовлено к интеграции с `investments-api` (React Query hooks)

**Соответствие:** SPA, компактный UI, shared/ui, MUI, темный киберпанк стиль.
