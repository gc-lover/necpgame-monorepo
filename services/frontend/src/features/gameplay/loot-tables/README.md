# Loot Tables Feature
Система лут-таблиц - генерация добычи с формулами вероятностей (Tarkov / Diablo).

**OpenAPI:** loot-tables.yaml | **Роут:** /game/loot-tables

## Функционал
- **Источники лута:** квесты, враги, контейнеры, события, боссы
- **Тиры:** t1-t5 (влияют на качество лута)
- **Редкость:** Common, Uncommon, Rare, Epic, Legendary, Unique
- **Генерация лута:** учитывает уровень источника, зону, удачу персонажа
- **Модификаторы:**
  - Уровень источника (враг, босс)
  - Зона (опасность)
  - Удача персонажа
  - События мира
  - Сложность
  - Размер группы
- **Вероятности:** формулы для расчета шансов выпадения

## Компоненты
- **LootTableCard** - карточка лут-таблицы с источником, тиром
- **LootTablesPage** - список таблиц, фильтры, генерация лута

## Механики
- Формулы вероятностей
- Модификаторы влияют на шансы
- Breakdown по редкости
- Total value расчет

**Соответствие:** Полная реализация на основе OpenAPI, React Query хуки, MUI компоненты.
