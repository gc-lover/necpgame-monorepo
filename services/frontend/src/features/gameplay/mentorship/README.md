# Mentorship Feature
Система наставничества - опытные игроки помогают новичкам (Mentor ↔ Mentee).

**OpenAPI:** mentorship.yaml | **Роут:** /game/mentorship

## Функционал
- **Роли:**
  - Mentor (опытный игрок) - обучает новичков
  - Mentee (новичок) - получает помощь
- **Бонусы для Mentee:**
  - XP boost (ускоренный опыт)
  - Skill bonuses (бонусы к навыкам)
  - Rewards (награды)
- **Бонусы для Mentor:**
  - Mentorship XP
  - Rewards за прогресс ученика
  - Репутация
- **Прогрессия:**
  - Уровни наставничества
  - Достижения
  - Репутация системы
- **Обязательства:**
  - Времени (минимальное время онлайн)
  - Помощи (ответы на вопросы)
  - Обучения (передача знаний)

## Компоненты
- **MentorCard** - карточка ментора (имя, уровень, специализация, рейтинг, количество учеников)
- **MentorshipPage** - доступные менторы, система запросов

## Механики
- Поиск ментора
- Запрос наставничества
- Принятие/отклонение запросов
- Прогресс наставничества
- Завершение программы

**Соответствие:** Полная реализация на основе OpenAPI, React Query хуки, MUI компоненты.
