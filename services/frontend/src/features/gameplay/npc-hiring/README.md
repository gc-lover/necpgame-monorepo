# NPC Hiring Feature
Система найма NPC - bodyguards, mercs, vendors, specialists (RimWorld / Kenshi).

**OpenAPI:** npc-hiring.yaml | **Роут:** /game/npc-hiring

## Функционал
- **Типы NPC:**
  - Боевые: bodyguards, mercenaries
  - Торговцы: vendors (магазины)
  - Специалисты: ripperdocs, fixers, netrunners
- **Процесс:** поиск → переговоры → контракт → управление
- **Контракты:** temporary/permanent, длительность (дни), зарплата
- **Ограничения:** слоты найма, зарплата, лояльность
- **Интеграция:** репутация (требования для найма), экономика (зарплата)

## Компоненты
- **HireableNPCCard** - карточка NPC с тиром, навыками, стоимостью, кнопкой найма
- **NPCHiringPage** - список доступных NPC, активные контракты, фильтры

## Механики
- Зависимость от репутации
- Тиры NPC (1-5)
- Навыки NPC
- Ежедневная зарплата
- Лояльность и управление

**Соответствие:** Полная реализация на основе OpenAPI, React Query хуки, MUI компоненты.
