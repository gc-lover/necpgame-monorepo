# Resources Catalog Feature
Каталог ресурсов и материалов - сырьё, компоненты, данные (крафт и торговля).

**OpenAPI:** resources-catalog.yaml | **Роут:** /game/resources-catalog

## Функционал
- **Категории ресурсов:**
  - Raw Materials (сырьё) - базовые материалы
  - Processed (обработанные) - переработанное сырьё
  - Components (компоненты) - части для крафта
  - Data (данные) - информационные ресурсы
  - Special (специальные) - уникальные ресурсы
- **Tier:** 1-5 (качество ресурса)
- **Rarity:** Common → Uncommon → Rare → Epic → Legendary
- **Параметры:**
  - Base value (базовая цена)
  - Stack size (размер стака)
  - Weight (вес)
  - Sources (источники получения)
  - Uses (применение в крафте)
- **Интеграция:**
  - Крафт (используются в рецептах)
  - Торговля (продажа/покупка)
  - Квесты (требования для заданий)

## Компоненты
- **ResourceCard** - карточка ресурса (название, tier, категория, rarity, цена, stack, источники)
- **ResourcesCatalogPage** - каталог ресурсов, фильтры по категории/tier/rarity

## Механики
- Фильтрация по категории
- Сортировка по tier/rarity
- Просмотр источников получения
- Просмотр применения в крафте
- Информация о ценности

**Соответствие:** Полная реализация на основе OpenAPI, React Query хуки, MUI компоненты.
