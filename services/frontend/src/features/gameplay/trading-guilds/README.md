# Trading Guilds Feature
Торговые гильдии - создание, управление, казна, репутация (EVE Online / WOW механики).

**OpenAPI:** trading-guilds.yaml | **Роут:** /game/trading-guilds

## Функционал
- **Типы гильдий:**
  - MERCHANT (торговля товарами)
  - CRAFTSMAN (крафт и продажа изделий)
  - TRANSPORT (логистика и перевозки)
  - FINANCIAL (инвестиции и кредиты)
  - MIXED (универсальные гильдии)
- **Роли в гильдии:**
  - Guild Master (глава гильдии)
  - Treasurer (казначей)
  - Merchant (торговец)
  - Trader (трейдер)
- **Уровни гильдии:** 1-5
- **Казна гильдии:** общий капитал, распределение прибыли
- **Торговые квоты:** лимиты на торговлю
- **Эксклюзивные маршруты:** специальные торговые пути
- **Guild halls:** здания гильдии с апгрейдами
- **Межгильдийная торговля:** торговля между гильдиями
- **Репутация:** система репутации гильдии

## Компоненты
- **TradingGuildCard** - карточка торговой гильдии (тип, уровень, члены, казна, репутация)
- **TradingGuildsPage** - список гильдий, фильтры по типу/уровню/региону

## Механики
- Создание гильдий (минимальный капитал)
- Вступление/выход из гильдии
- Управление ролями
- Казна и распределение прибыли
- Торговые операции
- Репутация и уровни

**Соответствие:** Полная реализация на основе OpenAPI, React Query хуки, MUI компоненты.
