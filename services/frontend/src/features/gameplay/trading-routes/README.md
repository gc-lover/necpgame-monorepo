# Trading Routes Feature
Глобальные торговые маршруты - межконтинентальная торговля (EVE Online / KENSHI механики).

**OpenAPI:** trading-routes.yaml | **Роут:** /game/trading-routes

## Функционал
- **Торговые узлы:**
  - Tier 1 (Мегаполисы): Night City, Токио, Лондон, Шанхай, Москва
  - Tier 2 (Региональные центры): 20+ городов
- **Маршруты:**
  - Trans-Pacific (Азия ↔ Америка)
  - Euro-Express (Европа)
  - American Run (Америка)
  - Silk Road 2.0 (Азия)
  - African Network (Африка)
- **Товары:** Кибер-импланты, электроника, braindance, оружие, наркотики
- **Время доставки:** 48-96 часов игрового времени
- **Прибыль:** 10-40% (зависит от Trading skill и риска)
- **Риски:** Пираты, таможня, картели, корп-проверки, банды
- **Механики:**
  - Покупка товаров в стартовом узле
  - Доставка по маршруту
  - Продажа в конечном узле
  - Случайные события (риски)
  - Страхование груза
  - Эскорт (наёмники)

## Компоненты
- **TradeRouteCard** - карточка торгового маршрута (откуда→куда, расстояние, время, прибыль, риск)
- **TradingRoutesPage** - список маршрутов, узлы, товары, риски

## Механики
- Межконтинентальная торговля
- Риск vs прибыль
- Время доставки
- Случайные события на маршруте
- Trading skill влияет на прибыль

**Соответствие:** Полная реализация на основе OpenAPI, React Query хуки, MUI компоненты.
