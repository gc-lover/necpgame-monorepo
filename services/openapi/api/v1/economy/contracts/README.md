## Economy Contracts API

- `contracts.yaml` — основные эндпоинты жизненного цикла контрактов, escrow, споров и аналитики.
- `contracts-models.yaml` — базовые модели контрактов, участников, эскроу, таймлайна и споров.
- `contracts-models-operations.yaml` — запросы/ответы, события Kafka `economy.contracts.*`, аналитические структуры.

Каждый файл удерживает объём ≤ 400 строк, соблюдая правила репозитория.
