## Описание

API для управления инвентарем персонажа в текстовой версии NECPGAME.

Реализует:
- Просмотр инвентаря и экипировки
- Экипировка предметов (оружие, броня, импланты)
- Снятие экипированных предметов
- Использование предметов (медикаменты, расходники)
- Управление весом и лимитом инвентаря
- Выбрасывание предметов

## API Файлы

### inventory.yaml
**Описание:** Основной API для работы с инвентарем персонажа.

**Endpoints:**
- `GET /inventory` - Получить инвентарь персонажа
- `GET /inventory/equipment` - Получить экипировку персонажа
- `POST /inventory/equip` - Экипировать предмет
- `POST /inventory/unequip` - Снять экипированный предмет
- `POST /inventory/use` - Использовать предмет
- `DELETE /inventory/drop` - Выбросить предмет

**Модели:**
- `InventoryResponse` - Ответ с инвентарем
- `InventoryItem` - Предмет в инвентаре
- `EquipmentSlot` - Слот экипировки
- `ItemCategory` - Категория предмета (enum)
- `EquipRequest` - Запрос экипировки
- `UseItemRequest` - Запрос использования предмета

---

## Связанные API

- `api/v1/game/start.yaml` - Начальные предметы персонажа
- `api/v1/characters/` - Характеристики персонажа (влияют на инвентарь)

---

## Примечания

- Все endpoints требуют аутентификации (Bearer Token)
- Инвентарь имеет ограничение по весу
- Предметы разделены по категориям
- Некоторые предметы stackable (складываются)
- Экипировка влияет на характеристики персонажа
