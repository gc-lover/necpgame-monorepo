# Gameplay Actions API
**Источник:** `.BRAIN/05-technical/ui-main-game.md` (v1.1.0)  
**Task ID:** API-TASK-034  
**Версия API:** v1.0.0

## Описание
API для выполнения игровых действий в локациях.

## Endpoints
- `POST /gameplay/actions/explore` - Осмотреться
- `POST /gameplay/actions/rest` - Отдохнуть
- `POST /gameplay/actions/use` - Использовать объект
- `POST /gameplay/actions/hack` - Хакнуть систему
