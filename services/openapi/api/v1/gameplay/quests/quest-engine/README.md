## Quest Engine API

- `quest-engine.yaml` — основной контракт движка квестов: старт, прогресс, диалоги, завершение, отказ и симуляция шаблонов.
- `quest-engine-models.yaml` — базовые схемы: квестовые инстансы, прогресс, диалоговые узлы, условия, skill checks, награды, аудит.
- `quest-engine-models-operations.yaml` — структуры запросов/ответов, Kafka события `quest.*`, подписки на внешние события и результаты симуляций.

Каждый файл ≤ 400 строк; структура удовлетворяет правилу разбиения спецификаций.
