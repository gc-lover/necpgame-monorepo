Random Events Extended API
==========================

Структура спецификации разбита на несколько файлов, чтобы соблюдать лимит в 400 строк на файл и облегчить навигацию:

- `random-events.yaml` — основной файл OpenAPI (метаданные, теги, ссылки на пути и схемы).
- `paths/` — отдельные файлы для каждого endpoint.
- `schemas/index.yaml` — набор схем данных, используемых спецификацией.

Все общие компоненты (ответы, пагинация, безопасность) подключаются из `api/v1/shared/common/`.
