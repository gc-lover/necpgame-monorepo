# Events API
**Источник:** `.BRAIN/05-technical/mvp-content/events.json`  
**Task ID:** API-TASK-036  
**Версия API:** v1.0.0

## Описание
API для системы случайных событий в игре.

## Endpoints
- `GET /events/random` - Получить случайное событие
- `POST /events/{eventId}/respond` - Ответить на событие
- `GET /events/active` - Активные события
