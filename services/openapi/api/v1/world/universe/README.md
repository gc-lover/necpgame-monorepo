## Universe Timeline API

- `universe.yaml` — endpoints for leagues, timeline, simulation stages, anomalies, blackwall insights and metrics.
- `universe-models.yaml` — reusable parameters and core schemas (League, TimelineEvent, SimulationStage, etc.).
- `universe-models-operations.yaml` — request/response envelopes for listings, detail payloads and mutation bodies.
