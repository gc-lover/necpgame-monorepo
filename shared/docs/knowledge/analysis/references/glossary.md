## Игровые термины

### MMORPG
Massively Multiplayer Online Role-Playing Game - Массовая многопользовательская онлайн ролевая игра.

### Экстрактшутер
Жанр игр, где игроки заходят в локацию, собирают ресурсы/лут, и должны выйти живыми (например, Tarkov).

### PvP
Player versus Player - игрок против игрока.

### PvE
Player versus Environment - игрок против окружения (NPC, мобы).

### Крафт
Процесс создания предметов из ресурсов.

### Заточка
Процесс улучшения экипировки за счет повышения её характеристик.

### Имплант
Кибернетическое устройство, вживляемое в тело персонажа для улучшения способностей.

### Гильдия/Клан
Социальная группа игроков, объединившихся для совместной игры.

### Рейд
Групповое мероприятие для прохождения сложного контента.

## Технические термины

### API
Application Programming Interface - программный интерфейс приложения.

### REST
Representational State Transfer - архитектурный стиль веб-сервисов.

### gRPC
Google Remote Procedure Call - фреймворк для создания распределенных систем.

### БД
База данных.

### Миграция
Изменение структуры базы данных с сохранением данных.
