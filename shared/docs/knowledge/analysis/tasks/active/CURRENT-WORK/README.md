# CURRENT-WORK — активные направления

- `current-status.md` — сводка текущих направлений и прогресса.
- `active/` — детальные заметки по задачам (Task Notes).
- `archive/` — завершённые заметки (структура зеркальная каталогу `active/`).

Используйте шаблон из `TEMPLATES-GUIDE.md` для заметок. После завершения работы перенесите файл в `archive/` и обновите `TODO.md`.
