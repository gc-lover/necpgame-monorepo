# Архив завершённых работ `.BRAIN/06-tasks/archive`

**Назначение:** хранение завершённых задач, трекеров и статистики после очистки активной зоны.

## Структура
- `INDEX.md` — сводный указатель по архиву.
- `completed-features/` — отчёты по завершённым фичам (группировка по месяцам).
- `completed-trackers/` — архивные копии трекеров (`readiness`, `brain-mapping`, `implementation`).
- `statistics/` — агрегированные метрики и помесячные отчёты.
- `completed-todos/` — закрытый backlog (по месяцам).
- `resolved-questions/` — ответы на вопросы после эскалации.

## Правила
- Каждая запись ≤ 500 строк; при переполнении создавайте части `_0001`, `_0002`, ….
- Файлы содержат шапку: период, дата архивирования, агент.
- Перед удалением из активной зоны убедитесь, что данные перенесены.
- Фиксируйте изменения вручную через git (`git add`, `git commit`, `git push`) после формирования архива.
