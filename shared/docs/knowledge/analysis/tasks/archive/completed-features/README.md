# Архив завершённых фич — `completed-features/`

## Структура
- `YYYY-MM/` — каталог месяца (например, `2025-11/`).
  - `feature-reports-001.md`, `feature-reports-002.md` — отчёты батчами по 400–500 строк.
  - `README.md` — краткая сводка месяца (опционально).

## Правила
- Каждая запись содержит: название фичи, документы `.BRAIN`, задания API, статусы реализации, время завершения.
- При переносе указываем `archived_at` и агента, выполнившего архивирование.
- Поддерживаем хронологический порядок.
