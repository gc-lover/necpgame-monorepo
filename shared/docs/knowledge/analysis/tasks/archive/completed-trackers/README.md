# Архив трекеров — `completed-trackers/`

## Подкаталоги
- `implementation-tracker/` — копии записей после завершения backend/frontend.
- `brain-mapping/` — архив связей `.BRAIN` → API задачи.
- `readiness-tracker/` — история проверок готовности.

## Правила
- Каждый файл содержит шапку: период, дата архивирования, агент, источник.
- Для каждой записи добавляйте поле `archived_at`.
- Используйте `YYYY-MM.yaml` для группировки; при переполнении создавайте `_0001`.
