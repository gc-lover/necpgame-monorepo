# Архив статистики — `statistics/`

## Структура
- `monthly/` — помесячные отчёты (`YYYY-MM.md`).
- `overall-stats.md` — накопительная статистика по проекту (создать при необходимости).

## Правила
- Каждая запись включает: ключевые метрики, количество задач, статус реализаций, время.
- Обновляется после завершения Clear-процедур.
