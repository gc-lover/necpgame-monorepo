# Процедурная матрица снаряжения (оружие/импланты) как в Borderlands

**Статус:** recorded  
**Тема:** combat  
**Приоритет:** high  
**Дата создания:** 2025-11-03  
**Автор:** User

## Краткое описание

Нужна система генерации оружия, имплантов и экипировки через матрицу характеристик и модификаторов по образцу Borderlands: бренды/корпорации со своими особенностями + базовый набор характеристик, из комбинаций которых процедурно создаются предметы.

## Детальное описание

Идея: предметы описываются не фиксированными статами, а набором параметров и модулей. Генерация происходит как комбинация:
- Бренд/корпорация (влияет на фрейм-параметры, сигнатурные эффекты, визуал)
- Базовый тип (пистолет, штурмовая винтовка, снайперка, кибердека, нейроимплант и т.д.)
- Класс редкости (common → legendary → artifact)
- Пулы характеристик (урон, скорострельность, точность, отдача, проникновение, тип урона, перегрев, энергопотребление, шум, взломоустойчивость и др.)
- Модификаторы/аффиксы (уникальные эффекты брендов, синергии с имплантами/скиллами, сет-баффы, проки)
- Слоты модов (ствол/прицел/магазин/ИБП/охладитель/ИК-сканер/ICE-лимитер и пр.)

Система должна поддерживать:
- Процедурную генерацию лута (дроп/квесты/крафт)
- Детерминированные сиды для воспроизводимости
- Балансируемые пулы статов по лига-уровню/TTK/зоне (PvE/PvP/экстракт)
- Лорные корпорации с чёткими идентичностями (киберпанк-стиль)

## Обоснование

- Высокая реиграбельность и глубина лут-охоты
- Поддержка долгосрочного контент-пайплайна (бренды/сеты/сезоны)
- Сильная интеграция с экономикой (торговля, крафт, заказные сборки)

## Связь с другими системами

- Бой: TTK, типы урона, хитбоксы, укрытия, перегрев/энергия
- Импланты/киберпсихоз: синергии, лимиты, энергетический бюджет, риск перегруза
- Хакерство/ICE: параметры взлома/защиты, слоты ICE-модулей, термоконтроль
- Экономика/крафт: бренды, редкости, рецепты, рынки, заказы
- Социальные: фракционные бренды, эксклюзивы, клановые сеты

## Примеры использования

- «Militech» штурмовая винтовка: высокая стабильность, бронепробитие, минус скрытность
- «Arasaka» нейроимплант ОС: бонус к взлому, риск перегрева, требовательность к охлаждению
- «Kiroshi» оптика: крит-скоуп, шумоподавление выстрелов при ADS
- Сет «NetWatch»: +ICE-резисты, -скорость перезарядки

## Вопросы для проработки

- [ ] Перечень корпораций/брендов и их сигнатуры (по лору + авторские)
- [ ] Полный список базовых характеристик для оружия/имплантов/экипировки
- [ ] Пулы модификаторов/аффиксов по редкостям и слотам
- [ ] Модель генерации (веса, ограничения совместимости, сиды)
- [ ] Баланс по режимам (PvE/PvP/экстракт/киберпространство)
- [ ] Интеграция с крафтом (рецепты, рероллы, апгрейды, фьюзинг)
- [ ] Влияние корпораций на экономику и прогрессию лиг

## Заметки

- Учитывать энергобюджет и теплоотвод (перегрев, киберпсихоз-риск)
- Добавить фракционные эксклюзивы и события с ограниченными сериями
