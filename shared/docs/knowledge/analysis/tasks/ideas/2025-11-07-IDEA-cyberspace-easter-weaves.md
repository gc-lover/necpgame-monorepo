# Киберпространственные пасхалки и чизи-ткани

## Краткое описание

Восьмой набор пасхалок, ориентированный на киберпространство и VR-слои города. Сцены меняют поведение в зависимости от сетевых условий и действий игроков. Все намёки осторожные, без прямых IP.

## Детальное описание

### 1. «Поляна протокольных нитей»
- **Суть:** внутри VR-хаба появляется поле света, где нити меняют цвет от действий игроков.
- **Пасхалки:**
  - Если игроки выстраивают нити в последовательность «синий-золотой-пурпурный», появляется сообщение «Мы держим связь, даже когда сервер падает».
  - Команда, повторившая рисунок трижды, открывает косметику «Каскадный плащ».
- **Микросервисы:** `cyberspace-service`, `social-service`, `inventory`.

### 2. «AR-пазл “Лабиринт точек синхры”»
- **Суть:** на городской карте появляются точки, соединяющиеся в фигуру.
- **Пасхалки:**
  - Собранная фигура показывает силуэт и подпись «Если потерялся — ищи отражение».
  - Каждый сезон фигура другая; при прохождении всех сезонов выдаётся эмоция «Синхронный жест».
- **Микросервисы:** `world-service`, `cyberspace-service`, `inventory`.

### 3. «Чат-капсула “Нейронный шёпот”»
- **Суть:** скрытый канал чата, активный только при определённой погоде в игре.
- **Пасхалки:**
  - Сообщения бота: «Заполни пробелы в памяти — получишь доступ».
  - Выполнение мини-задания выдаёт титул «Узел связи».
- **Микросервисы:** `social-service`, `world-service`, `auth-service`.

### 4. «VR-катакомбы “Архив трёх контуров”»
- **Суть:** многоуровневый подземный VR-слой с тремя комнатами.
- **Пасхалки:**
  - В первой комнате фраза «Второй контур уже смотрит».
  - В третьей — «Не пробивай наносетку — она записывает».
  - Прохождение всех уровней без ошибок даёт предмет «Светящийся контур».
- **Микросервисы:** `cyberspace-service`, `gameplay-service`, `inventory`.

### 5. «Мультимедиа-флот “Ленты свидетельств”»
- **Суть:** над городом пролетают голографические дроны, транслирующие архивные сцены.
- **Пасхалки:**
  - Случайные фразы: «Они снимали всё на вторую камеру».
  - Если игроки отсканируют три дрона подряд, активируется AR-панорамма с намёком на культовую погоню.
- **Микросервисы:** `world-service`, `audio-service`, `cyberspace-service`.

### 6. «Симулятор “Лаборатория перемотки”»
- **Суть:** киберпространственный модуль, позволяющий проигрывать действия в обратном порядке.
- **Пасхалки:**
  - Если игрок “перематывает” бой, слышит голос «Шаг назад не отменяет выбор».
  - Повтор успешного манёвра даёт временный бафф «Deja Reaction».
- **Микросервисы:** `gameplay-service`, `cyberspace-service`, `inventory` (бафф).

## Обоснование

- Расширяет пасхалки в цифровых слоях мира, где игроки проводят много времени.
- Предлагает новые типы кооперативных задач и визуальных наград.
- Закладывает базу для API-задач по киберпространству, AR и чатам.

## Связь с другими системами

- **cyberspace-service:** VR/AR сцены, нити, каталоги.
- **world-service:** погодные условия, точки на карте, дроны.
- **social-service:** скрытые чаты, кооперация.
- **gameplay-service:** баффы, пазлы.
- **inventory:** косметика, титулы, предметы.
- **auth-service:** доступ к скрытым каналам и VR-уровням.

## Примеры использования

- Команда синхронизирует нити протокольного поля и делает скриншот для гильдии.
- Игроки ловят дронов, чтобы открыть AR-панорамму и делятся подсказкой.
- Спидранеры тренируются в «Лаборатории перемотки» для PvP арен.
