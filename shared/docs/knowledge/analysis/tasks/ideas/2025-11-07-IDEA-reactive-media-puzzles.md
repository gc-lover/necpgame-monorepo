# Реактивные пасхалки и медиа-головоломки

## Краткое описание

Набор пасхалок, реагирующих на действия игроков: мультимедийные головоломки, динамические AR-инсталляции и социальные челленджи. Все намёки завуалированы, совпадения с IP избегаются.

## Детальное описание

### 1. «Коридор спектральных афиш»
- **Суть:** переход между районами, где голографические афиши реагируют на эмоции персонажа.
- **Пасхалки:**
  - При выборе «саркастической» эмоции афиши превращаются в чёрно-белые пиктограммы с подписью «Не верь лицу».
  - Если игрок останавливается у четвёртой афиши более 10 секунд, появляется скрытый QR-узор, ведущий в ARG квест.
  - Собрав три AR-фрагмента, можно получить «Плакат-артефакт», меняющий загрузочный экран.
- **Микросервисы:** `world-service` (триггеры эмоций), `social-service` (ARG прогресс), `auth-service` (загрузочный скин).

### 2. «Кибер-бар “Память о втором дубле”»
- **Суть:** бар, где напитки открывают звуковые флешбеки.
- **Пасхалки:**
  - Заказ напитка «Вечный дубль» запускает диалог о «петле ночи».
  - Если три игрока заказывают разные напитки, возникает мини-сцена с голографическим режиссёром.
  - Найдя скрытый чип в барной стойке, активируешь бафф «Режиссёрская версия» на команду.
- **Микросервисы:** `social-service` (кооперативная сцена), `gameplay-service` (бафф), `economy-service` (уникальные напитки).

### 3. «AR-квест “Шепоты витрины”»
- **Суть:** витрина магазина показывает сообщения, если рядом нет NPC.
- **Пасхалки:**
  - Ночью витрина проявляет текст «Пакет доставлен в 88-ю ячейку».
  - При использовании AR-сканера игрок видит силуэт с плащом и фразу «Используй старый пароль». Ввод шифра открывает тайный мини-магазин.
  - Покупка всех предметов даёт эмоцию «Ночной курьер».
- **Микросервисы:** `realtime-server` (детект NPC), `world-service` (витрина), `economy-service` (мини-магазин).

### 4. «Станция “Резонанс памяти”»
- **Суть:** подземная станция метро, где записываются «эхо» прошлых действий игроков.
- **Пасхалки:**
  - Если игрок возвращается через 24 часа, услышит свой прошлый шёпот.
  - Совместное воспроизведение трёх шёпотов запускает короткий клип о сопротивлении.
  - Раз в день появляется «Хранитель резонанса» с вопросом про «Зеркало без отражения»; правильный ответ выдаёт титул.
- **Микросервисы:** `world-service` (хранение шёпотов), `social-service` (синхронизация), `inventory` (титул).

### 5. «Кластер “Фрактальный плейлист”»
- **Суть:** общественная зона со звуковыми терминалами, реагирующими на последовательность треков.
- **Пасхалки:**
  - Правильная последовательность включает «фрактальный свет» и скрытую танцевальную миссию.
  - Случайное включение «третьей композиции» вызывает голос «Слушатель из прошлого».
  - Если команда выполнит миссию без ошибок, открывается сет артифактов для `inventory`.
- **Микросервисы:** `audio-service`, `gameplay-service`, `inventory`.

### 6. «Мобильный музей “Параллельные реальности”»
- **Суть:** временный музей, появляющийся в разных районах.
- **Пасхалки:**
  - Каждый экспонат содержит подсказку к пазлу «Собери координаты».
  - При сборе всех подсказок доступен VR-фрагмент с фразой «Мы видели их решения».
  - Посетители, завершившие пазл, получают косметику «Голографический шарф».
- **Микросервисы:** `world-service` (перемещения музея), `gameplay-service` (пазл), `inventory` (косметика).

## Обоснование

- Стимулирует исследование города и социальное взаимодействие.
- Добавляет слои медиа-головоломок и ambient storytelling без прямых IP.
- Готовит почву для будущих эвентов и API задач (ARG, аудио, AR).

## Связь с другими системами

- **world-service:** динамические локации, витрины, музей.
- **social-service:** кооперативные сцены, ARG прогресс.
- **gameplay-service:** баффы, мини-квесты, миссии.
- **economy-service:** тайные магазины, артефакты.
- **audio-service:** плейлисты и клипы.
- **auth-service:** загрузочные экраны, титулы, косметика.

## Примеры использования

- Команда решает ARG «Коридора афиш» и запускает скрытую сцену.
- Игроки собираются в баре для синхронного флешбека.
- Гильдия организует спидин ран по «Фрактальному плейлисту» ради артефактов.
