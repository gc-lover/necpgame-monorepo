# Лор и сеттинг

**api-readiness:** not-applicable  
**api-readiness-check-date:** 2025-11-06 18:10
**api-readiness-notes:** Служебный файл-индекс раздела лора. Сам не содержит механик для API. Дочерние документы имеют свои статусы готовности.

Этот раздел содержит информацию о лоре, сеттинге и вселенной игры NECPGAME в стиле CYBERPUNK.

**Статус:** Детальная проработка завершена (2025-11-06). Раздел содержит полную энциклопедию мира.

## Архитектурный контекст

- Лоровые документы синхронизированы с полностью микросервисным backend’ом проекта и не опираются на устаревшие монолитные сценарии.
- Production API для всех игровых сервисов доступен только по `https://api.necp.game/v1` и `wss://api.necp.game/v1`.
- При подготовке связанных API спецификаций необходимо указывать корректный `info.x-microservice` для сервиса, управляющего соответствующей механикой (world-service, social-service и т.д.).
