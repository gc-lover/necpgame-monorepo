## Описание

Этот раздел содержит авторские (оригинальные) события временной линии, согласованные с:
- `01-concepts/` (видение, столпы, философия, уникальные особенности)
- `03-lore/universe.md` (система лиг 2020–2090, раскрытие симуляции)
- `04-narrative/` (система квестов, сценарии, события)

События разделены по периодам (10–20 лет), регионам, фракциям и городам, встроены в концепцию лиг (2020–2093) и поддерживают киберпанк-эстетику.
