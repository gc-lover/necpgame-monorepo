## Описание
Выучить культуру мате — традиционный напиток Аргентины.

## Региональные отсылки
- **Mate (мате)** — традиционный напиток
- **Калабаса** — сосуд из тыквы
- **Bombilla** — металлическая трубочка
- **Социальный** ритуал (передают по кругу)

## Этапы
1. Купить набор: калабаса + bombilla + yerba mate
2. Обучение:
   - Заполнить калабасу травой
   - Горячая вода (не кипяток!)
   - Пить через bombilla
3. Передавать по кругу (социальный ритуал!)
4. Горький вкус (кофеин!)
5. Понять: мате = социальная связь
