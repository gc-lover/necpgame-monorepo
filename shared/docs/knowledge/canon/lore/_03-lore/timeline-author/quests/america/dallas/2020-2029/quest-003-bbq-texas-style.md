# Барбекю по-Техасски

**quest-id:** DALLAS-2029-003  
**city:** Даллас | **region:** Америка | **period:** 2020-2029  
**quest-type:** social | **difficulty:** easy | **solo-group:** solo | **time:** 1-2h

## Описание
Попробовать Texas BBQ — копчёная грудинка (brisket) коронное блюдо.

## Региональные отсылки
- **Texas BBQ** — коронное блюдо штата
- **Brisket** — грудинка копчёная 12+ часов
- **Pecan wood** — орех пекан для дыма
- **Очередь** до 3 часов в лучших местах!

## Этапы
1. Найти BBQ место (Pecan Lodge, Lockhart)
2. Очередь (долгая!)
3. Заказать:
   - Brisket (грудинка) — тает во рту
   - Ribs (рёбра)
   - Sausage (колбаса)
4. Едят руками, без соуса (мясо само вкусное!)
5. Понять: Texas BBQ = религия

## Региональные детали
- **12+ часов** копчения
- **Brisket** — король
- **Pecan дым** — аромат
- **Очередь 3 часа** — норма
