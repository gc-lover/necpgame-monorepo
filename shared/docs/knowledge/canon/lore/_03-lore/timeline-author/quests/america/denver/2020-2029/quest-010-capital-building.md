# Капитолий Колорадо

**quest-id:** DENVER-2029-010  
**city:** Денвер | **region:** Америка | **period:** 2020-2029  
**quest-type:** social | **difficulty:** easy | **solo-group:** solo | **time:** 1-2h

## Описание
Посетить Colorado State Capitol — ступенька "Mile High" и золотой купол.

## Региональные отсылки
- **13-я ступенька** = ровно 1 миля (1,609 м)!
- **Золотой купол** — из настоящего золота (Колорадо = золотая лихорадка!)
- **"Colorful Colorado"** — прозвище штата
- **Панорама** Скалистых гор

## Этапы
1. Colorado State Capitol
2. Найти 13-ю ступеньку: надпись "One Mile Above Sea Level"
3. Фото на ступеньке (все делают!)
4. Золотой купол — настоящее золото!
5. Смотровая: панорама Rockies
6. История золотой лихорадки

## Региональные детали
- **13-я ступенька** — ровно миля!
- **Золотой купол** — из золота лихорадки
- **Панорама** гор
- **Символ** штата
