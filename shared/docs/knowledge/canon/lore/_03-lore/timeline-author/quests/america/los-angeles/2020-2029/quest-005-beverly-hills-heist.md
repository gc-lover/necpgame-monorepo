# Ограбление Беверли-Хиллз

**quest-id:** LA-2029-005  
**city:** Лос-Анджелес | **region:** Америка | **period:** 2020-2029  
**quest-type:** main | **difficulty:** extreme | **solo-group:** group-2-4 | **time:** 4h+

## Описание
Ограбить особняк знаменитости в Беверли-Хиллз — районе миллионеров.

## Региональные отсылки
- **Beverly Hills** — самый богатый район ЛА
- **Rodeo Drive** — улица luxury shopping
- **Mansions** — особняки знаменитостей
- **"90210"** — культовый zip code

## Этапы
1. Выбрать цель: особняк звезды кино/музыки
2. Планирование: изучить охрану
3. Проникновение ночью (стелс, хакерство)
4. Кража: драгоценности, искусство, импланты
5. Эвакуация до прибытия LAPD

## Региональные детали
- **Огромные особняки** с бассейнами
- **Private security** — охрана
- **Роскошь** везде: искусство, машины, украшения
- **Palm trees** — пальмы
