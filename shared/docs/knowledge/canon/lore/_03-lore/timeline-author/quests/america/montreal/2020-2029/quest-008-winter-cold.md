# Зимний Холод

**quest-id:** MONTREAL-2029-008  
**city:** Монреаль | **region:** Америка | **period:** 2020-2029  
**quest-type:** side | **difficulty:** hard | **solo-group:** solo | **time:** 1 день

## Описание
Пережить монреальскую зиму — -30°C и снежные бури.

## Региональные отсылки
- **-30°C зимой** — обычная температура
- **"Feels like -40°C"** с ветром
- **Снега** по пояс
- **Все прячутся** в RÉSO (подземный город)

## Этапы
1. Январь-февраль: пик холода (-30°C)
2. Выйти наружу:
   - Воздух обжигает лёгкие
   - Ресницы замерзают
   - Нос немеет за секунды
3. Снежная буря: видимость 0
4. Все уходят в RÉSO
5. Понять: зима суровая

## Региональные детали
- **-30°C** — норма
- **-40°C** с ветром
- **Снега** много
- **RÉSO** — спасение
