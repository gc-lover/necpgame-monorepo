# Восхождение на Эмпайр-стейт

**quest-id:** NY-2029-009  
**city:** Нью-Йорк | **region:** Америка | **period:** 2020-2029  
**quest-type:** side | **difficulty:** hard | **solo-group:** solo | **time:** 2-4h

## Описание
Подняться на крышу Empire State Building, чтобы установить хакерский узел на шпиле.

## Региональные отсылки
- **Empire State Building** — 381 метр, символ Нью-Йорка
- **102 этажа** — один из самых высоких небоскрёбов
- **King Kong** — фильм (горилла на шпиле)
- **Цветная подсветка** по праздникам

## Этапы
1. Проникнуть в здание (охрана)
2. Подняться: лифт или лестница (1,576 ступеней)
3. Преодолеть корп-охрану на крыше
4. Установить узел на шпиль
5. Спуск (прыжок с парашютом или zip-line)

## Региональные детали
- **Art Deco** архитектура
- **Вид на весь Нью-Йорк** с крыши
- **Ветер** на высоте
- **Подсветка** меняет цвета

## Награды
**XP:** 3,000 | **Валюта:** 15,000 Eddies | **Ачивка:** «Король Кинг-Конг»

**Бонус:** Узел даёт бесплатный интернет
