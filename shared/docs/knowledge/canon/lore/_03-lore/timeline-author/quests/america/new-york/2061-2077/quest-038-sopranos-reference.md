## Описание
Выполнить контракт для мафиозной семьи в стиле сериала "Sopranos" — классический мафиозный квест.

## Региональные отсылки
- **The Sopranos** — культовый сериал о мафии
- **New Jersey** — штат рядом с Нью-Йорком
- **Italian-American mob** — итало-американская мафия
- **"Gabagool!"** — мясная нарезка (мем из сериала)

## Этапы
1. Встреча с боссом в итальянском ресторане
2. Заказ: "решить проблему" (устранить/запугать/собрать)
3. Выполнение контракта
4. Возврат в ресторан
5. Ужин с семьёй (cannoli и espresso)

## Региональные детали

- **Gabagool** (capicola) на столе
