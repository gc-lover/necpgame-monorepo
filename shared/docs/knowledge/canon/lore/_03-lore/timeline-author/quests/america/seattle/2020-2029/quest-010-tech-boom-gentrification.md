## Описание
Исследовать трансформацию Сиэтла — от гранжа к tech hub, джентрификация.

- **Tech boom** — Amazon, Microsoft (штаб-квартира рядом)
- **Аренда выросла** — $3,000+/месяц за однушку!
- **Бездомность** — 12,000+ бездомных
- **Потеря души** — старый Сиэтл исчезает

## Этапы
1. Изучить трансформацию:

   - 2020-е: Кризис жилья
2. Capitol Hill: был артистичным → теперь дорогой
3. Палаточные городки бездомных
4. Местные жалуются: город изменился
5. Выбор: Tech прогресс vs потеря души
