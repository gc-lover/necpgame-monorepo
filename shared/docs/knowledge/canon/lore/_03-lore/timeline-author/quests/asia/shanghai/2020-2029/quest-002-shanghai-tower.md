# Башня Шанхая

**quest-id:** SHANGHAI-2029-002  
**city:** Шанхай | **region:** Азия | **period:** 2020-2029  
**quest-type:** side | **difficulty:** hard | **solo-group:** solo | **time:** 2-4h

## Описание
Подняться на Shanghai Tower — второе по высоте здание мира (632 м), установить хакерский узел.

## Региональные отсылки
- **Shanghai Tower** — 632 метра, 128 этажей
- **Второе в мире** (после Burj Khalifa)
- **Спиральная форма** — уникальная архитектура
- **Lujiazui** — финансовый район Пудун

## Этапы
1. Проникнуть в башню (соц. инженерия)
2. Подняться на лифте (самый быстрый в мире: 20.5 м/с)
3. Смотровая на 118-м этаже (561 м)
4. Пробраться выше, на технический этаж
5. Установить узел на шпиле (632 м)

## Региональные детали
- **Спиральная башня** из стекла
- **Облака** на уровне 500 м
- **Панорама** на весь Шанхай
- **Три башни Пудуна:** Shanghai Tower, Shanghai World Financial Center, Jin Mao
