# Взлом Токийской Башни

**quest-id:** TOKYO-2029-005  
**city:** Токио | **region:** Азия | **period:** 2020-2029  
**quest-type:** main | **difficulty:** hard | **solo-group:** solo | **time:** 2-4h

## Описание
Взломать Токийскую Башню, превращённую в гигантскую передающую антенну для контроля AR-рекламы города.

## Региональные отсылки
- **Tokyo Tower** — красная башня, символ города
- **AR-реклама** — голограммы по всему Токио
- **Контроль медиа** — информационная война

## Этапы
1. Нанять хакера для проникновения
2. Физическое проникновение в башню (стелс или соц. инженерия)
3. Подняться на вершину (333 метра)
4. Взлом центральной системы (DC Хакерство 20)
5. Выбор: Отключить рекламу / Контролировать / Саботаж

## Выборы
- **Отключить:** Токио без AR-рекламы (чистое небо), репутация народа +10
- **Контролировать:** Продавать рекламное место, пассивный доход 10,000 Eddies/неделя
- **Саботаж:** Хаос, реклама показывает неприличное

## Награды
**XP:** 3,000 | **Валюта:** 0-10,000/неделя | **Последствия:** Влияет на весь Токио

## Механики
**Хакерство:** Взлом  
**Стелс:** Проникновение  
**Мировое событие:** Изменение города

## Связи
Влияет на визуальный облик Токио
