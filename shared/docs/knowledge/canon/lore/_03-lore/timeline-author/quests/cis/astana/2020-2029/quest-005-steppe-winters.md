# Степные Зимы

**quest-id:** ASTANA-2029-005  
**city:** Астана | **region:** СНГ | **period:** 2020-2029  
**quest-type:** side | **difficulty:** hard | **solo-group:** solo | **time:** 4h+

## Описание
Пережить зиму в Астане — одна из самых холодных столиц мира (-40°C).

## Региональные отсылки
- **-40°C зимой** — обычная температура
- **Вторая самая холодная столица** (после Улан-Батора)
- **Степные ветры** усиливают холод
- **Buran (буран)** — снежные бури

## Этапы
1. Декабрь-январь: температура -40°C
2. Выйти на улицу:
   - Дыхание замерзает
   - Ресницы обледеневают
   - Ветер пронизывает
3. Переждать буран в помещении
4. Понять: люди живут здесь!
5. Купить казахскую тюбетейку и шапку

## Региональные детали
- **-40°C** — норма
- **Буран** — снежная буря
- **Степь** открытая
- **Люди привыкли** — стойкие
