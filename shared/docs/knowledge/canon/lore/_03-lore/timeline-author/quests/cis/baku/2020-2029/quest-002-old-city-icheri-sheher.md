# Старый Город Ичери Шехер

**quest-id:** BAKU-2029-002  
**city:** Баку | **region:** СНГ | **period:** 2020-2029  
**quest-type:** side | **difficulty:** easy | **solo-group:** solo | **time:** 2-4h

## Описание
Исследовать Ичери Шехер — крепость старого города с узкими улочками и Девичьей башней.

## Региональные отсылки
- **Ичери Шехер (İçərişəhər)** — "Внутренний город"
- **Девичья башня (Qız Qalası)** — символ Баку (XII век)
- **UNESCO** — объект всемирного наследия
- **Средневековая крепость**

## Этапы
1. Войти через крепостные ворота
2. Прогулка по узким улочкам (лабиринт)
3. Подняться на Девичью башню (панорама)
4. Легенда: Девушка прыгнула с башни
5. Дворец Ширваншахов (XV век)

## Региональные детали
