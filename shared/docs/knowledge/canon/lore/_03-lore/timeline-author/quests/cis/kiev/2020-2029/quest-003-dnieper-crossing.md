# Переправа через Днепр

**quest-id:** KIEV-2029-003  
**city:** Киев | **region:** СНГ | **period:** 2020-2029  
**quest-type:** side | **difficulty:** medium | **solo-group:** solo | **time:** 1-2h

## Описание
Пересечь Днепр — могучую реку, разделяющую Киев на правый и левый берег.

## Региональные отсылки
- **Днепр** — одна из крупнейших рек Европы
- **Правый берег (Киев)** — исторический центр
- **Левый берег (Дарница)** — современный район
- **Мосты:** Патона, Метро, Южный

## Этапы
1. Выбрать способ переправы:
   - Мост Патона (автомобиль)
   - Метромост (метро)
   - Лодка (водный путь)
2. Переправиться через реку
3. Встреча с контрабандистами на воде
4. Выбор: Помочь / Арестовать / Игнорировать

## Региональные детали
- **Широкий Днепр** — 1 км ширины
- **Мост Патона** — первый сварной мост в мире
- **Острова** посередине реки
- **Течение** сильное
