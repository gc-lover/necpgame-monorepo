# Район Немига

**quest-id:** MINSK-2029-005  
**city:** Минск | **region:** СНГ | **period:** 2020-2029  
**quest-type:** faction | **difficulty:** medium | **solo-group:** solo | **time:** 2-4h

## Описание
Исследовать Немигу — исторический район с древними корнями и современной молодёжной культурой.

## Региональные отсылки
- **Немига** — древнее название (битва на Немиге 1067)
- **Старейшая часть** Минска
- **Молодёжный район** — клубы, бары
- **Подземный торговый центр** "Столица"

## Этапы
1. Прогулка по Немиге
2. Спуститься в подземный ТЦ "Столица"
3. Ночью: район оживает (клубы)
4. Встреча с местной бандой
5. Выбор: Присоединиться / Противостоять / Игнорировать

## Региональные детали
- **Река Немига** — малая речка
- **Подземный** торговый центр
- **Клубы** с белорусским хип-хопом
- **Молодёжь** тусуется
