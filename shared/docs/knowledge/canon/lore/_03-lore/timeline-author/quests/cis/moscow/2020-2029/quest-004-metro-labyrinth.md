# Лабиринт Метро

**quest-id:** MOSCOW-2029-004  
**city:** Москва | **region:** СНГ | **period:** 2020-2029  
**quest-type:** side | **difficulty:** hard | **solo-group:** group-2-4 | **time:** 2-4h

## Описание
Экстракция редких компонентов из заброшенных станций метро, населённых бандами и роуг-дронами.

## Этапы
1. **Вход:** Найти вход в заброшенную ветку (Скрытность DC 14)
2. **Навигация:** Пройти лабиринт туннелей (Навигация DC 16, ловушки)
3. **Зачистка:** Столкновение с бандой «Подземных Крыс» (бой 5-7 врагов)
4. **Сбор:** Компоненты в старых серверных (Технологии DC 15)
5. **Экстракция:** Выход с лутом (риск потери при смерти)

## Выборы
- **Пройти боем:** Стандартный путь
- **Договориться с бандой:** Переговоры DC 18, делёж добычи 50/50
- **Скрытно:** Стелс DC 20, никто не узнает

## Награды
**XP:** 1,200 | **Валюта:** 0 (продажа компонентов 3,000-5,000) | **Лут:** Редкие компоненты имплантов

**Риск:** Потеря всего при смерти (экстракт-механика)

## Механики
**Экстракт-зона:** Потеря лута при смерти  
**PvPvE:** Возможна встреча других игроков  
**Процедурный лут:** Рандомные компоненты

## Связи
Открывает доступ к «Подземной Карте»
