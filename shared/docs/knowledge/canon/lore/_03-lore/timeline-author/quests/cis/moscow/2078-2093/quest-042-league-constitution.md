# Конституция Лиги

**quest-id:** MOSCOW-2093-042  
**city:** Москва | **region:** СНГ | **period:** 2078-2093  
**quest-type:** main | **difficulty:** hard | **solo-group:** solo | **time:** 4h+

## Описание
Участвовать в создании «Конституции Лиги» — свода правил для следующего игрового сезона, который голосуется игроками.

## Этапы
1. **Предложения:** Игроки предлагают правила для следующей лиги
2. **Обсуждение:** Форумы и дебаты (внутри игры)
3. **Голосование:** Игроки голосуют за правила
4. **Принятие:** Топ-10 правил становятся законами следующей лиги

## Примеры правил
- PvP разрешён/запрещён в определённых зонах
- Скорость прокачки увеличена/уменьшена
- Новые механики: пермадеф, хардкор-режим
- Экономические изменения
