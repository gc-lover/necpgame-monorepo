# Мировой Босс

**quest-id:** MOSCOW-2093-048  
**city:** Москва | **region:** СНГ | **period:** 2078-2093  
**quest-type:** event | **difficulty:** extreme | **solo-group:** raid-10+ | **time:** 2-4h

## Описание
Сразиться с «Мировым Боссом» — гигантским роуг-ИИ, атакующим Москву. Событие для всего сервера.

## Этапы
1. **Вторжение:** Босс появляется в центре Москвы
2. **Эвакуация:** NPC эвакуируются, игроки готовятся к бою
3. **Фазы боя:**
   - **Фаза 1:** Физический бой (100 млн HP)
   - **Фаза 2:** Хакерская атака (взлом щитов)
   - **Фаза 3:** Финальная фаза (берсерк-режим)
4. **Победа:** Босс побеждён или время вышло (1 час)

## Механики
- Все игроки сервера могут участвовать
- Вклад каждого учитывается
- Награды по рейтингу урона

## Награды
**Топ-1 по урону:** Легендарное оружие, 500,000 Eddies  
**Топ-10:** Редкие импланты, 100,000 Eddies  
**Топ-100:** Редкое оружие, 50,000 Eddies  
**Все участники:** XP 10,000, 10,000 Eddies

## Механики
**Мировое событие:** Весь сервер  
**Рейдовый контент:** Массовый бой  
**Ограничено по времени:** 1 час

## Связи
Weekly event
