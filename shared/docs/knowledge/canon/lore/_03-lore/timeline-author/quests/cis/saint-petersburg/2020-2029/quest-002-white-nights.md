# Белые Ночи

**quest-id:** SPB-2029-002  
**city:** Санкт-Петербург | **region:** СНГ | **period:** 2020-2029  
**quest-type:** event | **difficulty:** easy | **solo-group:** solo | **time:** 1-2h

## Описание
Пережить феномен Белых ночей Петербурга — когда солнце почти не заходит летом.

## Региональные отсылки
- **Белые ночи** — природное явление (июнь-июль)
- **Светло всю ночь** — солнце не заходит за горизонт
- **Разведение мостов** — ночной ритуал
- **Романтическая атмосфера**

## Этапы
1. Прийти на набережную Невы в полночь
2. Свет как днём (но полночь)
3. Наблюдать разведение мостов (Дворцовый мост)
4. Прогулка по ночному городу
5. Встретить рассвет в 3 утра

## Региональные детали
- **Нева** отражает светлое небо
- **Мосты** разводятся для кораблей
- **Романтические пары** на набережных
- **Музыка** уличных музыкантов
