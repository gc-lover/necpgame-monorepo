# Петропавловская Крепость

**quest-id:** SPB-2029-003  
**city:** Санкт-Петербург | **region:** СНГ | **period:** 2020-2029  
**quest-type:** side | **difficulty:** hard | **solo-group:** group-2-4 | **time:** 2-4h

## Описание
Проникнуть в Петропавловскую крепость — историческую тюрьму, превращённую в корп-тюрьму максимальной безопасности.

## Региональные отсылки
- **Петропавловская крепость** — первое здание Петербурга (1703)
- **Исторически — тюрьма** для политических заключённых
- **Петропавловский собор** — усыпальница императоров
- **Заячий остров** — расположение

## Этапы
1. Контракт: освободить заключённого
2. Проникновение на остров (Нева вокруг)
3. Взлом систем безопасности
4. Освобождение из камеры
5. Эвакуация через подземные ходы

## Региональные детали
- **Жёлтые стены** крепости
- **Ангел** на шпиле собора (золотой)
- **Пушка** стреляет в полдень (традиция)
- **Бастион** и **равелины**
