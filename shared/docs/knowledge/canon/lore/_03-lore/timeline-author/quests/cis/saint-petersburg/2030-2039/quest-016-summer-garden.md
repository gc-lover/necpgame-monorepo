# Летний Сад

**quest-id:** SPB-2039-016  
**city:** Санкт-Петербург | **region:** СНГ | **period:** 2030-2039  
**quest-type:** social | **difficulty:** easy | **solo-group:** solo | **time:** 30-60min

## Описание
Романтическая прогулка по Летнему саду — первому парку Петербурга с античными статуями.

## Региональные отсылки
- **Летний сад** — создан Петром I (1704)
- **89 мраморных статуй** — античные боги и герои
- **Летний дворец Петра I** — скромная резиденция
- **Ограда сада** — шедевр художественного литья

## Этапы
1. Войти через знаменитую ограду
2. Прогулка по аллеям
3. Осмотреть статуи (каждая с историей)
4. Посетить Летний дворец Петра
5. Романтический момент у фонтана

## Региональные детали
- **Мраморные аллеи**
- **Античные боги** в статуях
- **Тишина** и покой
- **Лебеди** на пруду
