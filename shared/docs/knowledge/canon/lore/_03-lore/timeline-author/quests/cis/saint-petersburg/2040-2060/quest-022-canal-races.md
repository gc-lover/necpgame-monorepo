# Гонки по Каналам

**quest-id:** SPB-2060-022  
**city:** Санкт-Петербург | **region:** СНГ | **period:** 2040-2060  
**quest-type:** event | **difficulty:** hard | **solo-group:** solo | **time:** 1-2h

## Описание
Участвовать в гонках на гидроциклах по узким каналам Петербурга.

## Региональные отсылки
- **Каналы СПб** — узкие водные артерии
- **Канал Грибоедова** — самый извилистый
- **Мосты** — препятствия на трассе
- **"Петербургская Венеция"**

## Этапы
1. Регистрация (вх. взнос 2,000 Eddies)
2. Модификация гидроцикла
3. Гонка: Нева → Мойка → Грибоедова → Фонтанка → Крюков
4. Препятствия: низкие мосты, повороты, гондолы
5. Финиш у Адмиралтейства

## Региональные детали
- **Узкие каналы** — сложная навигация
- **Под мостами** — нагибаться
- **Дворцы** проплывают мимо
- **Брызги** на фасады зданий

## Награды
**1 место:** 25,000 Eddies, легендарный гидроцикл «Канальный Гонщик»  
**Участие:** XP 1,500

## Механики
**Гоночная аркада:** Водные гонки  
**Уникальная трасса:** Каналы СПб  
**Навигация:** Узкие повороты

## Связи
Weekly event, водная линия
