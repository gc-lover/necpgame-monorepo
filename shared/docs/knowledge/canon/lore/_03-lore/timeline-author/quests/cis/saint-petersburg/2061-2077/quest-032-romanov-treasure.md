# Сокровища Романовых

**quest-id:** SPB-2077-032  
**city:** Санкт-Петербург | **region:** СНГ | **period:** 2061-2077  
**quest-type:** main | **difficulty:** extreme | **solo-group:** group-2-4 | **time:** 4h+

## Описание
Найти легендарные сокровища Романовых, спрятанные перед революцией 1917 года.

## Региональные отсылки
- **Романовы** — императорская династия (1613-1917)
- **Императорские регалии** — корона, скипетр, держава
- **Яйца Фаберже** — 50 императорских яиц
- **Алмазный фонд** — драгоценности короны

## Этапы
1. Найти дневник последнего камергера (архивы)
2. Расшифровать записи о тайнике
3. Поиск ведёт в подземелья под Зимним дворцом
4. Данжен: ловушки, охрана-роботы
5. Обнаружение тайника с сокровищами

## Сокровища:
- **Большая Императорская корона** — 5,000 бриллиантов
- **10 яиц Фаберже** — бесценные
- **Скипетр с Орловым алмазом** — 190 карат
- **Золотые слитки** — 1 тонна

## Выборы
- **Вернуть государству:** Национальный герой, музей
- **Продать:** 2,000,000,000 Eddies
- **Оставить себе:** Личная коллекция
