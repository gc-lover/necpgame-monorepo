# Дворцовый Концерт

**quest-id:** SPB-2093-047  
**city:** Санкт-Петербург | **region:** СНГ | **period:** 2078-2093  
**quest-type:** event | **difficulty:** medium | **solo-group:** solo | **time:** 2-4h

## Описание
Финальный концерт классической музыки во всех дворцах одновременно — симфония прощания.

## Региональные отсылки
- **Классическая музыка** — русская школа
- **Чайковский, Мусоргский, Римский-Корсаков** — композиторы
- **Дворцы как концертные залы**
- **"Лебединое озеро"** — финальное произведение

## Этапы
1. Выбрать дворец для посещения:
   - Зимний (Эрмитаж)
   - Екатерининский (Царское Село)
   - Петергоф
   - Павловск
   - Михайловский
2. Концерт симфонического оркестра
3. Одновременно во всех дворцах играют
4. Финал: "Лебединое озеро" Чайковского
5. Овации, слёзы, прощание

## Региональные детали
- **Золотые залы** дворцов
- **Оркестр** в исторических костюмах
- **Люстры** мерцают
- **Акустика** имперских залов
