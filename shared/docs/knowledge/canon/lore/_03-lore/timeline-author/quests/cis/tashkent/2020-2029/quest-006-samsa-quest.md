# Квест Самсы

**quest-id:** TASHKENT-2029-006  
**city:** Ташкент | **region:** СНГ | **period:** 2020-2029  
**quest-type:** social | **difficulty:** easy | **solo-group:** solo | **time:** 30-60min

## Описание
Найти лучшую самсу в Ташкенте — треугольные пирожки из тандыра.

## Региональные отсылки
- **Самса (somsa)** — пирожок с мясом/тыквой
- **Тандыр** — печь из глины
- **Треугольная** форма традиционная
- **Горячая** прямо из печи

## Этапы
1. Опрос местных: где лучшая самса?
2. Посетить 3 места
3. Дегустация:
   - Самса с бараниной
   - Самса с тыквой
   - Самса с картошкой
4. Выбрать победителя
5. Научиться лепить самсу

## Региональные детали
- **Тандыр** горячий
- **Хрустящая** корочка
- **Сочная** начинка
- **Запах** невероятный
