# Затопленный Ист-Энд

**quest-id:** LONDON-2029-009  
**city:** Лондон | **region:** Европа | **period:** 2020-2029  
**quest-type:** side | **difficulty:** medium | **solo-group:** group-2-4 | **time:** 2-4h

## Описание
Экстракция ценностей из затопленного Ист-Энда — опасной зоны с водными бандами.

## Этапы
1. Аренда лодки/субмарины
2. Навигация по затопленным улицам
3. Поиск ценностей в зданиях (рандомный лут)
4. Столкновение с водными бандами
5. Экстракция с лутом

## Награды
**XP:** 1,500 | **Валюта:** 0 (продажа лута 5,000-20,000) | **Риск:** Потеря при смерти

## Механики
Экстракт-зона (как Tarkov)

## Связи
Repeatable quest
