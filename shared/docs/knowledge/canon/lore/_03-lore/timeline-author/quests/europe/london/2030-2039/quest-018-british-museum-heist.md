# Ограбление Британского музея

**quest-id:** LONDON-2039-018  
**city:** Лондон | **region:** Европа | **period:** 2030-2039  
**quest-type:** side | **difficulty:** hard | **solo-group:** group-2-4 | **time:** 2-4h

## Описание
Украсть артефакт из Британского музея по заказу коллекционера.

## Региональные отсылки
- **Британский музей** — хранилище мировых артефактов
- **Розеттский камень** — один из целевых артефактов
- **Колониальное наследие** — этическая дилемма

## Этапы
1. Выбрать артефакт (Розеттский камень / Египетские мумии / Греческие статуи)
2. Планирование (изучить охрану)
3. Проникновение (ночью, стелс или хакерство)
4. Кража артефакта
5. Побег

## Выборы
- **Продать коллекционеру:** 80,000 Eddies
- **Вернуть стране происхождения:** Репутация страны +10, моральное удовлетворение
- **Оставить себе:** Уникальный декор для дома

## Награды
**XP:** 3,000 | **Валюта:** 0-80,000 | **Уникальное:** Артефакт

## Механики
**Стелс-миссия:** Скрытность  
**Этическая дилемма:** Колониальное наследие

## Связи
Открывает квесты о возврате артефактов
