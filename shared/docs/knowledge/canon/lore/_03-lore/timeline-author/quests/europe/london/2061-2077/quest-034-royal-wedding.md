# Королевская Свадьба

**quest-id:** LONDON-2077-034  
**city:** Лондон | **region:** Европа | **period:** 2061-2077  
**quest-type:** event | **difficulty:** easy | **solo-group:** solo | **time:** 1-2h

## Описание
Присутствовать на королевской свадьбе — грандиозном событии, транслируемом на весь мир.

## Региональные отсылки
- **Королевские свадьбы** — национальное событие
- **Вестминстерское аббатство** — место церемонии
- **Парад** по Лондону

## Этапы
1. Получить приглашение (или проникнуть)
2. Церемония в аббатстве (катсцена)
3. Парад по The Mall
4. Приём во дворце (если элита)
5. Празднование по городу

## Региональные детали
- **Королевская гвардия** в парадной форме
- **Карета** с молодожёнами
- **Толпы** британцев с флагами

## Награды
**XP:** 1,000 | **Валюта:** 0 | **Социальное:** Свидетель истории

**Ачивка:** «Гость Королевы»

## Механики

**Катсцены:** Церемония
