# Память Блица

**quest-id:** LONDON-2077-040  
**city:** Лондон | **region:** Европа | **period:** 2061-2077  
**quest-type:** main | **difficulty:** medium | **solo-group:** solo | **time:** 2-4h

## Описание
Найти последнего ветерана Блица (бомбардировки Лондона 1940-41), чьи воспоминания хранят дух британской стойкости.

## Региональные отсылки
- **The Blitz** — бомбардировки Лондона во Второй мировой
- **"Keep Calm and Carry On"** — лозунг стойкости
- **Бункеры** — подземные укрытия

## Этапы
1. Найти столетнего ветерана в доме престарелых
2. Выслушать его истории о Блице (катсцены-воспоминания)
3. Помочь оцифровать воспоминания для архива
4. Ветеран просит выполнить последнее желание
5. Прощание и смерть ветерана (эмоциональная сцена)

## Региональные детали
- **Воздушные убежища** в метро
- **Сирены** воздушной тревоги (звуковой эффект)
- **"We shall never surrender"** — цитата Черчилля
