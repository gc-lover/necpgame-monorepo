# Гринвич и Симуляция

**quest-id:** LONDON-2093-047  
**city:** Лондон | **region:** Европа | **period:** 2078-2093  
**quest-type:** main | **difficulty:** extreme | **solo-group:** solo | **time:** 4h+

## Описание
Обнаружить глюк в Гринвичской обсерватории: время течёт неправильно, намёк на симуляцию.

## Региональные отсылки
- **Гринвич** — нулевой меридиан, стандарт времени
- **GMT** — Greenwich Mean Time
- **Обсерватория** — контроль времени

## Этапы
1. Аномалия: часы Гринвича показывают разное время
2. Расследование обсерватории
3. Обнаружение: время — параметр симуляции
4. Встреча с «Хранителем Времени» (ИИ)
5. Выбор: Исправить / Использовать / Раскрыть

## Выборы
- **Исправить:** Время нормализуется
- **Использовать:** Контроль времени для себя (манипуляция)
- **Раскрыть:** Доказательство симуляции для всех

## Награды
**XP:** 10,000 | **Валюта:** 0 | **Уникальное:** Контроль времени (если выбор 2)

**Последствия:** Раскрытие симуляции

**Лор-бомба:** Ключевое раскрытие  
**Манипуляция временем:** Игровая механика  
**Метагейм:** Осознание симуляции
