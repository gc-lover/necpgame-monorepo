# Реал Мадрид

**quest-id:** MADRID-2029-006  
**city:** Мадрид | **region:** Европа | **period:** 2020-2029  
**quest-type:** event | **difficulty:** easy | **solo-group:** solo | **time:** 2-4h

## Описание
Посетить матч Реал Мадрид на Сантьяго Бернабеу — легендарный клуб.

## Региональные отсылки
- **Real Madrid** — самый титулованный клуб (14 Лиг Чемпионов!)
- **Сантьяго Бернабеу** — стадион (81,000 мест)
- **"Hala Madrid!"** — боевой клич
- **Белая форма** — "Los Blancos"

## Этапы
1. Купить билет на матч
2. Стадион Бернабеу (огромный!)
3. Матч: 90 минут страсти
4. Фанаты кричат: "Hala Madrid!"
5. Гол — стадион взрывается!
6. Музей Реала (трофеи)
