## Цель документа

- Сформировать единый визуальный язык для территорий, зданий, рейдов и социальных пространств.
- Обеспечить быстрый перенос описаний в карточки локаций, UI и текстовые лоры.
- Увязать атмосферу NECPGAME с источниками вдохновения (Cyberpunk 2077, Valorant, Tarkov, BG3).

## Методика визуализации

- **Силуэт:** каждая локация имеет узнаваемый силуэт для карты и превью.
- **Слойность:** базовая геометрия, функциональные модули, акцентное освещение/голограммы.
- **Материалы:** комбинация металлизированных поверхностей, бетон-композита, стекла, неонового света и растительных вставок.
- **Динамика:** погодные эффекты, толпы NPC, транспорт, дроны, голографические объявления.
- **Фракционность:** каждая зона подчёркивает присутствие корпораций, банд или кланов через палитру и визуальные мотивы.

## Макро-локации (города и регионы)

**Night City Megapolis**
- Горизонт из игольчатых небоскрёбов, красно-жёлтые голограммы, слои транспортных магистралей.
- Улицы пропитаны туманом от кондиционеров, неон отражается в влажном асфальте.
- Дроны безопасности и рекламные голограммы пересекают небо.

**Neo Tokyo Arcology**
- Многоуровневые пласты с садами на крышах, прозрачные купола, плавающие храмы из световых линий.
- Световая палитра: холодный белый, аквамариновый, розовый.

**Berlin Free Zone**
- Переплетение индустриальных руин и новых биокуполов, граффити, лазерные прожекторы.
- Железнодорожные арки перепрофилированы в рынки и концертные площадки.

**Lagos Skyport**
- Взлётные площадки на разных уровнях, висящие контейнерные кварталы, тёплый янтарный свет.
- Голографические сетки показывают трафик воздушных грузовиков и дронов.

**Amazon Cloud Basin**
- Подвесные платформы над туманом, гидропонные фермы с биолюминесцентными растениями.
- Переходы освещены мягким зелёным, вода отражает неоновые символы эко-культов.

## REST API и JSON схемы

- `GET /world/visuals/locations` — список `VisualLocationProfile`.
- `GET /world/visuals/locations/{locationId}` — детальное описание и атмосферные теги.
- `GET /social/visuals/hubs` — визуальные профили социальных хабов (`VisualHubProfile`).
- `GET /economy/visuals/markets` — данные рынков и инфраструктуры (`VisualMarketProfile`).
- `POST /world/visuals/locations/export` — экспорт пакета профилей для UI и маркетинга.

**JSON схемы:**  
- `schemas/world/visual-location-profile.schema.json`  
- `schemas/social/visual-hub-profile.schema.json`  
- `schemas/economy/visual-market-profile.schema.json`  
- `schemas/world/visual-export-request.schema.json`

## Метрики и аналитика

- `LocationMoodConsistency` — соответствие визуальных тегов игровым событиям.
- `HubEngagementIndex` — вовлечённость в социальных хабах (по визуальному профилю).
- `MarketVisualConversion` — влияние визуала на торговую активность.
- Метрики передаются в telemetry и мониторятся через dashboards world/social команд.
