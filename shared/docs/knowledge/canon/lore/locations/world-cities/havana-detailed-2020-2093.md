## Краткая справка

- **Название:** Havana Data Reef Port
- **Страна / Регион:** Карибский Федеративный Союз
- **Население:** ~9,3 млн (плавучие дата-рифы 3,1 млн, сигарные аркологии 2,8 млн, подземные туры 3,4 млн)
- **Доминирующая корпорация:** Caribe Data Syndicate
- **Ключевые банды:** Malecón Tidecutters, Cohiba Signal Cartel, Havana Resonance Club, Buena Vista Foxes
- **Ключевой символ:** Плавучий маяк «La Perla Coil», оборачивающий сигарный дым плазменными фильтрами

### Горизонтальные кластеры

1. **La Perla Plaza** — сердце Syndicate, где регулируют потоки данных и контрабанду.  
   - Ключевые точки: Perla Command Tower, Rum Data Ledger, Diplomacy Lounge.  
   - Контроль: Caribe Data Syndicate.  
   - Атмосфера: аромат рома, музыка сон, мерцающие голограммы.
2. **Malecón Sound Strip** — протяжённая набережная с клубами, гонками и концертами.  
   - Ключевые точки: Sound Drift Stage, Tidecutter Hangar, Holo-Salsa Arena.  
   - Контроль: Malecón Tidecutters и Resonance Club.  
   - Атмосфера: басы, неоновые волнорезы, морской бриз.
3. **Cohiba Data Reef** — сеть рифов и дата-хранилищ, где Cartel производит ароматизированные импланты.  
   - Ключевые точки: Cohiba Flavor Labs, Perfumery Vault, Data Wave Spires.  
   - Контроль: Cohiba Signal Cartel.  
   - Атмосфера: дым сигар, лазурная вода, гудящие серверы.
4. **Buena Vista Vaults** — подземные джазовые бароводы и контрабандные тоннели.  
   - Ключевые точки: Buena Vista Legacy Club, Fox Relay, Underground Humidor.  
   - Контроль: Buena Vista Foxes.  
   - Атмосфера: плавные мелодии, приглушенный свет, запах табака.

## Улицы и маршруты

- **Malecón Flux Route:** скоростной канал для дронов-курьеров, связывающий набережную с Майами и Манилой.
- **Cohiba Spiral:** вертикальный маршрут подъёма сигарных грузов к La Perla Coil.
- **Buena Vista Tunnel:** скрытый тоннель, ведущий к подземным клубам и нарко-картелям.
- **Taino Subline:** подводный маршрут для Tidecutters и Caribe Syndicate (`FQ-HAVANA-003`).

### Контактные цепочки

- **«Perla Accord»:** Эстела Рон → Аделаида Сантос → Atlantic Shield Кейптауна — карибские конвои (`FQ-HAVANA-001`).
- **«Tide Runner»:** Рауль Мендоса → Pacific Couriers → Mekong Flux — гонки и спасательные миссии (`FQ-HAVANA-002`).
- **«Cohiba Lattice»:** Доктор Исабель → Naniwa Data Brokers → Tyger Claws — подпольные поставки вкусов (`FQ-HAVANA-004`).

## Сюжетные узлы и легенды

1. **«Дымовой контракт»**  
   - 2065: Syndicate хочет продать Cohiba моды Arasaka. Игрок решает, сдать ли сделку Vanguard.
