# Квесты авторской хронологии

**Дата создания:** 2025-11-06  
**Статус:** В процессе наполнения  
**Целевое количество:** 4,700 квестов

## Структура

Квесты организованы по городам и периодам:
- 94 города
- 5 периодов каждый
- 10 квестов на период
- **Итого: 4,700 квестов**

## Формат именования файлов

```
quests/[region]/[city]/[period]/quest-[###].md
```

**Примеры:**
- `quests/europe/london/2020-2029/quest-001.md`
- `quests/asia/tokyo/2040-2060/quest-005.md`
- `quests/cis/moscow/2078-2093/quest-010.md`

## Типы квестов

### По типу
- **Main** — основные сюжетные квесты
- **Side** — побочные квесты
- **Daily** — ежедневные
- **Weekly** — еженедельные
- **Faction** — фракционные
- **Dynamic** — динамические (генерируются)
- **Event** — событийные
- **Social** — социальные (романсы, семья, отношения)

### По сложности
- **Easy** — для новичков
- **Medium** — средняя сложность
- **Hard** — для опытных
- **Extreme** — хардкор

### По составу группы
- **Solo** — одиночный
- **Group 2-4** — малая группа
- **Group 5-10** — полная группа
- **Raid 10+** — рейд
