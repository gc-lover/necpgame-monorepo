# Кладбище Неона

**quest-id:** VEGAS-2029-006  
**city:** Лас-Вегас | **region:** Америка | **period:** 2020-2029  
**quest-type:** social | **difficulty:** easy | **solo-group:** solo | **time:** 1-2h

## Описание
Посетить Neon Museum — кладбище старых неоновых вывесок Вегаса.

## Региональные отсылки
- **Neon Boneyard** — музей старых вывесок
- **200+ вывесок** из закрытых казино
- **История Вегаса** через неон
- **"Neon graveyard"** — кладбище неона

## Этапы
1. Neon Museum (North Las Vegas)
2. Старые вывески:
   - Stardust Casino (закрыто 2006)
   - Moulin Rouge (первое интегрированное казино)
   - Desert Inn
3. История каждой вывески = история Вегаса
4. Ночной тур: некоторые вывески светятся!
5. Ностальгия по старому Вегасу

## Региональные детали
- **200+ вывесок** — история
- **Старый Вегас** умер
- **Неон** ржавеет
