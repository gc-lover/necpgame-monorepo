## Описание
Исследовать водный кризис — город в пустыне без воды!

- **Пустыня** — 20 см осадков в год
- **Река Колорадо** — источник воды
- **Озеро Mead** — высыхает!
- **Город не должен** существовать здесь

## Этапы
1. Изучить проблему:

   - Вода из реки Колорадо
   - Озеро Mead высыхает

2. Центральный канал Аризоны — вода течёт сотни км

4. Выбор: Остаться / Уехать / Решать проблему
