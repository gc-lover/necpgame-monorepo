# Квест Путина

**quest-id:** TORONTO-2029-003  
**city:** Торонто | **region:** Америка | **period:** 2020-2029  
**quest-type:** social | **difficulty:** easy | **solo-group:** solo | **time:** 30-60min

## Описание
Попробовать путин (poutine) — культовое канадское блюдо.

## Региональные отсылки
- **Poutine** — картофель фри + соус + сырные кубики
- **Квебекское** происхождение (но популярно везде)
- **Comfort food** — калорийная бомба
- **Национальное** блюдо Канады

## Этапы
1. Найти место с путином (везде!)
2. Заказать классический путин:
   - Картофель фри
   - Gravy (коричневый соус)
   - Cheese curds (сырные кубики)
3. Попробовать: жирно, вкусно, калорийно!
4. Варианты: с беконом, курицей, pulled pork
5. Понять: это Канада в тарелке

## Региональные детали
- **Cheese curds** — скрипят на зубах
- **Gravy** — коричневый соус
- **Калорийная бомба** — 1,000+ ккал
- **Comfort food** — еда для души
