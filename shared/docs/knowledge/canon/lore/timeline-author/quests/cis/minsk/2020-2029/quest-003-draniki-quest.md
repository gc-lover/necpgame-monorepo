# Квест Драников

**quest-id:** MINSK-2029-003  
**city:** Минск | **region:** СНГ | **period:** 2020-2029  
**quest-type:** social | **difficulty:** easy | **solo-group:** solo | **time:** 30-60min

## Описание
Найти лучшие драники (картофельные оладьи) — национальное блюдо Беларуси.

## Региональные отсылки
- **Драники (дранікі)** — картофельные оладьи
- **Со сметаной** — обязательно
- **Национальное блюдо** Беларуси
- **"Бульба"** — картошка (прозвище белорусов)

## Этапы
1. Опрос минчан: где лучшие драники?
2. Посетить 3 места
3. Попробовать с разными начинками:
   - Классические (только картошка)
   - С мясом
   - С грибами
4. Выбрать победителя
5. Научиться готовить у бабушки

## Региональные детали
- **Золотистые** оладьи
- **Сметана** сверху
- **Хрустящие** снаружи, мягкие внутри
- **"Смачна!"** — вкусно!
