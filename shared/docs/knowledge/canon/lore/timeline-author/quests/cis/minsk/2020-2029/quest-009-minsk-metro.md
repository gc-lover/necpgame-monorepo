# Минское Метро

**quest-id:** MINSK-2029-009  
**city:** Минск | **region:** СНГ | **period:** 2020-2029  
**quest-type:** side | **difficulty:** medium | **solo-group:** solo | **time:** 1-2h

## Описание
Исследовать минское метро — самое чистое и красивое метро постсоветского пространства.

## Региональные отсылки
- **Мінскі метрапалітэн** — открыто в 1984
- **Самое чистое** метро (легенда)
- **Советская архитектура** — мрамор, люстры
- **Глубокое** — двойное назначение (бомбоубежище)

## Этапы
1. Тур по красивейшим станциям:
   - "Площадь Ленина" (мрамор, люстры)
   - "Октябрьская" (мозаики)
   - "Площадь Победы" (военная тема)
2. Оценить чистоту (ни пылинки!)
3. Спуститься на самую глубокую станцию
4. Прокатиться по всем линиям
5. Понять: это бомбоубежище тоже

## Региональные детали
- **Чистота** идеальная
- **Мрамор** на станциях
- **Советские мозаики**
- **Глубоко** под землёй
