# Сетевое Ребро

**quest-id:** MOSCOW-2060-021  
**city:** Москва | **region:** СНГ | **period:** 2040-2060  
**quest-type:** main | **difficulty:** hard | **solo-group:** solo | **time:** 2-4h

## Описание
Помочь хакерам создать «Сетевое Ребро» — защищённое соединение между локальными сетями через Blackwall, чтобы обойти фрагментацию NET.

## Этапы
1. **План:** Встреча с хакерским коллективом
2. **Компоненты:** Собрать 5 старых серверов pre-DataKrash
3. **Установка:** Установить серверы в стратегических точках Москвы
4. **Прокладка:** Проложить «ребро» через Blackwall (хакерство DC 24)
5. **Защита:** Отбить атаку NetWatch, пытающихся закрыть ребро

## Выборы
- **Открытый доступ:** Ребро для всех, но менее стабильно
- **Закрытый доступ:** Только для хакеров, стабильнее
- **Монетизация:** Продавать доступ, 5,000 Eddies/месяц пассивно

## Награды
**XP:** 3,000 | **Валюта:** 0-5,000/месяц | **Уникальное:** Доступ к глобальной сети

**Последствия:** Открывает торговлю и контакты с другими городами

## Механики
**Хакерство:** Высокого уровня  
**Строительство:** Установка инфраструктуры  
**Пассивный доход:** Монетизация

## Связи
Глобальное событие, влияет на все города
