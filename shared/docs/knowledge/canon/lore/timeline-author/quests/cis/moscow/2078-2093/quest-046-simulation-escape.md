# Побег из Симуляции

**quest-id:** MOSCOW-2093-046  
**city:** Москва | **region:** СНГ | **period:** 2078-2093  
**quest-type:** main | **difficulty:** extreme | **solo-group:** solo | **time:** 4h+

## Описание
Попытаться выйти из симуляции в «реальный мир». Квест с множественными концовками.

## Этапы
1. **План:** Разработать план побега из симуляции
2. **Компоненты:** Собрать компоненты для устройства выхода
3. **Устройство:** Создать «Прорыватель Симуляции»
4. **Активация:** Активировать устройство
5. **Выход:** Попытка выхода

## Концовки

### Концовка 1: Успех
- Персонаж «выходит» в реальный мир (катсцена)
- Обнаруживает, что реальный мир — киберпанк-антиутопия
- Может вернуться в симуляцию или остаться снаружи

### Концовка 2: Ловушка
- «Выход» оказывается другой симуляцией
- Рекурсия: сколько слоёв симуляций?

### Концовка 3: Отказ
- Решить остаться в симуляции (она комфортнее реальности)
