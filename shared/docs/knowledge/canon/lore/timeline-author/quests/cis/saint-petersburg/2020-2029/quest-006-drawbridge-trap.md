# Ловушка Разводного Моста

**quest-id:** SPB-2029-006  
**city:** Санкт-Петербург | **region:** СНГ | **period:** 2020-2029  
**quest-type:** side | **difficulty:** hard | **solo-group:** solo | **time:** 1-2h

## Описание
Использовать разводные мосты Петербурга для ловушки преследователей.

## Региональные отсылки
- **Разводные мосты** — 21 мост разводится ночью
- **Дворцовый мост** — самый знаменитый
- **График разведения** — строго по расписанию
- **"Застрять на той стороне"** — кошмар петербуржца

## Этапы
1. Погоня по городу ночью
2. Расчёт времени разведения моста
3. Проехать мост за секунду до разведения
4. Преследователи застревают на другой стороне
5. Эвакуация

## Региональные детали
- **Крылья моста** поднимаются
- **Сирена** предупреждает о разведении
- **Корабли** проплывают между крыльями
- **2-4 часа** разведения
