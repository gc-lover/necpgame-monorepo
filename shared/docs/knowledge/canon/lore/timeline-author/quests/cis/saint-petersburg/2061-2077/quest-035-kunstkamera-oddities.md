# Кунсткамера Странностей

**quest-id:** SPB-2077-035  
**city:** Санкт-Петербург | **region:** СНГ | **period:** 2061-2077  
**quest-type:** side | **difficulty:** medium | **solo-group:** solo | **time:** 1-2h

## Описание
Исследовать Кунсткамеру — первый музей России с коллекцией анатомических редкостей и странностей.

## Региональные отсылки
- **Кунсткамера** — создана Петром I (1714)
- **"Кабинет редкостей"** — анатомические уродства
- **Двухголовый телёнок** — знаменитый экспонат
- **Глобус** — огромный глобус-планетарий

## Этапы
1. Посетить Кунсткамеру на Стрелке
2. Осмотреть коллекцию (заспиртованные уродства)
3. Ночью: экспонаты оживают (ИИ-симуляция)
4. Встретить ИИ-Петра I (он создавал музей)
5. Выбор: Сохранить традицию / Модернизировать / Закрыть

## Региональные детали
- **Банки** с заспиртованными экспонатами
- **Анатомический театр** — зал вскрытий
- **Готторпский глобус** — гигантский
- **Мрачная атмосфера** — хоррор
