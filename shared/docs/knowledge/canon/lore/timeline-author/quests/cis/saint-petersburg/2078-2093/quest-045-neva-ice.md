# Лёд Невы

**quest-id:** SPB-2093-045  
**city:** Санкт-Петербург | **region:** СНГ | **period:** 2078-2093  
**quest-type:** social | **difficulty:** easy | **solo-group:** solo | **time:** 1-2h

## Описание
Пройти по замёрзшей Неве зимой — опасная, но традиционная петербургская забава.

## Региональные отсылки
- **Ледоход на Неве** — традиция ходить по льду
- **"Не ходите по льду!"** — предупреждение
- **Крещенские купания** — окунуться в прорубь
- **Зимняя Нева** — толстый лёд

## Этапы
1. Дождаться морозов (Нева замерзает)
2. Рискнуть пройти по льду (DC Смелость 15)
3. Дойти от одного берега до другого
4. Опционально: окунуться в прорубь (Крещение)
5. Вернуться живым

## Региональные детали
- **Толстый лёд** (или трещины?)
- **Морозный воздух** -20°C
- **Прорубь** для купания
- **"Не провалиться!"** — главное
