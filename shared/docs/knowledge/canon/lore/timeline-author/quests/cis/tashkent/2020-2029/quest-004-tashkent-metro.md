# Метро Ташкента

**quest-id:** TASHKENT-2029-004  
**city:** Ташкент | **region:** СНГ | **period:** 2020-2029  
**quest-type:** social | **difficulty:** easy | **solo-group:** solo | **time:** 1-2h

## Описание
Тур по станциям ташкентского метро — подземные дворцы с уникальным дизайном.

## Региональные отсылки
- **Ташкентское метро** — первое в Центральной Азии (1977)
- **Каждая станция** — произведение искусства
- **Запрет на фото** был до 2018 (секретно)
- **Глубокое** — сейсмоопасная зона

## Этапы
1. Тур по красивейшим станциям:
   - "Космонавтов" (космическая тема)
   - "Алишер Навои" (узбекская культура)
   - "Мустакиллик Майдони" (независимость)
2. Оценить мозаики и люстры
3. Узнать: раньше фото были запрещены (военная тайна)
4. Прокатиться по всем линиям

## Региональные детали
- **Мозаики** узбекские
- **Люстры** хрустальные
- **Мрамор** повсюду
- **Советская архитектура** + узбекский стиль
