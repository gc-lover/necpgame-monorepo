## Описание
Понять роль Владивостока как восточных ворот России в Азиатско-Тихоокеанский регион.

## Региональные отсылки
- **"Владей Востоком!"** — название города
- **Восточный форпост** России
- **АТР (Азиатско-Тихоокеанский регион)** — связь с Азией
- **Порт** — один из крупнейших

## Этапы
1. Изучить историю Владивостока:
   - Основан в 1860
   - "Владей Востоком" — название
   - Восточный форпост
2. Порт Владивостока — контейнеры, корабли
3. Связь с Азией:
   - Япония (600 км)
   - Китай (граница)
   - Корея (море)
4. АТЭС-2012 — саммит здесь
5. Понять: Владивосток = окно в Азию
