# Гонки в Метро

**quest-id:** LONDON-2039-015  
**city:** Лондон | **region:** Европа | **period:** 2030-2039  
**quest-type:** event | **difficulty:** medium | **solo-group:** solo | **time:** 30-60min

## Описание
Гонки на ховербайках по туннелям лондонского метро (Underground).

## Региональные отсылки
- **The Tube** — знаменитое метро Лондона
- **Mind the Gap** — фраза используется в гонках
- **Круглые туннели** — уникальная геометрия трассы

## Этапы
1. Регистрация на станции Piccadilly Circus
2. Модификация ховербайка
3. Гонка по Circle Line (кольцевая линия)
4. Финиш на станции King's Cross

## Особенности
- **Круговая трасса:** 3 круга по Circle Line
- **Препятствия:** Движущиеся поезда, ремонтные дроны
- **Голос объявлений:** «Mind the Gap» при прыжках

## Награды
**1 место:** 15,000 Eddies, легендарный ховербайк «Underground Racer»  
**2-3 место:** 5,000-8,000 Eddies  
**Участие:** Репутация гонщиков +1

## Связи
Weekly event
