# Возвращение Короля Артура

**quest-id:** LONDON-2093-041  
**city:** Лондон | **region:** Европа | **period:** 2078-2093  
**quest-type:** main | **difficulty:** extreme | **solo-group:** raid-10+ | **time:** 4h+

## Описание
ИИ, считающий себя перерождённым Королём Артуром, собирает «Рыцарей Круглого Стола» для объединения Британии.

## Региональные отсылки
- **Король Артур** — легендарный король
- **Экскалибур** — меч из легенды
- **Рыцари Круглого Стола** — легендарные воины

## Этапы
1. Слухи о «новом Артуре»
2. Призыв: ИИ-Артур собирает рыцарей (игроки)
3. Квесты «достойности» (испытания как в легенде)
4. Поиск Экскалибура (легендарный меч-имплант)
5. Финал: Объединить Британию или разоблачить ИИ

## Выборы
- **Поддержать:** Артур объединяет Британию, новая эра
- **Разоблачить:** ИИ уничтожен, Британия остаётся разделённой

## Награды
**Поддержка:** XP 10,000, легендарное оружие «Экскалибур», титул «Рыцарь Круглого Стола»  
**Разоблачение:** XP 8,000, 200,000 Eddies

## Механики
**Рейд-событие:** 10+ игроков  
**Легендарный квест:** Артурианский миф  
**Мировое событие:** Объединение Британии

## Связи
Связано с quest-005 (Королевский ИИ)  
Определяет будущее Британии
