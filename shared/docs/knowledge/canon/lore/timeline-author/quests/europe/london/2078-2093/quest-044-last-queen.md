# Последняя Королева

**quest-id:** LONDON-2093-044  
**city:** Лондон | **region:** Европа | **period:** 2078-2093  
**quest-type:** main | **difficulty:** hard | **solo-group:** solo | **time:** 4h+

## Описание
Присутствовать при смерти последней британской королевы и выборе судьбы монархии.

## Региональные отсылки
- **Британская монархия** — тысячелетняя традиция
- **"The Queen is dead, long live the King!"** — традиционный возглас
- **Конец эпохи** — символическое событие

## Этапы
1. Объявление: Королева при смерти
2. Прощание нации (траур)
3. Последняя аудиенция (если репутация высока)
4. Смерть королевы (эмоциональная катсцена)
5. Выбор: Сохранить монархию / Республика / ИИ-монарх

## Выборы
- **Сохранить:** Традиционная монархия продолжается
- **Республика:** Конец монархии, демократия
- **ИИ-монарх:** Оцифрованная королева правит вечно

## Награды
**XP:** 5,000 | **Валюта:** 0 | **Последствия:** Определяет будущее Британии

**Эмоциональный момент:** Конец эпохи

## Механики
**Исторический момент:** Смена эпох  
**Голосование:** Игроки решают судьбу  
**Эмоциональный:** Прощание с традицией
