# Парк Ретиро

**quest-id:** MADRID-2029-005  
**city:** Мадрид | **region:** Европа | **period:** 2020-2029  
**quest-type:** social | **difficulty:** easy | **solo-group:** solo | **time:** 1-2h

## Описание
Прогулка по парку Ретиро — "зелёные лёгкие" Мадрида.

## Региональные отсылки
- **Parque del Retiro** — 125 гектаров
- **Хрустальный дворец (Palacio de Cristal)** — стекло и металл
- **Estanque Grande** — пруд с лодками
- **Памятник Альфонсо XII**

## Этапы
1. Войти в парк Ретиро
2. Прогулка к Хрустальному дворцу (стеклянный!)
3. Пруд: арендовать лодку
4. Гребля по пруду
5. Памятник Альфонсо XII — колоннада

## Региональные детали
- **Хрустальный дворец** — весь из стекла
- **Пруд** с лодками
- **Парк** огромный
- **Мадридцы** отдыхают здесь
