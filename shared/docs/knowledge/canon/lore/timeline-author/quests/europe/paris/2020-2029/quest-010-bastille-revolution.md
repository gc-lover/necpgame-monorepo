# Революция Бастилии

**quest-id:** PARIS-2029-010  
**city:** Париж | **region:** Европа | **period:** 2020-2029  
**quest-type:** main | **difficulty:** extreme | **solo-group:** group-5-10 | **time:** 4h+

## Описание
Возглавить новую революцию, вдохновлённую штурмом Бастилии 1789 года — восстание против корп-элиты.

## Региональные отсылки
- **Штурм Бастилии** — 14 июля 1789 (начало Французской революции)
- **"Liberté, Égalité, Fraternité"** — лозунг революции
- **Гильотина** — символ революции
- **Марсельеза** — гимн Франции

## Этапы
1. Собрать революционную толпу на Place de la Bastille
2. Марш по Парижу с флагами
3. Штурм корп-здания (символическая Бастилия)
4. Провозглашение новой республики
5. Выбор: Террор / Умеренность / Компромисс

## Выборы
- **Террор:** Гильотина для элиты (радикально)
- **Умеренность:** Реформы без крови
- **Компромисс:** Конституционная монархия

## Региональные детали
- **Триколор** — синий, белый, красный флаг
- **Фригийский колпак** — символ свободы
- **"Aux armes, citoyens!"** — к оружию, граждане!
- **Баррикады** на улицах
