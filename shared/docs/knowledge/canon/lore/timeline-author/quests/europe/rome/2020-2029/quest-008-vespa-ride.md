# Поездка на Веспе

**quest-id:** ROME-2029-008  
**city:** Рим | **region:** Европа | **period:** 2020-2029  
**quest-type:** social | **difficulty:** easy | **solo-group:** solo | **time:** 1-2h

## Описание
Прокатиться на Vespa по улицам Рима — иконичный итальянский скутер и стиль жизни.

## Региональные отсылки
- **Vespa** — культовый итальянский скутер (с 1946)
- **"Roman Holiday"** — фильм с Одри Хепберн на Веспе
- **Dolce Vita** — сладкая жизнь
- **Chaotic traffic** — хаотичное движение Рима

## Этапы
1. Арендовать Vespa (классическую модель)
2. Поездка по Риму:
   - Вокруг Колизея
   - Через Via Veneto
   - К Испанской лестнице
   - Финал у Треви
3. Лавировать между машинами
4. Parkour на скутере (итальянский стиль)
5. Gelato stop — остановка на джелато

## Региональные детали
- **Vespa** яркого цвета
- **Хаос** на дорогах (но работает)
- **"Bella figura"** — красивый стиль
- **Gelato** обязателен
