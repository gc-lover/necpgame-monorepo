## 1. Методика детализации

- **Слои визуала:** базовая форма, функциональные модули, подсветка, динамические эффекты.
- **Палитры:** указаны основные цветовые схемы, вторичные акценты и поведение подсветки.
- **Материалы:** металл, композит, кожа, ткань, синтетика, голографические материалы.
- **Динамика:** описаны эффекты при движении, атаке, использовании способностей.
- **Фракции и бренды:** каждый образ связан с корпорацией или бандой, указаны логотипы и узоры.
- **Модули микроанимации:** вспышки, пар, искры, глитч-эффекты.
- **Микросервисы:** `character-service` (персонажи, NPC), `gameplay-service` (оружие, импланты), `economy-service` (предметы, торговля), `social-service` (социальные роли, NPC связи).

## 2. Архетипы игроков и ключевых NPC — расширенные профили

### 2.1 Corpo Operative

- **Силуэт:** прямые линии, угловатые плечи, усиленные предплечья. Плащ с встроенными голографическими нитями.
- **Материалы:** черный титан, хромированные вставки, стеклопластик. Лакированные детали отражают свет.
- **Акценты:** лазурные прожилки вдоль позвоночника и предплечий; голограммы логотипов Arasaka, Zetatech.
- **Микроэффекты:** при использовании имплантов — вспышки голубого света, цифровой ореол вокруг головы.
- **Социальные атрибуты:** кейс-дрон с инкрустацией, нано-галстук с динамической анимацией.
- **Фракционный след:** коридоры корпораций окрашены под акценты персонажа; NPC реагируют на корпоративную маркировку.

### 2.2 Street Ronin

- **Силуэт:** асимметрия, слойные плащи, свободные лямки. Киберклинки и кабели, свисающие с пояса.
- **Материалы:** потертый деним, переработанный металл, кожа с LED-нитями.
- **Палитра:** кислотно-зеленый, пурпурный, оранжевые искры.
- **Микроэффекты:** татуировки меняют рисунок при повышении адреналина; дым, исходящий из имплантов.
- **Фракции:** Neon Ronin, Tyger Claws — соответствующие граффити на одежде.

### 2.3 Nomad Mechanist

- **Силуэт:** экзокаркас, усиленные суставы, модульные ранцы. Пылевые фильтры на шлеме.
- **Материалы:** пескоструйный металл, кожа, композитные пластины.
- **Эффекты:** янтарные индекс-LED, пыльные шлейфы при беге, голограмма маршрулов.
- **Социальные элементы:** набор инструментов, набедренные сумки, талисманы клана.

### 2.4 Synth Mystic

- **Силуэт:** плащи из световых нитей, полупрозрачные ткани, парящие аксессуары.
- **Свет:** фиолетово-розовые переходы, голографические символы вокруг головы.
- **Динамика:** при использовании способностей — точечные вспышки, стеклянные аккорды.
- **Фракции:** Synth Faith, AI-культы; символы ИИ, парящие цифровые руны.

### 2.5 Дополнительные роли NPC

- **Data Broker:** костюм с портами, голографический воротник, перчатки с лазерной сеткой, голографические папки-кластеры.
- **Synth Street Healer:** мягкое пастельное свечение, биомониторы на запястьях, карманные мед-юниты.
- **Corporate Handler:** нанопокрытие плаща, бронированные наплечники, кейс-дрон.
- **Nomad Quartermaster:** утилитарный плащ, экзоскелет для переноски грузов, янтарные очки.
- **Fixer Elite:** элегантные костюмы с LED-вставками, кольца-коммуникаторы, сигариллы с неоновым дымом.
- **Gang Lieutenant:** шипованный бронежилет, светящиеся шрамы, металлические клыки.
- **Trauma Team Commander:** бело-красный бронекостюм, дроны-помощники, динамические пиктограммы.

## 3. Оружие — глубокая детализация

### 3.1 Пистолеты

- **Smart Pistols:** карбоновый корпус, синие линии, голографический прицел. Магнитная рельса подсвечивает траекторию.
- **Tech Pistols:** прозрачные камеры, видимые конденсаторы, янтарные индикаторы перегрева.
- **Динамика:** при выстреле — вспышки, цифровой след пули, звуковой резонанс.

### 3.2 Штурмовые винтовки

- **Militech Assault:** оливковый + янтарь, ячейки гибернации модулей, лазерные указатели.
- **Arasaka Precision:** черный, белый, красный глянец, плавные линии, подсветка затвора.
- **Zetatech Adaptive:** полупрозрачные панели, лазурные подстветки, голографические индикаторы модификаций.

### 3.3 Снайперские системы

- **Корпус:** монолитный ствол, гравитационные стабилизаторы, цифровые чехлы.
- **Визуальные эффекты:** прозрачные батареи, подсветка при прицеливании, зернистые голограммы ветра.
- **Пристрелки:** видимый лазерный луч, паровые эффекты при выстреле.

### 3.4 Дробовики

- **Городские модели:** оранжево-серые кожухи, предупреждающие полосы, голографический конус поражения.
- **Тактические:** металлические планки, красные индикаторы отдачи, встроенный сенсор перегрева.

### 3.5 Тяжёлые платформы

- **Railguns:** магнитные рельсы, красные рёбра, мощные вспышки.
- **Miniguns:** вращающиеся барабаны, динамическое освещение, предупреждающие пиктограммы.

### 3.6 Метательное и тактическое оружие

- **Лезвия:** свечение по кромке, голографические руны, следы в воздухе.
- **Гранаты:** полупрозрачные гильзы, голографические таймеры, цветовая кодировка эффектов.

## 4. Импланты и моды

### 4.1 Нейрошунты

- **Внешний вид:** серебристые пластины вокруг шеи, мини-гравировки, импульсные вспышки.
- **Функционал:** отображают статус, соединение с сетями. Установка — `character-service`.

### 4.2 Когнитивные усилители

- **Дизайн:** полупрозрачные капсулы над висками, пульсирующие нейрофиламенты.
- **Эффекты:** мягкое фиолетовое свечение, гличи при перегрузке.

### 4.3 Боевые усилители мышц

- **Внешний вид:** карбоновые ребра по рукам, энергетические жгуты с янтарным свечением.
- **Первичная реакция:** вспышки, усиленные шаги, пар.

### 4.4 Кожные хроматические пластины

- **Текстуры:** переключение между металлическим блеском, карбоном, голографической сеткой.
- **Цвета:** реакция на эмоции персонажа (спектр от холодного голубого до агрессивного красного).

### 4.5 Сенсорные сетки

- **Видимость:** оптоволоконные линии под кожей, подсветка при взаимодействии.
- **Применение:** поддержка боевых и социальных сценариев.

### 4.6 Тактильные моды и нейрослоты

- **Аксессуары:** накладки с микрошипами, магнитные соединители.
- **Нейрослоты:** док-станции в плечах для дронов, зелёно-янтарная индикация.

### 4.7 Системы катализаторов эмоций

- **Дизайн:** кристаллические механизмы в груди, меняющие цвет.
- **Интеграция:** `social-service` (реакции NPC, репутация).

### 5.4 Кибер-шлемы и рюкзаки

- **Шлемы:** панорамные HUD, маскировка, проекции переговоров.
- **Рюкзаки:** жёсткие каркасы, платформы для дронов, индикаторы заряда.

### 5.5 Имплантируемые аксессуары

## 10. JSON схемы и DTO

- `CharacterVisualProfileDetailed` (`schemas/world/character-visual-detailed.schema.json`) — добавляет `gearLayers`, `ambientAnimations`, `emissiveBehavior`.
- `WeaponVisualProfileDetailed` (`schemas/combat/weapon-visual-detailed.schema.json`) — поля `recoilFx`, `heatStages`, `altFireModes`.
- `ImplantVisualProfile` (`schemas/social/implant-visual-profile.schema.json`) — `implantType`, `socket`, `activationFx`.
- `ItemPreviewPayload` (`schemas/economy/item-preview.schema.json`) — `assetId`, `materials`, `lighting`, `particleFx`.

### Пример `WeaponVisualProfileDetailed`

```json
{
  "assetId": "ASSET-WEAPON-AR-MIL-010",
  "class": "assault",
  "manufacturer": "Militech",
  "materials": ["carbon", "titanium"],
  "effects": ["muzzle-flare-amber", "cooling-shutters"],
  "recoilFx": {
    "type": "vertical-kick",
    "lightPattern": "amber-strobe"
  },
  "heatStages": [
    { "threshold": 0.3, "color": "#FFAE00" },
    { "threshold": 0.6, "color": "#FF4C00" },
    { "threshold": 0.9, "color": "#FF0000" }
  ],
  "altFireModes": [
    { "mode": "burst", "fx": "laser-guidance" }
  ]
}
```

## 11. REST и Kafka интеграции

- `GET /world/npcs/{npcId}/visual/detailed` → `CharacterVisualProfileDetailed`.
- `GET /combat/weapons/{weaponId}/visual` → возвращает расширенный профиль оружия.
- `GET /economy/items/{itemId}/preview` → карточка предмета с визуальными эффектами.
- `POST /social/implants/visual/export` → выгрузка визуала имплантов для UI.
- Kafka:
  - `world.visual.assets.updated` — `{ assetId, category, version, updatedAt }`
  - `combat.visual.assets.updated` — синхронизация оружия/имплантов.
  - `economy.visual.items.updated` — визуальные данные для магазинов.
