## Цель документа

- Установить единый визуальный язык для ключевых типов персонажей, оружия, экипировки и предметов.
- Подготовить основу для описаний в базе данных и UI, включая текстовые лоры и превью.
- Обеспечить передаваемость настроения CYBERPUNK 2077, сохраняя авторский стиль NECPGAME.

## Общие принципы визуального стиля

- **Контраст:** сочетание неоновых акцентов с приглушёнными базовыми материалами (карбон, матовый пластик, композиты).
- **Слойность:** каждый ассет имеет минимум три визуальных слоя (база, функциональные элементы, акценты/свет).
- **Износ:** контролируемый wear-and-tear отображает происхождение (корпоративное = минимальный износ, уличное = явные следы эксплуатации).
- **Силуэт:** читается на расстоянии, подчёркивает роль (боевые классы = массивные формы, разведка = стреловидные силуэты, поддержка = модульность).
- **Брендинг:** визуальные мотивы мегакорпораций и банд интегрированы в материалы, узоры и логотипы.

## Оружие

### Классы дальнего боя

**Пистолеты (Smart & Tech)**
- Корпус: матовый металл с магнитными гарнитурами для модулей.
- Акценты: линии синего/янтарного цвета вдоль ствола, голографические индикаторы боезапаса.
- Особенность: смарт-рельсы, подсвечивающие траекторию подконтрольных боеприпасов.

**Штурмовые винтовки**
- Корпуса из карбона с полимерными вставками; двойные магазины, соединённые оптоволоконными жгутами.
- Цветовые схемы зависят от происхождения: Militech (оливковый + янтарь), Arasaka (чёрный + белый + красный глянец).
- Динамические жалюзи охлаждения, подсвеченные при автоматическом огне.

**Снайперские системы**
- Длинные монолитные стволы, гравитационные стабилизаторы, цифровые чехлы с голографическими прицелами.
- Декор: монохромные поверхности с лазерными гравировками координат.
- Особенность: полупрозрачные энергетические батареи, подсвечивающие оставшийся заряд.

**Энергетическое оружие**
- Плотные цилиндрические корпуса, кольца ускорителей с ярким голубым свечением.
- Материалы: керамика, стеклокомпозит, магнитные слои.
- Визуальные эффекты: плазменные дуги вокруг ствола, мерцающие конденсаторы.

**Дробовики**
- Короткие, массивные стволы с перфорированными кожухами, усиленные приклады с амортизирующими подушками.
- Палитра: индустриальные серо-оранжевые тона, предупреждающие полосы.
- Эффекты: яркие вспышки с разлётами искр, голографический конус поражения.

**Тяжёлые платформы**
- Рельсовые пушки и миниганы с вращающимися барабанами, механизированные манжеты.
- Материалы: армиранный титан, магнитные рельсы, керамические компенсаторы.
- Акценты: жёлтые габаритные огни, динамические предупреждения об отдаче.

**Метательное оружие и тактические заряды**
- Аэродинамические лезвия с световыми кромками, импульсные гранаты с прозрачными гильзами.
- Голографические таймеры, меняющие цвет от зелёного к красному.
- При активации возникают кратковременные глич-эффекты в воздухе.

### Ближний бой и имплант-артефакты

**Моноблейды**
- Лезвие из чёрного наноугля с почти невидимой кромкой, подсветка проходит по ребру.
- Рукоять: кожаные вставки, электродные пластины, голографические руны бренда.

**Электрохлысты**
- Сегментированное тело с гибкими магнитными соединениями, спиральное свечение.
- Хранение: на бедре удерживается полем, при активации раскручивается с искрами.

**Интегрированные имплант-оружия**
- Киберруки-трансформеры: видимые швы, вспышки при смене режима, встроенные голограммы безопасности.
- Плечевые мини-турели: складываются в бронепластины, при активации раскрываются с механическим щелчком.

## Прочие предметы и NPC-атрибуты

**Обычные дроны сопровождения**
- Обтекаемые корпуса, перестраиваемые крылья, динамическая подсветка состояния (зелёный = патруль, красный = бой).
- Голографические маски, проецирующие корпоративные логотипы или бандитские граффити.

**Кибердеки и взломщик-модули**
- Плоские блоки с стеклопанелями, комбинированные с металлическими ребрами.
- Интерактивные глифы, смена цветов в зависимости от уровня доступа.
- Подсветка портов и кабелей, амбиентные звуковые эффекты, синхронизированные с визуалом.

**Медицинские стимы**
- Капсулы с полупрозрачными стенками, внутри — вихревые жидкости.
- Цветовая палитра: целительные (бирюзовый), адреналин (красный), стабилизация нервов (индиго).
- Голографические ярлыки и QR-коды для сканирования имплантами.

**Полевые мастерские Nomad**
- Модули-контейнеры с выдвижными столами, техника на магнитных рейках.
- Песчаные покрытия, сигнальные флажки неонового цвета, солнечные панели, раскладывающиеся из стен.

**Святыни Synth-культов**
- Парящие кристаллические структуры, полупрозрачные панели со сквозным свечением.
- Монохромные фигуры в плащах из света, проецирующие символику ИИ.

## Импланты и моды

- **Нейрошунты:** гладкие серебристые пластины вокруг шеи, импульсные вспышки при активации, микрогравировки с кодами владельца.
- **Когнитивные усилители:** полупрозрачные капсулы над висками с пульсирующими нейрофиламентами, мягкое фиолетовое свечение.
- **Боевые усилители мышц:** наружные карбоновые ребра вдоль рук и ног, энергетические жгуты, мерцающие при рывках.
- **Кожные хроматические пластины:** динамические узоры, синхронизированные с эмоциями, возможность переключения текстур (металл, карбон, голографическая сетка).
- **Сенсорные сетки:** тонкие оптоволоконные линии под кожей ладоней, подсвеченные при взаимодействии с интерфейсами.
- **Тактильные моды:** съёмные накладки на пальцы с микрошипами и магнитными контактами, голографические индикаторы давления.
- **Нейрослоты для дронов:** имплантированные в плечи док-станции, механические соединители, индикация статуса зелёно-янтарными огнями.
- **Системы катализаторов эмоций:** встроенные в грудную клетку кристаллические механизмы, меняющие цвет от спокойного голубого до агрессивного красного.

## REST и Kafka интеграция

- `GET /world/npcs/{npcId}` → `visualProfile` (CharacterVisualProfile).
- `GET /social/npc/{npcId}` → включает `appearance.assetId` и `palette`.
- `GET /combat/weapons/{weaponId}` → `visualProfile` и эффекты.
- `POST /inventory/items/previews` → получает `assetId` и возвращает карточку.
- Kafka `world.visual.assets.updated` уведомляет фронтенд о смене ассетов.

## DTO и фронтенд

- `world/npcs/*.json` и `combat/weapons/*.json` генерируются скриптом `scripts/export-visual-assets.ps1`.
- FRONT-WEB: модуль `shared/ui/visual` читает JSON, чтобы строить карточки персонажей/оружия.
- UI компоненты указаны в таблице asset registry.
