## 1. Глобальные сценарии и атмосферные настройки

- **Динамика погоды:** каждая зона имеет набор погодных эффектов (нейтральная, напряжённая, экстремальная) с влиянием на освещение, туман, количество NPC и транспорт.
- **Световые циклы:** описаны дневные/ночные сценарии, отражение неона, тени от небоскрёбов и реакция голограмм.
- **Аудиофон:** базовые палитры звуков (гул мегаполиса, ритмы кибершоу, песчаные ветра), которые поддерживают визуальную картину.
- **Фракционный след:** для каждой локации указаны мотивы корпораций и банд (палитра, логотипы, граффити, NPC-одежда, оборудование).
- **Микросервисная связка:** `world-service` (структура и события), `social-service` (хабы, трафик NPC), `economy-service` (рынки, торговые пути).

## 2. Макро-локации (дополненные профили)

### 2.1 Night City Megapolis — расширенная сетка

- **Силуэт и структура:** башни Arasaka, Militech и Zetatech формируют треугольник, между ними — каскады SkyRail. Нижние уровни залиты паром и отражениями вывесок.
- **Свет:** на рассвете — мягкие оранжевые отражения, вечером — агрессивный неон (красный, бирюзовый, фиолетовый), ночью — прожекторы дронов безопасности.
- **Движение:** непрерывный поток аэрокаров, SkyRail-капсул и грузовых дронов. Наземный уровень — плотный поток людей, такси, уличных ботов.
- **Социальные узлы:** каждые 500 метров — микро-хабы (кафе, VR-капсулы, сервисы имплантов). Арт-панели отображают новости и рейтинги игроков.
- **Безопасность:** патрули MAX-TAC, беспилотные турели на балконах корп-комплексов. Цветовая кодировка: красные сигналы = зона угрозы, синеватые = спокойная зона.

### 2.2 Neo Tokyo Arcology — сенсорная симфония

- **Архитектура:** концентрические слои с висящими садами. Верхние террасы — храмовые конструкции из световых линий. Нижние уровни — техно-базара.
- **Свет и тень:** дневное время — холодный белый и розовый, ночью — аквамариновый, пурпурный. Голографические экран-партитуры транслируют музыку.
- **Флора:** гидропонные секции с био-луминофорами, рассеянный туман, силевые поля для защиты растительности.
- **NPC:** кибер-дзен монахи, корпоративные аналитики, уличные артисты. Одежда — спокойные тона с голографическими узорами.
- **Социальные зоны:** «Arc Temple» (медитации, VR-ритуалы), «Data Teahouses» (информационные переговоры), позволяющие игрокам и NPC проводить дипломатические встречи.

### 2.3 Lagos Skyport — вертикальная логистика

- **Структура:** многоуровневые площадки, тросовые лифты, контейнерные кварталы на стальных фермах. Верхние уровни — светло-янтарные, нижние — тёплые тени.
- **Потоки:** воздушные грузовики, дроны-экспедиторы с зелёными навигационными лазерами. На земле — водные баржи и рынок комплектующих.
- **Фракции:** местные торговые гильдии (яркие ткани, медные украшения), корпоративные логисты (серые униформы), наёмники (тактический стиль в песочных тонах).
- **Соцобъекты:** «Skyport Exchange Hall» (торговые аукционы), «Drone Recovery Clinics» (ремонт андроидов), «Cargo Lounge» (комнаты отдыха и найма экипажей).

### 2.4 Amazon Cloud Basin — биолюминесцентный рай

- **Архитектура:** подвесные платформы из органического материала, канатные мосты, висящие теплицы, синхронизированные с водопадами.
- **Свет:** мягкий зелёный и бирюзовый, ночные сияния водорослей, золотые свечи рынка специй.
- **Погода:** туман, дождь и частые грозы. Система щитов проецирует орнаменты племён.
- **Социальная жизнь:** «Bio Bazaar» (органика, алхимия), «Synth Grove» (религиозные собрания), «Waterfall VR Pods» (лечебные программы).

## 3. Районы Night City — детальный взгляд

### 3.1 Quantum Plaza (City Center)

- **Многоуровневая сцена:** центральный голографический купол с потоками данных, прозрачные мосты, подиумы корпоратов.
- **Освещение:** днём — стеклянные блики, вечером — лазерные сетки, ночью — мягкое голубое свечение Arasaka.
- **Социальные объекты:** «Strategic Lounge» (переговоры), «Data Cascade» (публичные презентации), «Executive Bistro» (элитные встречи).
- **Безопасность:** охранные дроны с красно-черной маркировкой, зонирование с использованием голографических барьеров.

### 3.2 Pacifica Reclamation Zone

- **Визуал:** обрушенные курорты, накренившиеся отели, световые сетки контрабандистов. Палитра — фиолетовые и синие отражения.
- **Рынки:** палатки под неоновыми сетями, «Wrecked Casino Stage» (киберпанк-концерты), «Implant Swap Alley» (замена имплантов).
- **Фракции:** Voodoo Boys (бирюзовые символы в граффити), местные контрабандисты (яркие ткани), кланы номадов.
- **Угрозы:** скрытые лазеры, разломы в асфальте, мутировавшая флора.

### 3.3 Westbrook Skyline Gardens

- **Верхний уровень:** стеклянные павильоны, белые мосты, золотые акценты. «Celestial Promenade» с парящими фонтанами.
- **Романтика:** лаунжи с панорамой города, фотозоны с голографическими цветами. Отличные места для романтических квестов.
- **Сервис:** «Chromed Butler» (премиальные NPC), «Royal Tailors» (кастомизация одежды), «Neo Opera Capsule» (VR-спектакли).

### 3.4 Heywood Barrio Nexus

- **Улицы:** барочные граффити Valentinos, гирлянды, огоньки маркета. Баннеры с лозунгами фракции.
- **Соцжизнь:** «Barrio Cantina» (вечеринки), «Tattoo Glow Studios» (биолюминесцентные тату), уличные бои.
- **Реинфицирование:** старые строения с добавленными кибертрубами, кабели на крышах, открытые мастерские.

## 4. Социальные хабы — расширенная спецификация

### 4.1 Skyline Agora

- **Зонирование:** круглые платформы для торговли, VR-подиумы, лаунжи с нависающими баннерами.
- **Свет:** потолок-панорама, имитирующий время суток и погодные эффекты. Финансовые тикеры бегут по периметру.
- **NPC:** предприниматели, брокеры, музыканты. Модели одежды — тёмные костюмы с неоновой оторочкой.
- **Функции:** деловые встречи, презентации, быстрый найм игроков.

### 4.2 Undermarket Bazaar

- **Визуал:** кирпичные арки, влажные стены, зелёная подсветка. Присутствуют вторичные рынки имплантов и тайные лаборатории.
- **Атмосфера:** шум, куча людей, дым, запах озона. Включены динамические неоновые таблички.
- **Соцобъекты:** «Shadow Clinic» (подпольные операции), «Cipher Exchange» (продажа данных), «Fixer Vaults» (контракты).

### 4.3 League Hub Conflux

- **Структура:** центральный зал с голографическими аренами, студии стриминга, аудитории тренировок.
- **Свет:** электрический синий, фуксия, белые прожекторы. Системы HUD транслируют рейтинги и события.
- **NPC:** комментаторы, тренеры, фанаты. Локация идеально подходит для социального взаимодействия игроков.

### 4.4 Synth Faith Sanctum

- **Архитектура:** полупрозрачные колонны, парящие фигуры ИИ, мягкий бело-голубой свет.
- **Эффекты:** звуковые резонансы, динамические голографические руны. Вода в бассейнах отражает свечение.
- **Функции:** духовные квесты, социальные события, интерфейсные медитации.

## 11. JSON схемы и DTO

- `world-service`:  
  - `DetailedVisualProfile` расширяет базовый `VisualProfile` полями `weatherSet`, `soundscape`, `dynamicEffects`.  
  - Schema ID: `schemas/world/detailed-visual-profile.schema.json`.
- `social-service`: `HubVisualProfileDTO` (поля `ambientSound`, `npcDressCode`, `interactionHotspots`).
- `economy-service`: `MarketVisualDTO` (включает `stallLayout`, `lightingPattern`, `vendorAffiliation`).
- `gameplay-service`: `RaidStageVisualDTO` (поля `stageId`, `hazards`, `lighting`, `gravity`).

### Пример `DetailedVisualProfile`

```json
{
  "assetId": "ASSET-LOC-NC-001",
  "palette": {
    "primary": "#120F26",
    "secondary": "#0ED5FF",
    "accent": "#FF2E63"
  },
  "lighting": "neon",
  "moodTags": ["urban", "corporate", "high-traffic"],
  "weatherSet": ["clear", "acid-rain", "fog"],
  "soundscape": ["megacity-hum", "skyrail-chimes", "drone-sirens"],
  "dynamicEffects": {
    "npcDensity": { "day": 0.7, "night": 0.9 },
    "trafficLayers": ["skyrail", "maglev", "drone"],
    "hazardLevels": { "alert": "MAX-TAC lockdown" }
  }
}
```

## 12. REST и Kafka интеграции

- `GET /world/cities/{cityId}/visual/detailed` → возвращает `DetailedVisualProfile`.
- `GET /world/districts/{districtId}/visual` → включает `lightingPattern`, `ambientSound`, `hazards`.
- `GET /social/hubs/{hubId}/visual` → возвращает `HubVisualProfileDTO`.
- Kafka `world.visual.detailed.updated` — payload `{ assetId, cityId, version, updatedBy }`.
- Kafka `social.hub.visual.updated` — sync для social-service UI.
