## Общее описание

Диалоги и разговоры в игре. Включает диалоги NPC, диалоги квестов, диалоги событий, ветвление диалогов и романтические диалоги.

---

## Типы диалогов

### Диалоги NPC
- Приветствия NPC
- Обычные диалоги
- Диалоги торговли
- Диалоги квестов

### Диалоги квестов
- Диалоги принятия квестов
- Диалоги выполнения квестов
- Диалоги завершения квестов
- Диалоги ветвления квестов

### Диалоги событий
- Диалоги мировых событий
- Диалоги событий фракций
- Диалоги событий локаций
- Диалоги динамических событий

### Ветвление диалогов
- Варианты ответов
- Ветвление по выбору
- Ветвление по репутации
- Ветвление по характеристикам

### Романтические диалоги
- Романтические линии
- Диалоги отношений
- Диалоги развития отношений
- Диалоги романтических событий

---

## Шаблоны и структура

- Все новые диалоги создавайте по файлу `DIALOGUE-TEMPLATE.md`.
- Одна сущность (NPC, квест, событие) оформляется в отдельном файле внутри директории `dialogues/`.
- При расширении набора диалогов сверяйтесь с шаблоном и обновляйте его при появлении новых полей.

---
