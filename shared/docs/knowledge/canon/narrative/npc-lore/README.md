## Общее описание

Детальные описания NPC и их историй. Включает историю NPC, отношения, диалоги и развитие персонажей.

---

## Структура раздела

### important/ - Важные NPC
Важные NPC для игры:
- Лидеры фракций
- Ключевые персонажи сюжета
- Влиятельные персонажи
- Фиксеры и посредники

### common/ - Второстепенные NPC
Второстепенные NPC для игры:
- Торговцы
- Обычные горожане
- Враги и бандиты
- Служебные NPC

### NPC-TEMPLATE.md
Шаблон для создания новых NPC. Используйте этот шаблон при создании новых NPC.

### important-npcs-list.md
Список важных NPC с кратким описанием каждого.

### npc-status-tracker.md
Трекер статусов всех NPC с краткой информацией и статусом работы над каждым NPC.

---

## Правила создания NPC

1. **Используйте шаблон:** Всегда используйте `NPC-TEMPLATE.md` при создании новых NPC
2. **Категоризация:** Определите, является ли NPC важным или второстепенным
3. **Детализация:** Детализируйте историю, личность и отношения NPC
4. **Согласованность:** Следите за согласованностью с общим лором
5. **Вдохновение:** Используйте лор Cyberpunk и авторские элементы
