## Общее описание

Второстепенные NPC для NECPGAME. Эти NPC менее важны для сюжета, но играют важную роль в игровом мире как торговцы, обычные горожане, враги и служебные NPC.

---

## Структура раздела

### traders/ - Торговцы
Торговцы и продавцы:
- Независимые торговцы
- Торговцы оружием
- Торговцы имплантами
- Торговцы ресурсами

### citizens/ - Обычные горожане
Обычные горожане Night City:
- Рабочие
- Служащие
- Бездомные
- Другие горожане

### enemies/ - Враги и бандиты
Враги и бандиты:
- Бандиты
- Корпоративные агенты (враги)
- Киберпсихотики
- Другие враги

### service/ - Служебные NPC
Служебные NPC:
- Риппердоки
- Медицинские работники
- Информаторы
- Другие служебные NPC

---

## Правила создания второстепенных NPC

1. **Используйте шаблон:** Всегда используйте `NPC-TEMPLATE.md` при создании новых NPC
2. **Категоризация:** Определите категорию NPC (торговец, горожанин, враг, служебный)
3. **Детализация:** Детализируйте историю, личность и функции NPC
4. **Согласованность:** Следите за согласованностью с общим лором
5. **Функциональность:** Фокусируйтесь на игровых функциях NPC
