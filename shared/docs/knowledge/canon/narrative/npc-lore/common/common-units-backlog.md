## Примечания к планированию
- На каждую категорию запланировать минимум 5–10 вариаций для MVP+, с расширением в сезонах.
- Все common NPC должны иметь краткие роли, простые циклы заданий, и базовую реактивность к репутации и действиям игрока.
- Для боевых фракций — привязка к локациям/случайным эвентам.
- Для сервисов — связка с экономикой/логистикой.
