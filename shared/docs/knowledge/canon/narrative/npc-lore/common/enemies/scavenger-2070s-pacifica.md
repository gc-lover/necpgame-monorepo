## Профиль

Член банды Scavengers — мародёров, специализирующихся на краже кибервара с живых и мёртвых жертв. Опасны, работают группами, устраивают засады. Не брезгуют ничем ради наживы.

## Боевые характеристики

- Средний уровень угрозы  
- Использует разношёрстное оружие  
- Работает в группах 3-6 человек  
- Устраивает засады  
- Может использовать ловушки

## Лут

- Кибервар (разного качества)  
- Медицинские принадлежности  
- Оружие  
- Награбленные вещи

## Особенности

- Крайне опасны для одиноких путников  
- Могут похитить для извлечения кибервара  
- Презираются всеми фракциями  
- Работают из заброшенных зданий
