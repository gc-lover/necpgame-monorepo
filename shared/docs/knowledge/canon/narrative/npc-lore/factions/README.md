## Иерархия уровней

### Корпорации
1. **CEO / Совет директоров** — высший уровень
2. **Топ-менеджмент** — вице-президенты, директора
3. **Директора отделов** — Counter-Intel, Security, R&D, Legal
4. **Менеджеры** — среднее звено
5. **Рядовые сотрудники** — аналитики, оперативники
6. **Охранники / Солдаты** — низший уровень

### Банды
1. **Лидеры** — Jefe, Boss, Kingpin
2. **Заместители** — Teniente, Right Hand
3. **Элитные бойцы** — Veterans, Lieutenants
4. **Рядовые члены** — Soldado, Member
5. **Посыльные / Новички** — Aspirante, Recruit

## Эпохи

NPC распределены по эпохам:
- **2020-2030** — Восстановление после войны
- **2030-2045** — Купольные войны
- **2045-2060** — Красное время (Time of the Red)
- **2060-2077** — Преддверие современности
- **2077** — События игры
- **2078-2093** — Поздняя лига
