## История (краткая)

Рози родилась в семье механиков кочевников. С 8 лет умела чинить двигатели. Присоединилась к Wraiths в 2025, стала легендой — может собрать машину из обломков, оживить мёртвый двигатель, создать оружие из хлама. Её мастерская — сердце клана: без неё машины стоят, без машин — клан умирает.

**Ключевое событие (2070):** Создала "Железного Дракона" — бронированный грузовик-таран с встроенными пушками из обломков танка Militech. Используется для прорыва укреплений.

---

## Личность

**Характер:** Грубая снаружи, добрая внутри. Материнская фигура для молодых Wraiths. Не любит драматизм, ценит практичность. "Машина не врёт. Либо работает, либо нет."

**Цель:** Передать знания следующему поколению механиков, чтобы Wraiths всегда имели рабочую технику.

---

## Квесты

**"Воскрешение железа" (средний):**  
Рози просит игрока найти редкие детали в заброшенной военной базе для ремонта флагманского грузовика клана. База кишит дронами-охранниками.

**"Ученик мастера" (низкий):**  
Рози обучает игрока основам механики пустошей — как чинить машины из подручных средств. После обучения игрок получает уникальный перк.
