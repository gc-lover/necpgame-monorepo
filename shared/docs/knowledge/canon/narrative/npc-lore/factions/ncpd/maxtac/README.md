# MAX-TAC (Maximum Force Tactical Division) — Элитное подразделение NCPD

MAX-TAC — специальное подразделение NCPD для нейтрализации киберпсихопатов. Опер аторы — либо бывшие киберпсихопаты (получившие второй шанс), либо высоко модифицированные бойцы. Самые опасные и эффективные копы Night City.

---

## Операторы

- **Commander "Atlas" (Maya Cross)** — `commander-atlas-2070-2090.md` **(ДЕТАЛИЗИРОВАННЫЙ)**  
  Командир отряда, бывший киберпсихопат с трагичной историей, 75% кибернетизация, квест "Охота на себя"

- **Operator "Razor"** — `operator-razor-2075-2090.md`  
  Снайпер, 120+ нейтрализаций, 98% точность, холодный профессионал
