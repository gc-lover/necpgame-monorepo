# Фиксеры Night City — Индекс

Фиксеры — посредники между клиентами и исполнителями, ключевые фигуры криминального андерграунда Night City.

---

## Заметки

Фиксеры Night City образуют неформальную сеть взаимного уважения и конкуренции. Каждый контролирует свою территорию, но иногда кооперируются для крупных сделок.
