## Общее описание

Список важных NPC для NECPGAME, включающий ключевых персонажей из лора Cyberpunk и авторских персонажей. Каждый NPC имеет свою историю, мотивацию и роль в игровом мире.

**Принцип:** Гибридная система - использование NPC из лора Cyberpunk + авторские NPC + возможность игроков стать NPC (фиксеры, торговцы, политики, лидеры фракций).

## 1. NPC из лора Cyberpunk

### 1.1. Корпоративные персонажи

**Хироши Танака (Arasaka)**
- **Роль:** Корпоративный агент Arasaka

- **Локация:** Downtown (штаб-квартира Arasaka)
- **Функция:** Дает задания от Arasaka, развивает отношения с корпорацией
- **История:** Из лора Cyberpunk, адаптирован для MMORPG

- **Локация:** Downtown (штаб-квартира Militech)
- **Функция:** Дает задания от Militech, развивает отношения с корпорацией
- **История:** Из лора Cyberpunk, адаптирован для MMORPG
