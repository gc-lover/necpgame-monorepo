# Netrunners — Хакеры и исследователи NET

Netrunners — хакеры, погружающиеся в NET. От легенд старой эпохи до современных дайверов Blackwall.

## Легендарные Netrunners

- **Spider Murphy** — `spider-murphy-legend-2020-2077.md`  
  Легенда нетраннинга, участница взрыва Arasaka Tower 2023, статус неизвестен (возможно жива)

- **T-Bug** — `t-bug-prodigy-2050-2077.md`  
  Профессиональный netrunner, погибла в операции Konpeki Plaza 2077 (канон CP2077)

- **"Ghost Walker"** — `ghost-walker-blackwall-diver-2065-2090.md` **(АВТОРСКИЙ)**  
  Нелегальный исследователь Blackwall, торговец запрещённой информацией, частично "заражён" ИИ
