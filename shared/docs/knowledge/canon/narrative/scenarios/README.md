## Общее описание

Детальные сценарии для различных игровых ситуаций. Сценарии описывают последовательность событий, диалогов и действий в игре.

---

## Типы сценариев

### Сценарии начала игры
- Уникальные старты для разных комбинаций (происхождение, фракция, класс)
- Первые встречи с NPC
- Первые квесты
- Введение в игру

### Сценарии событий
- Мировые события
- События фракций
- События локаций
- Динамические события

### Сценарии взаимодействий
- Взаимодействия с NPC
- Взаимодействия с фракциями
- Взаимодействия с игроками
- Социальные взаимодействия

### Сценарии ветвления сюжета
- Ветвление квестов
- Ветвление диалогов
- Ветвление событий
- Ветвление сюжета
