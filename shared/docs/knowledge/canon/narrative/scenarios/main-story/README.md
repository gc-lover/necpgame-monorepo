# Основной сюжет — Матрица (классы × фракции × origins × эпохи)

**Цель:** задать единую структуру основного сюжета NECPGAME на временной шкале 2020–2093 с учётом класса, фракции/корпорации и начальной истории персонажа (origin).

## Матрица измерений
- Классы: Solo, Netrunner, Techie, Fixer, Rockerboy, Media, Nomad, Corpo, Lawman, Medtech, Politician, Trader, Teacher, «Свой путь»
- Фракции/Корпорации: Arasaka, Militech, Voodoo Boys, NetWatch, NCPD, Trauma Team, Nomad Clans (Aldecaldos и др.), банды (Valentinos, Maelstrom, 6th Street, Tyger Claws), городские DAO/куполы
- Origins: Street Kid, Corpo, Nomad (и авторские: Outlaw Scholar, Clinic Rat, Data Orphan)
- Эпохи: 2020–2030, 2030–2045, 2045–2060 (Time of the Red+), 2060–2077 (Pre-modern), 2077 (Game), 2078–2093 (Late League)
