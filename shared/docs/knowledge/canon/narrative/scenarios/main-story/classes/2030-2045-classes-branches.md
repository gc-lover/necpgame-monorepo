# 2030–2045 — Ветки по классам

## Конфигурация эпохи
- Купольные режимы (налоги/охрана/PvP), нейроэтика 1.0, контрабанда пропускной способности, дата‑убежища.

### Solo
- Аренные/охранные контракты куполов, приватные рейды
- Shooter thresholds: Combat 0.66–0.74; крит.успех → статус «Преторианец Купола»

### Netrunner
- Чистка заражённых микрокодов, защита каналов/логирования
- Shooter thresholds: Hacking 0.70–0.78; крит.провал → «баг недели» (серверный дебафф)

### Techie
- Тех-арбитраж нейроэтики, восстановление «Ковчегов»
- Shooter thresholds: Tech/Repair 0.66–0.74; бонус при доступе к мастерской (−0.04 к порогам)

### Fixer
- Контракты нейроэтики/прошивок, серые прайсы
- Shooter thresholds: Trading/Persuasion 0.66–0.74; крит.успех → эксклюзивные допуски

### Rockerboy / Media
- Анти-редактура «Тихой Сводки», публичные слушания
- Shooter thresholds: Charisma/Investigation 0.66–0.74; крит.успех → политическое окно

### Nomad
- Контрабанда пропускной способности, экстракция «пластов памяти»
- Shooter thresholds: Driving/Stealth 0.66–0.74; крит.провал → блокпост/изъятие груза

### Corpo
- Ликвидаторские тендеры: юридизация PvP-диверсий
- Shooter thresholds: Deception/Legal 0.66–0.74; conseq: межкорп-репутации

### Lawman
- Регламенты куполов/DAO, рейды на пиратские комбайнеры
- Shooter thresholds: Intimidation/Investigation 0.66–0.74; крит.успех → расширенные полномочия

### Medtech
- Клинические пути нейроэтики, серые импланты
- Shooter thresholds: Medicine 0.66–0.74; крит.провал → смертность (соц-удар)

### Politician
- «Устав Куполов», тарифы/режимы, этические шкалы
- Shooter thresholds: Social/Legal 0.66–0.74; conseq: глобальный сдвиг параметров района

### Trader
- Сетевые квоты/лицензии, серые рынки имплантов
- Shooter thresholds: Trading 0.66–0.74; крит.успех → оптовые каналы

### Teacher
- Обучение нейроэтике/гигиене каналов
- Shooter thresholds: Communication 0.62–0.70; бонус в куполах с грантами (−0.04 к порогу)
