# Ветки основного сюжета по классам (эпохи 2020–2093)

Матрица ветвления для 13 классов на каждой эпохе. Для каждой эпохи описаны:
- Ключевые конфликты и роли класса
- Выбор фракции/корпорации и последствия (кросс-эпохи)
- Shooter skill tests (типовые thresholds, модификаторы, крит-эффекты)

- 2060–2077 — `2060-2077-classes-branches.md`
- 2077 — `2077-classes-branches.md`
- 2078–2093 — `2078-2093-classes-branches.md`
