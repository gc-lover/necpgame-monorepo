# Основной сюжет — Каркас

## Актовая структура (каждая эпоха)
- Акт I: Завязка эпохи — конфликт эпохи, ввод фракций и выборы
- Акт II: Эскалация — персональные ветки классов и корп-напряжения
- Акт III: Развилка — ключевые проверки (соц/тех/бой) и фракционные решения
- Акт IV: Развязка — последствия на экономику/сети/политику; подготовка к следующей эпохе

## Правила ветвления
- Ветки по Классу: дают уникальные пути решения узлов (навыки/импланты)
- Ветки по Фракции/Корпорации: доступ/награды/репутация; конфликты интересов
- Ветки по Origin: стартовые бонусы/связи, ранние альтернативные маршруты
- Shooter skill tests: узлы проверок с порогами по эпохам (см. world-events), модификаторы от имплантов/окружения

## Переходы между эпохами
- Переносимые состояния: репутации фракций, артефакты, юридические статусы, «человечность»
- Непереносимые: ситуативные бафы/долги; часть перков (сброс/модификаторы)
- Мосты: специальные задания «порог эпохи», фиксирующие концовки эпохи и открывающие бонусы/штрафы следующей

## Фракционные правила
- Конфликты Arasaka vs Militech: дуальные ветки соглашений/саботажа/рейдов
- NetWatch и Blackwall: ветки доступа/стандартов vs. чёрные сети
- Городские DAO/Купола: режимы районов (голосования), параметры экономики/безопасности

## Origins (стартовые истории)
- Street Kid: уличные связи, бонус к Stealth/Intimidation, доступ к бандитским нитям
- Corpo: доступ к корпоративным узлам, бонус к Persuasion/Trading, риски политических интриг
- Nomad: мобильность/логистика, бонус к Survival/Driving, доступ к контрабанде/экстракции
- Outlaw Scholar (author): доступ к редким знаниям, Analysis/Investigation бонусы
- Clinic Rat (author): мед/имплантные связи, Medicine/Repair бонусы
- Data Orphan (author): аномальные сетевые следы, Hacking/Deception бонусы
