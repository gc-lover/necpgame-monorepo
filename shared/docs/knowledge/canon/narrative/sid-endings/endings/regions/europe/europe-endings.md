# Европа: Концовки региона (2090-2093)

**Регион:** Европа  
**Ключевые города:** London, Paris, Berlin, Warsaw, Istanbul, Amsterdam, Rome, Barcelona, Stockholm, Prague, Vienna  
**Население:** ~500 млн  
**Экономика:** Высокоразвитая, разнообразная

---

## КОНЦОВКА E1: "Европейская утопия" (European Utopia)

**Условия активации:**
- `regions.europe.prosperity > 85`
- `regions.europe.stability > 80`
- `regions.europe.tech_level > 75`
- `regions.europe.freedom > 70`
- `corporate_power (Europe) < 60` (не слишком корпоративная)
- `world_inequality (Europe) < 30`

**Описание:**
Европа достигла идеала. Социальное государство + технологии + свобода. Высокий уровень жизни. Низкое неравенство. Баланс корпораций, правительств и граждан. Культурное процветание.

**Судьбы городов:**
- **London:** Финансовый и культурный центр мира
- **Paris:** Город искусств и инноваций
- **Berlin:** Технологическая столица
- **Stockholm:** Экологичный город будущего
- **Amsterdam:** Центр личных свобод и толерантности

**Экономика:**
- Процветание без эксплуатации
- Всеобщий базовый доход
- Технологии для всех
- Зеленая энергетика

**Социум:**
- Высокое доверие
- Низкая преступность
- Культурное разнообразие
- Образование и здравоохранение для всех

**Титры (Европа):**
```
ЕВРОПА 2093: Утопия реализована

Высокий уровень жизни. Низкое неравенство.
Технологии служат людям. Культура процветает.
Баланс достигнут. Мечта стала реальностью.

London — финансовый центр мира.
Paris — город искусств.
Berlin — технологическая столица.

Европа показала: утопия возможна.
```

**Влияние на следующую лигу:**
- Европа стартует как образец для подражания
- Другие регионы пытаются копировать модель
- Высокий уровень иммиграции в Европу

## КОНЦОВКА E2: "Европейская империя" (European Empire)

**Условия активации:**
- `regions.europe.power > 80`
- `corporate_power (Europe) > 75`
- `military_strength (Europe) > 70`
- Европа доминирует в глобальной политике

**Описание:**
Европа объединилась в империю. Милитаризация. Экспансия. Европейские корпорации доминируют глобально. Неоколониализм. Сила через единство.

**Судьбы городов:**
- **London:** Столица империи, центр власти
- **Paris:** Военная столица
- **Berlin:** Промышленный центр империи
- **Warsaw:** Восточный форпост

**Политика:**
- Единое правительство
- Милитаризация
- Экспансия в Африку и Ближний Восток
- Конфликт с Азией и Америкой

**Титры (Европа):**
```
ЕВРОПА 2093: Империя восстановлена

Европа едина. Европа сильна. Европа правит.

London — столица новой империи.
Paris командует армиями.
Berlin производит оружие.

Африка и Ближний Восток — под контролем.
Конфликт с Азией неизбежен.

Империя вернулась. К добру или злу?
```

## КОНЦОВКА E3: "Распад Европы" (European Collapse)

**Условия активации:**
- `regions.europe.stability < 30`
- `regions.europe.prosperity < 40`
- `world_violence (Europe) > 70`
- Экономический или политический кризис

**Описание:**
ЕС распался. Экономический кризис. Миграционный коллапс. Локальные войны. Каждая страна за себя. Возврат к национализму. Европа разделена.

**Судьбы городов:**
- **London:** Изолирован, экономический спад
- **Paris:** Гражданские беспорядки
- **Berlin:** Экономический кризис
- **Warsaw:** Военный конфликт с соседями

**Проблемы:**
- Миграционный кризис
- Экономический коллапс
- Рост национализма
- Локальные войны

**Титры (Европа):**
```
ЕВРОПА 2093: Распад союза

ЕС мертв. Каждая страна за себя.
Экономический кризис. Миграционный хаос.
Войны на границах.

London изолирован.
Paris горит.
Berlin в кризисе.

Европейская мечта умерла.
```

## КОНЦОВКА E4: "Киберпанк Европа" (Cyberpunk Europe)

**Условия активации:**
- `tech_level (Europe) > 85`
- `corporate_power (Europe) > 70`
- `world_inequality (Europe) > 70`
- `implant_adoption (Europe) > 75`

**Описание:**
Европа превратилась в киберпанк-дистопию. Корпорации правят. Неравенство максимально. Технологии для богатых. Бедные в трущобах. Классический киберпанк.

**Судьбы городов:**
- **London:** Корпоративные башни vs трущобы
- **Paris:** Подпольный мир vs элита
- **Berlin:** Хакеры vs корпоративная безопасность
- **Amsterdam:** Нелегальные импланты и черный рынок

**Социум:**
- Огромное неравенство
- Корпорации владеют всем
- Киберпсихоз распространен
- Подпольное сопротивление

**Титры (Европа):**
```
ЕВРОПА 2093: Киберпанк реализован

Корпорации правят. Неравенство максимально.
Богатые в башнях. Бедные в трущобах.

London — город контрастов.
Paris — подполье против элиты.
Berlin — хакеры воюют с корпорациями.

Классический киберпанк. Европейский стиль.
```

## КОНЦОВКА E5: "Зеленая Европа" (Green Europe)

**Условия активации:**
- `ecology_level (Europe) > 80`
- `tech_adoption (renewable energy) > 75`
- `implant_adoption (Europe) < 40` (возврат к природе)
- `corporate_power (Europe) < 50`

**Описание:**
Европа выбрала зеленый путь. Отказ от агрессивных технологий. Возобновляемая энергия. Экология приоритетна. Баланс с природой. Гармония.

**Судьбы городов:**
- **Stockholm:** Образец экологичного города
- **Amsterdam:** Город-сад
- **Copenhagen:** Зеленая столица
- **Vienna:** Культура + экология

**Экономика:**
- Зеленые технологии
- Возобновляемая энергия
- Экологичное производство
- Устойчивое развитие

**Титры (Европа):**
```
ЕВРОПА 2093: Зеленая революция

Европа выбрала природу. Экология — приоритет.
Возобновляемая энергия. Чистые города.

Stockholm — эко-столица мира.
Amsterdam — город-сад.
Copenhagen — образец для всех.

Европа зеленая. Европа дышит.
```

## КОНЦОВКА E6: "Балканизация" (Balkanization)

**Условия активации:**
- `political_fragmentation (Europe) > 80`
- `regions.europe.stability < 40`
- ЕС распался на множество мелких государств

**Описание:**
Европа раздроблена на десятки мелких государств. Каждый город — отдельное государство. Постоянные конфликты. Торговые войны. Хаос.

**Судьбы городов:**
- **London:** Город-государство, торговая республика
- **Paris:** Город-государство, культурный центр
- **Berlin:** Город-государство, промышленный
- Каждый крупный город — независим

**Политика:**
- Множество мелких государств
- Постоянные конфликты
- Торговые войны
- Нестабильность

**Титры (Европа):**
```
ЕВРОПА 2093: Балканизация завершена

Европа раздроблена. Каждый город — государство.
Постоянные конфликты. Торговые войны.

London — торговая республика.
Paris — культурный центр.
Berlin — промышленный город.

Единство потеряно. Хаос установлен.
```
