## Общее описание

Детальные описания квестов для игры. Каждый квест содержит полное описание, диалоги, цели, награды и ветвление.

---

## Типы квестов

### Основные квесты (Main Quests)
- Главная сюжетная линия
- Ветвление сюжета
- Ключевые события
- Развитие персонажа

### Побочные квесты (Side Quests)
- Дополнительные задания
- Исследование мира
- Развитие персонажа
- Взаимодействие с NPC

### Квесты фракций (Faction Quests)
- Квесты для фракций
- Развитие репутации
- Уникальные награды
- Ветвление отношений

### Квесты событий (Event Quests)
- Квесты мировых событий
- Квесты событий фракций
- Квесты событий локаций
- Временные квесты

### Квесты игроков (Player Quests)
- Квесты, созданные игроками
- Заказы от игроков
- Взаимодействие между игроками
- Экономические квесты
