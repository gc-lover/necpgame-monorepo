# Daily/Weekly квесты по регионам

Рандомные повторяемые квесты для каждого региона мира.

## Структура

- **Daily Quests** — Ежедневные квесты (короткие, 10-15 минут)
- **Weekly Quests** — Еженедельные квесты (средние, 30-60 минут)
- **Regional Rotation** — Квесты меняются в зависимости от региона

## Механики

### Daily Quests
- **Длительность:** 10-15 минут
- **Награды:** 200-800 ed, +2-5 reputation
- **DC Range:** 12-20
- **Cooldown:** 24 часа

### Weekly Quests
- **Длительность:** 30-60 минут
- **Награды:** 1000-3000 ed, +10-20 reputation
- **DC Range:** 18-28
- **Cooldown:** 7 дней

### Рандомизация
- Квесты выбираются из пула случайным образом
- Зависят от региона игрока
- Учитывают репутацию и класс
