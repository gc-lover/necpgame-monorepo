## Рандомизация

### Триггеры появления
- **Daily:** Каждый день в 00:00 UTC, 3 случайных квеста из 12
- **Weekly:** Каждый понедельник в 00:00 UTC, 2 случайных квеста из 8

### Модификаторы
- **Класс игрока:** +5 DC если квест не для вашего класса
- **Репутация:** -1 DC per 10 reputation с соответствующей фракцией
- **Эпоха:** +2 DC per era (2030=+0, 2045=+2, 2060=+4, 2078=+6)

### Награды (масштабирование)
- **2030-2045:** Base rewards
- **2045-2060:** Base × 1.5
- **2060-2077:** Base × 2
- **2078-2093:** Base × 3
