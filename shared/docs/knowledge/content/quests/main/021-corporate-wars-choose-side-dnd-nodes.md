# Квест 2.1 «Выбор стороны» — D&D узлы проверок

**Статус:** draft

- База: `02-gameplay/combat/combat-dnd-core.md`

## Узлы

### N-200 Анализ предложения корпорации
- Тип: Исследование / Analysis
- Формула: d20 + INT мод + Analysis бонус ≥ DC 18 (уровень 15–25)
- Модификаторы: корпоративный доступ +2; дефицит времени -2
- Успех: раскрывает скрытые условия контракта → бонус к переговорам
- Провал: стандартные условия

### N-201 Переговоры с Arasaka/Militech
- Тип: Социальная / Persuasion/Charisma
- Формула: d20 + EMP мод + Persuasion/Charisma + репутация ≥ DC 20–24
- Преимущество: подходящий внешний вид/связи; Помеха: конфликт интересов
- Успех: расширенный пакет наград, репутация +20
- Провал: базовый пакет, репутация +10
- Крит.провал: «условие-ловушка» (влияет на 2.2)

### N-202 Проверка благонадежности
- Тип: Социальная / Deception/Intimidation (в зависимости от выбранной тактики)
- Формула: d20 + COOL мод + Deception/Intimidation ≥ DC 19–22
- Успех: доступ к закрытому брифингу (снижает DC на следующем квесте)
- Провал: наблюдение за игроком (повышает тревожность сцен)

## Ветвления
- Выбор Arasaka/Militech/Нейтральная фиксирует будущие узлы и DC
- Скрытые условия контракта могут менять награды 2.2

## Групповые правила
- Переговоры: спикер + ассистенты (+1 за успешную «поддержку», макс +3)
