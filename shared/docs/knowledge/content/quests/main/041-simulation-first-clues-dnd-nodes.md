## Узлы

### N-400 Аномалия поведения NPC
- Perception: d20 + REF + Perception ≥ DC 20 → замечает «сбой»
- Analysis: d20 + INT + Analysis ≥ DC 20 → формирует гипотезу

### N-401 Трассировка события
- Hacking: серия DC 20→22 (INT/TECH + Hacking) → след к «черному заслону»
- Альтернатива: Social/Media — добыть информацию у инсайдера (DC 20–22)

### N-402 Мета-подсказка
- Lore/Universe check (INT/Analysis) DC 21 — сопоставление таймлайна и аномалий
- Успех: открывает 4.2 путь с -2 DC на первом шаге

## Ветвления
- Разные классовые подходы фиксируют разные подсказки, но ведут к 4.2
