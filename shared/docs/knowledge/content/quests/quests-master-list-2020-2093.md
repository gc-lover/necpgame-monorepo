## Формат квеста (детальный)

Каждый квест включает:
- **ID:** Уникальный идентификатор
- **Название:** Краткое и запоминающееся
- **Эпоха:** Годы действия
- **Тип:** Main / Side / World Event
- **Класс:** Фокус на класс(ы) или универсальный
- **Фракция:** Связанная фракция (опционально)
- **Origin:** Связанный origin (опционально)
- **Уровень:** Рекомендуемый уровень персонажа
- **Shooter skill tests:** Список проверок (thresholds, метрики, последствия)
- **Диалоговое дерево:** 20–30 узлов с ветвлениями
- **Репутация:** Изменения репутации фракций
- **Лут:** Таблица наград
- **Последствия:** Влияние на будущие квесты/эпохи
