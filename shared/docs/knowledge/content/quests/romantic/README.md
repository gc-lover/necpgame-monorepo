# Романтические квесты (Romance Quests)

Романтические линии с NPC из разных стран мира. 18 романсов с полными сюжетными арками.

## Структура романса

- **3-5 квестов** на каждую романтическую линию
- **Социальные проверки** (Persuasion, Empathy, Charisma)
- **Прогрессия отношений** (Знакомство → Друзья → Романтический интерес → Партнёр)
- **Моральные выборы** (влияние на отношения)
- **Уникальные награды** (Companion perks, Safe houses, Special items)

## Механика романсов

### Стадии отношений
1. **Stranger (0-20)** — Знакомство
2. **Friend (20-40)** — Дружба
3. **Close Friend (40-60)** — Близкий друг
4. **Romantic Interest (60-80)** — Романтический интерес
5. **Partner (80-100)** — Партнёр

### Влияние выборов
- **Моральные выборы** — влияют на отношения (±5-15)
- **Подарки** — +2-10 (зависит от качества)
- **Квесты** — +10-20 per quest
- **Предательство** — -30-50 (может разрушить отношения)

### Награды за романсы
- **Companion Perks** — Уникальные бонусы от партнёра
- **Safe Houses** — Доступ к жилью партнёра
- **Business Contacts** — Связи партнёра
- **Unique Items** — Подарки от партнёра

### Сексуальная ориентация
- **Bisexual (12)** — Романс с любым полом
- **Heterosexual (4)** — Только противоположный пол
- **Homosexual (2)** — Только одинаковый пол

## Конфликты и драма
- Романс может конфликтовать с фракционной лояльностью
- Романсы могут ревновать к другим NPC
- Некоторые NPC имеют тёмное прошлое
- Возможно разрушение отношений
