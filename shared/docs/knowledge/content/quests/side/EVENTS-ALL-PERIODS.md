### 1. [EVENT-2020-PROXY-CONFLICT] Прокси-конфликт
**Период:** 2020-2022  
**Локации:** Downtown, Westbrook  
**Триггер:** Уровень 1-3, любое время, шанс 12%

**Описание:** Случайное столкновение корпоративных агентов. Двое людей в строгих костюмах спорят, один достаёт оружие.

**Исходы:**
- **Вмешаться (Combat R1+):** Остановить конфликт силой → Лут: Corpo ID, 80-150 эдди, репутация +2 (случайная корпорация)
- **Собрать данные (Hacking R1):** Записать разговор → Предмет: Encrypted Data Shard, unlock квест "Corporate Espionage"
- **Игнорировать:** Конфликт разрешается сам, возможна случайная стрельба
