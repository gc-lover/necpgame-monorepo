## Ключевые системы

### Skill-Check система

```json
{
  "skillCheck": {
    "type": "attribute|skill",
    "attribute": "TECH|INT|COOL|REF|AGI|EMP|STEALTH",
    "skill": "Hacking|Combat|Social",
    "threshold": 8-16,
    "rank": "R1-R5",
    "successEffect": "описание эффекта",
    "failEffect": "описание провала"
  }
}
```

### Диалоговое дерево

```
[NODE-ID] Speaker: "Текст"
├─ [A] "Выбор 1" → NODE-NEXT
├─ [B] [SKILL-CHECK] "Выбор 2" → NODE-SUCCESS/FAIL
└─ [C] "Выбор 3" → QUEST_END
```

### Репутационная формула

```
ΔReputation = BaseAmount × (1 + ClassModifier + SkillModifier) × OutcomeMultiplier
```

**Модификаторы:**
- **ClassModifier:** +10-20% (зависит от класса и фракции)
- **SkillModifier:** +5-15% (зависит от навыков)
- **OutcomeMultiplier:** ×0.8-1.5 (зависит от исхода)

### Лут-таблица

```json
{
  "itemId": "item-name",
  "chance": 0.0-1.0,
  "quantity": {"min": X, "max": Y},
  "qualityModifier": 1.0
}
```

## Интеграция с игровыми системами

### Связь с атрибутами
- **TECH:** Установка оборудования, ремонт, крафт
- **INT:** Анализ данных, распознавание паттернов
- **COOL:** Блеф, переговоры, стрессоустойчивость
- **REF:** Уклонение, быстрые действия
- **AGI:** Паркур, акробатика
- **EMP:** Торговля, убеждение, чтение людей
- **STEALTH:** Скрытность, избегание обнаружения

### Связь с навыками
- **Hacking R1-R5:** Взлом, отключение систем, подделка ID
- **Combat R1-R5:** Бой, тактика, использование оружия
- **Social R1-R5:** Переговоры, блеф, убеждение
- **Tech R1-R5:** Ремонт, крафт, модификация
- **Stealth R1-R5:** Скрытность, избегание патрулей
- **Mobility R1-R5:** Паркур, быстрое перемещение

### Связь с имплантами
- **Кибердеки:** Hacking навыки, ioBandwidth
- **Боевые импланты:** Combat навыки, Reflexes
- **Социальные импланты:** Social навыки, Empathy
- **Мобильность импланты:** Mobility навыки, Agility

## Репутационные фракции

### Основные фракции
1. **Fixers** (Фиксеры) — контракты, доступ к сети
2. **NCPD** (Полиция) — легальность, поддержка города
3. **NetWatch** (Сетевая безопасность) — доступ к Blackwall
4. **Corporations** (Корпорации) — ресурсы, технологии
5. **Gangs** (Банды) — территории, чёрный рынок
6. **Ripperdocs** (Рипдоки) — импланты, обслуживание
7. **Media** (Медиа) — информация, влияние
8. **City Services** (Городские службы) — инфраструктура

### Эффекты репутации
- **0-49 (Stranger):** Нет бонусов
- **50-99 (Known):** Скидка 5%, базовые контракты
- **100-199 (Trusted):** Скидка 10%, редкие контракты
- **200-399 (Respected):** Скидка 15%, эксклюзивные контракты
- **400+ (Legendary):** Скидка 20%, уникальные предметы

## Альтернативные пути прохождения

### 1. Стелс-путь
- **Требования:** STEALTH ≥ 12, REF ≥ 10
- **Преимущества:** Без боёв, максимальная награда, бонус репутации
- **Недостатки:** Сложнее, требует планирования

### 2. Хакер-путь
- **Требования:** Hacking R2-R4, INT ≥ 12
- **Преимущества:** Бонус данные, unlock квест-цепочек, уникальные предметы
- **Недостатки:** Требует кибердеку, риск киберпсихоза

### 3. Боевой путь
- **Требования:** Combat R2+, REF ≥ 10
- **Преимущества:** Больше лута, проще прохождение
- **Недостатки:** Риск провала, меньше репутации

### 4. Социальный путь
- **Требования:** COOL ≥ 14, EMP ≥ 10, Social R2+
- **Преимущества:** Блеф, торг, дополнительные опции
- **Недостатки:** Не всегда доступен, зависит от репутации
