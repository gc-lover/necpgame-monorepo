```
- Связи: `progression-skills.md`, `equipment-matrix.md`, `combat-implants-types.md`
- Требования: атрибуты/импланты/уровень/репутация

Примечания дизайна:
- Баланс, анти‑абуз, повторяемость/одноразовость
```
