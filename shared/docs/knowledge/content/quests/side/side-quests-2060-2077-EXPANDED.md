## Квесты

### [SQ-2062-PROXY] Серые театры

**Метаданные:**
- **Период:** 2061–2070
- **Тип:** side
- **Тэги:** [combat|stealth|corporate]
- **Уровень:** 15-20
- **Длительность:** 120-150 мин
- **Повторяемость:** Да (разные купола)

**Синопсис:**  
В эпоху прокси-войн корпорации используют "серые театры" — нелегальные арены, где киберпсихи сражаются на потеху публике. Контракт в чужом куполе требует скрытности и смертоносности.

**Цели:**
1. Встретиться с Корпоративным Агентом
2. Получить доступ к чужому куполу
3. Проникнуть на арену (требуется Stealth R4+)
4. Выполнить контракт (убить цель или защитить)
5. Эвакуироваться
6. Вернуться за наградой

**Диалоговое дерево (26 узлов):**

```
[NODE-01] Corpo Agent: "Привет, оперативник. Есть контракт в чужом куполе — 'серый театр'."
├─ [A] "Что за театр?" → NODE-02
├─ [B] "Сколько платишь?" → NODE-03
├─ [C] [INT ≥ 16] "Почему именно я?" → NODE-04
└─ [D] "Не интересует" → QUEST_DECLINE

[NODE-02] Corpo Agent: "Нелегальная арена, где киберпсихи сражаются. Нужно проникнуть и выполнить контракт."
├─ [A] "Какой контракт?" → NODE-05
└─ [B] "Понял, беру" → NODE-06

[NODE-03] Corpo Agent: "8000 эдди базовая ставка, плюс бонус за скорость. Итого до 12000."
├─ [A] [EMP ≥ 16] "Опасно. Сделай 10000 базовая" → NODE-07 (Успех: +2000 эдди)
├─ [B] "Годится" → NODE-06
└─ [C] "Подумаю" → NODE-08

[NODE-04] Corpo Agent: "Нужен профессионал. Ты подходишь. Опыт, навыки, репутация."
├─ [A] "Понял" → NODE-02
└─ [B] "Кто заказчик?" → NODE-09

[NODE-05] Corpo Agent: "Два варианта: убить цель или защитить клиента. Выбирай."
├─ [A] "Убить цель" → QUEST_START (Mode: ASSASSINATE)
├─ [B] "Защитить клиента" → QUEST_START (Mode: PROTECT)
└─ [C] "Подумаю" → NODE-08

[NODE-06] Corpo Agent: "Отлично. Вот доступ к чужому куполу. Будь осторожен — обнаружение = смерть."
└─ [A] "Выдвигаюсь" → QUEST_START

[NODE-07] Corpo Agent: "Ладно, 10000. Но только если выполнишь контракт идеально."
└─ [A] "Договорились" → QUEST_START

[NODE-08] Corpo Agent: "Решай быстро. Контракт сгорит."
├─ [A] "Хорошо, беру" → NODE-05
└─ [B] "Пас" → QUEST_DECLINE

[NODE-09] Corpo Agent: "Анонимный клиент. Платит хорошо. Не задавай вопросов."
└─ [A] "Понял" → NODE-05

--- ПРОНИКНОВЕНИЕ ---

[NODE-10] (Чужой купол — Периметр)
Система: "Обнаружен периметр купола. Охрана: высокая. Проникнуть?"
├─ [A] [STEALTH ≥ 16] "Скрытное проникновение" → NODE-11 (Успех)
├─ [B] [Hacking R4] "Отключить системы охраны" → NODE-12 (Успех)
└─ [C] "Прямой штурм" → NODE-13 (Бой: Corpo Guards ×4)

[NODE-11] Система: "Проникновение успешно. Охрана не заметила."
└─ [A] "К арене" → NODE-14

[NODE-12] Система: "Системы отключены. Окно: 60 секунд."
└─ [A] "К арене" → NODE-14

[NODE-13] (Бой: Corpo Guards ×4)

└─ [A] После победы → NODE-15 (Риск: Подкрепление)

[NODE-14] (Арена — Внутри)
Система: "Обнаружена арена. Киберпсихи сражаются. Цель: [TARGET]."
├─ [A] [Mode: ASSASSINATE] "Убить цель" → NODE-16
├─ [B] [Mode: PROTECT] "Защитить клиента" → NODE-17

[NODE-16] (Режим: Убить цель)

├─ [A] [Combat R4] "Прямая защита" → NODE-22 (Бой: враги ×6)
├─ [B] [STEALTH ≥ 16] "Эвакуировать клиента" → NODE-23 (Успех: скрытно)
└─ [C] [Hacking R4] "Отключить атаку врагов" → NODE-24 (Успех: через системы)

[NODE-25] (Эвакуация)
Система: "Контракт выполнен. Эвакуироваться?"
├─ [A] [AGI ≥ 18] "Быстрая эвакуация" → NODE-26 (Успех: без преследования)
├─ [B] "Стандартная эвакуация" → NODE-27 (Риск: преследование)
└─ [C] "Прорыв силой" → NODE-28 (Бой: все враги)

[NODE-26] Система: "Эвакуация успешна. Преследования нет."
└─ [A] "Вернуться к Корпоративному Агенту" → QUEST_COMPLETE_SUCCESS

[NODE-27] Система: "Эвакуация началась. Преследование обнаружено."
├─ [A] [AGI ≥ 16] "Оторваться" → NODE-26 (Успех)
└─ [B] "Бой неизбежен" → NODE-28

[NODE-28] (Бой: все враги)
Система: "Прорыв силой! Все враги атакуют!"
└─ [A] После победы → QUEST_COMPLETE_SUCCESS (Но репутация -5)

--- ВОЗВРАЩЕНИЕ К КОРПОРАТИВНОМУ АГЕНТУ ---

[NODE-29] Corpo Agent: "Отличная работа! Контракт выполнен идеально. Держи свои эдди."
├─ [A] "Спасибо" → QUEST_REWARD
└─ [B] "Что дальше?" → NODE-30

[NODE-30] Corpo Agent: "Может быть ещё контракты. Заходи, если что — дам знать."
└─ [A] "Договорились" → QUEST_REWARD
```

**Skill-Checks:**
- **EMP ≥ 16:** Торг с Корпоративным Агентом (+2000 эдди)
- **INT ≥ 16:** Узнать причину выбора (unlock dialogue)
- **STEALTH ≥ 16:** Скрытное проникновение (успех)
- **Hacking R4:** Отключить системы охраны (успех)
- **STEALTH ≥ 18:** Скрытное убийство (без тревоги)
- **Combat R5:** Прямая атака (успех)
- **Hacking R5:** Дистанционное убийство (успех)
- **AGI ≥ 18:** Быстрая эвакуация (без преследования)

**Награды:**
- **Базовые:** 8000-12000 эдди, 1500 XP
- **Бонусные (идеальное выполнение):** +2000 эдди, +500 XP
- **Бонусные (скрытность):** Репутация Corporations +15
- **Репутация:** Corporations +20, Criminal Organizations -10 (за убийство)
