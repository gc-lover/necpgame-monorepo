### [SQ-2080-PFAIR] Параметрические ярмарки

**Метаданные:**
- **Период:** 2079–2085
- **Тип:** side
- **Тэги:** [social|economy|league]
- **Уровень:** 20-30
- **Длительность:** 90-120 мин
- **Повторяемость:** Да (разные параметры)

**Синопсис:**  
В эпоху лиг "параметрические ярмарки" позволяют тестировать наборы параметров мира. Нужно агитировать за свой набор, анализировать результаты и влиять на голосование лиг.

**Цели:**
1. Встретиться с Лидером Лиги
2. Выбрать набор параметров (Economy/Social/Combat)
3. Агитировать за набор (5 NPC)
4. Участвовать в тестировании
5. Влиять на голосование
6. Получить награду

**Диалоговое дерево (28 узлов):**

```
[NODE-01] League Leader: "Привет, участник лиги. Параметрические ярмарки — уникальный шанс изменить мир."
├─ [A] "Что за ярмарки?" → NODE-02
├─ [B] "Какие награды?" → NODE-03
├─ [C] [INT ≥ 18] "Как это влияет на мир?" → NODE-04
└─ [D] "Не интересует" → QUEST_DECLINE

[NODE-02] League Leader: "Ярмарки позволяют тестировать наборы параметров мира. Голосование определяет, какой набор будет применён."
├─ [A] "Какие наборы?" → NODE-05
└─ [B] "Понял, беру" → NODE-06

[NODE-03] League Leader: "Влияние на лигу, репутация, уникальные предметы. Итого: до 5000 эдди + влияние."
├─ [A] [EMP ≥ 18] "Мало для такого влияния. Сделай 7000" → NODE-07 (Успех: +2000 эдди)
├─ [B] "Годится" → NODE-06
└─ [C] "Подумаю" → NODE-08

[NODE-04] League Leader: "Параметры влияют на экономику, социальные системы, боевые механики. Это меняет весь мир."
├─ [A] "Понял" → NODE-02
└─ [B] "Кто решает?" → NODE-09

[NODE-05] League Leader: "Три набора: Economy (экономика), Social (социал), Combat (боевые механики). Выбирай."
├─ [A] "Economy" → QUEST_START (Mode: ECONOMY)
├─ [B] "Social" → QUEST_START (Mode: SOCIAL)
├─ [C] "Combat" → QUEST_START (Mode: COMBAT)
└─ [D] "Подумаю" → NODE-08

[NODE-06] League Leader: "Отлично. Вот список влиятельных NPC для агитации. Удачи!"
└─ [A] "Выдвигаюсь" → QUEST_START

[NODE-07] League Leader: "Ладно, 7000. Но только если твой набор выиграет."
└─ [A] "Договорились" → QUEST_START

[NODE-08] League Leader: "Решай быстро. Ярмарка начинается через 24 часа."
├─ [A] "Хорошо, беру" → NODE-05
└─ [B] "Пас" → QUEST_DECLINE

[NODE-09] League Leader: "Голосование лиг решает. Чем больше поддержки — тем больше шансов."
└─ [A] "Понял" → NODE-02

├─ [A] [Mode: ECONOMY] [EMP ≥ 16] "Экономика создаст рабочие места" → NODE-11 (Успех: +1 голос)
├─ [B] [Mode: SOCIAL] [SOCIAL R3] "Социал улучшит жизнь всех" → NODE-11 (Успех: +1 голос)
```

**Skill-Checks:**
- **EMP ≥ 18:** Торг с Лидером Лиги (+2000 эдди)
- **INT ≥ 18:** Узнать влияние на мир (unlock dialogue)
- **EMP ≥ 16:** Убедить NPC (успех)
- **SOCIAL R3:** Социальная агитация (успех)
- **INT ≥ 18:** Анализ результатов (+бонусы)
- **Hacking R5:** Взлом системы голосования (риск)
- **AGI ≥ 20:** Сбежать после взлома (успех)

**Награды:**
- **Базовые:** 5000-7000 эдди, 2000 XP, League Influence +10
- **Бонусные (выигрыш):** +2000 эдди, +500 XP, Unique Item
- **Бонусные (анализ):** +15% влияние на голосование
- **Репутация:** League +25, City Services +15
