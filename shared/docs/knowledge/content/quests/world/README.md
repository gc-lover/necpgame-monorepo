# Мировые квесты (World Quests)

Квесты с глобальным охватом, привязанные к конкретным регионам и странам мира.

## Структура

- **Общие квесты** — доступны для всех классов
- **Классовые квесты** — для конкретных классов (Solo, Netrunner, Corpo и т.д.)
- **Фракционные квесты** — для конкретных фракций (Arasaka, Militech, Gangs и т.д.)

## Типы квестов

- **WQ-[РЕГИОН]-[ГОД]-[НОМЕР]** — Общие мировые квесты
- **WQ-[РЕГИОН]-[ГОД]-[КЛАСС]-[НОМЕР]** — Классовые квесты
- **WQ-[РЕГИОН]-[ГОД]-[ФРАКЦИЯ]-[НОМЕР]** — Фракционные квесты
