## Общее описание

Детальное описание первого квеста для игрока в начале игры. Определяет диалоги, цели, награды и последовательность выполнения первого квеста.

**Цель:** Плавно ввести игрока в игровой мир через первый квест, который знакомит с основными механиками.

---

## 1. Первый квест: "Доставка груза"

### 1.1. Информация о квесте

**ID:** `quest-delivery-001`

**Название:** "Доставка груза"

**Тип:** Side quest (побочный квест)

**Уровень:** 1

**Данный NPC:** Сара Миллер (офицер NCPD)

**Локация получения:** Downtown, Night City

**Локация выполнения:** Watson, Night City

---

### 1.2. Описание квеста

**Краткое описание:**
Офицер NCPD Сара Миллер просит доставить посылку из Downtown в Watson торговцу Рите Морено. Это простое задание для новичков, но нужно быть осторожным при перемещении между районами.

**Полное описание:**
"Чомбата, у меня есть задание для тебя. Нужно доставить посылку из Downtown в Watson торговцу Рите Морено. Это простое задание, но будь осторожен - путь между районами может быть опасным. Если выполнишь задание, заплачу 200 eddies и дам немного опыта. Также это поможет тебе зарекомендовать себя перед NCPD."

---

### 1.3. Диалог с Сарой Миллер

**Приветствие (первое взаимодействие):**
"Привет. Я офицер Миллер. Если ты хочешь помочь полиции и заработать немного денег, у меня есть задания."

**Опции диалога:**

1. **"Расскажи о NCPD"**
   - **Ответ:** "NCPD - это полиция Night City. Мы поддерживаем порядок, но в этом городе это нелегко. Банды, корпорации, преступники - все хотят контролировать свои территории. Нам нужна помощь от таких как ты."

2. **"Какие задания у тебя есть?"**
   - **Ответ:** "У меня есть простое задание - доставить посылку в Watson торговцу Рите Морено. Это безопасно, но нужно быть осторожным. Заплачу 200 eddies."
   - **Действие:** Показывается информация о квесте "Доставка груза"

3. **"Расскажи о городе"**
   - **Ответ:** "Night City - это место возможностей и опасностей. В Downtown безопасно, но дорого. В Watson больше возможностей, но и больше риска. Будь осторожен, особенно в Heywood - там банды контролируют все."

4. **"До свидания"**
   - **Ответ:** "Удачи, чомбата. Если передумаешь - возвращайся."

---

### 1.4. Цели квеста (Objectives)

**1. Принять квест от Сары Миллер**
- **Тип:** NPC interaction
- **Цель:** Сара Миллер (npc-sarah-miller)
- **Описание:** "Принять квест от Сары Миллер"
- **Действие:** Игрок выбирает опцию "Принять квест" в диалоге

**2. Получить посылку**
- **Тип:** Item acquisition
- **Цель:** Посылка (item-package-delivery)
- **Описание:** "Получить посылку от Сары Миллер"
- **Действие:** После принятия квеста посылка автоматически добавляется в инвентарь игрока
- **Примечание:** Если посылка не добавлена автоматически, игрок должен получить её от Сары Миллер

**3. Переместиться в Watson**
- **Тип:** Location
- **Цель:** Watson (loc-watson-001)
- **Описание:** "Переместиться в Watson"
- **Действие:** Игрок выбирает "Переместиться" в меню действий и выбирает Watson

**4. Найти Риту Морено**
- **Тип:** NPC interaction
- **Цель:** Рита Морено (npc-rita-moreno)
- **Описание:** "Найти Риту Морено в Watson"
- **Действие:** Игрок выбирает "Найти NPC" в меню действий и выбирает Риту Морено

**5. Передать посылку Рите Морено**
- **Тип:** Delivery
- **Цель:** Рита Морено (npc-rita-moreno)
- **Описание:** "Передать посылку Рите Морено"
- **Действие:** Игрок начинает диалог с Ритой Морено и выбирает опцию "Передать посылку"
- **Условие:** Посылка должна быть в инвентаре игрока

---

### 1.5. Диалог с Ритой Морено

**Приветствие (первое взаимодействие):**
"О, новый клиент! Привет, чомбата. У меня есть все что нужно - оружие, броня, медикаменты. И я знаю, где найти что угодно."

**Опции диалога:**

1. **"Расскажи о Watson"**
   - **Ответ:** "Watson - это место где все возможно. Банды, торговцы, фиксеры - все здесь. Но будь осторожен - здесь не так безопасно как в Downtown. Знай, с кем имеешь дело."

2. **"Какие у тебя есть товары?"**
   - **Ответ:** "У меня есть базовое оружие, броня, медикаменты. Все качественное, проверенное. Если нужно что-то особенное - могу найти, но это будет дороже."
   - **Действие:** Показывается меню торговли

3. **"Передать посылку" (если квест активен и посылка в инвентаре)**
   - **Ответ:** "О, посылка от Сары Миллер! Спасибо, чомбата. Я давно ждала это. Вот твоя награда."
   - **Действие:** Посылка удаляется из инвентаря, квест завершается, выдаются награды

4. **"До свидания"**
   - **Ответ:** "Удачи, чомбата. Заходи еще."

---

### 1.6. Награды за квест

**Опыт:** 100 XP

**Деньги:** 200 eddies

**Предметы:** Нет

**Репутация:**
- Фракция: NCPD
- Количество: +5

**Дополнительные бонусы:**
- Разблокировка дополнительных квестов от Сары Миллер
- Разблокировка торговли с Ритой Морено (если еще не разблокирована)

---

### 1.7. Последовательность выполнения

**Шаг 1: Принятие квеста**
1. Игрок находит Сару Миллер в Downtown
2. Игрок начинает диалог
3. Игрок выбирает опцию "Какие задания у тебя есть?"
4. Игрок видит информацию о квесте "Доставка груза"
5. Игрок выбирает "Принять квест"
6. Квест добавляется в активные квесты
7. Посылка добавляется в инвентарь игрока

**Шаг 2: Перемещение в Watson**
1. Игрок открывает меню действий
2. Игрок выбирает "Переместиться"
3. Игрок видит список доступных локаций (Watson доступен)
4. Игрок выбирает Watson
5. Игрок подтверждает перемещение
6. Система показывает описание перемещения (5-10 минут игрового времени)
7. Игрок появляется в Watson

**Шаг 3: Поиск Риты Морено**
1. Игрок видит описание локации Watson
2. Игрок выбирает "Найти NPC" в меню действий
3. Игрок видит список NPC в Watson (Рита Морено)
4. Игрок выбирает Риту Морено
5. Игрок видит описание NPC

**Шаг 4: Передача посылки**
1. Игрок начинает диалог с Ритой Морено
2. Игрок видит опцию "Передать посылку" (если квест активен и посылка в инвентаре)
3. Игрок выбирает опцию "Передать посылку"
4. Система проверяет наличие посылки в инвентаре
5. Посылка удаляется из инвентаря
6. Квест завершается
7. Выдаются награды (опыт, деньги, репутация)
8. Показывается сообщение о завершении квеста

---

### 1.8. Обработка ошибок

**Ошибка: Посылка не в инвентаре**
- **Проблема:** Игрок пытается передать посылку, но её нет в инвентаре
- **Решение:** Показывать сообщение "У тебя нет посылки. Вернись к Саре Миллер, чтобы получить её."

**Ошибка: Квест не активен**
- **Проблема:** Игрок пытается передать посылку, но квест не активен
- **Решение:** Опция "Передать посылку" не показывается, если квест не активен

**Ошибка: NPC не найден**
- **Проблема:** Рита Морено не найдена в локации Watson
- **Решение:** Показывать сообщение "Рита Морено временно недоступна. Попробуй позже."

## 2. Альтернативные варианты первого квеста

### 2.1. Для разных классов

**Соло (Solo):**
- Квест может быть более боевым (например, "Устранить бандитов")
- Но для MVP остается "Доставка груза" для всех классов

**Нетраннер (Netrunner):**
- Квест может быть более хакерским (например, "Взломать систему")
- Но для MVP остается "Доставка груза" для всех классов

**Техник (Techie):**
- Квест может быть более техническим (например, "Починить устройство")
- Но для MVP остается "Доставка груза" для всех классов

**Примечание:** Для MVP все классы получают одинаковый первый квест. В будущем можно добавить уникальные квесты для каждого класса.

---

### 2.2. Для разных происхождений

**Street Kid:**
- Квест может быть более связан с бандами
- Но для MVP остается "Доставка груза" для всех происхождений

**Corpo:**
- Квест может быть более связан с корпорациями
- Но для MVP остается "Доставка груза" для всех происхождений

**Nomad:**
- Квест может быть более связан с перемещением
- Но для MVP остается "Доставка груза" для всех происхождений

**Примечание:** Для MVP все происхождения получают одинаковый первый квест. В будущем можно добавить уникальные квесты для каждого происхождения.
