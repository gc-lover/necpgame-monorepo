## Общее описание

Система уникальных стартов для всех комбинаций происхождений, фракций и классов персонажа. Каждая комбинация дает уникальный игровой опыт с разными стартовыми локациями, NPC, квестами и диалогами.

**Количество комбинаций:** 3 происхождения × 5 фракций × 3 класса = **45 уникальных стартов**

---

## 1. Компоненты системы стартов

### 1.1. Происхождения (3)

1. **Street Kid** - Уличный ребенок
2. **Corpo** - Корпоративный
3. **Nomad** - Номад

### 1.2. Фракции (5)

1. **Arasaka** - Корпорация
2. **Militech** - Корпорация
3. **Valentinos** - Банда
4. **Maelstrom** - Банда
5. **NCPD** - Организация

### 1.3. Классы (3)

1. **Solo** - Боец
2. **Netrunner** - Хакер
3. **Techie** - Техник

---

## 2. Структура уникального старта

### 2.1. Компоненты старта

Каждый уникальный старт включает:

**1. Стартовая локация:**
- Локация, в которой появляется персонаж
- Может отличаться в зависимости от комбинации

**2. Стартовый NPC:**
- NPC, который встречает персонажа при старте
- Дает первый квест
- Может быть разным для разных комбинаций

**3. Первый квест:**
- Уникальный первый квест для комбинации
- Адаптирован под происхождение, фракцию и класс

**4. Стартовое снаряжение:**
- Базовое снаряжение одинаковое
- Может быть дополнительное снаряжение в зависимости от комбинации

**5. Стартовая репутация:**
- Репутация с фракциями при старте
- Зависит от происхождения и выбранной фракции

**6. Вступительный диалог:**
- Уникальный вступительный диалог
- Адаптирован под комбинацию

## 3. Логика различий стартов

### 3.1. По происхождению

**Street Kid:**
- Стартовая локация: Watson (уличный район)
- Знакомство с бандами
- Первый NPC: Фиксер или член банды
- Бонус к репутации с бандами

**Corpo:**
- Стартовая локация: Downtown (корпоративный район)
- Знакомство с корпорациями
- Первый NPC: Корпоративный представитель
- Бонус к репутации с корпорациями

**Nomad:**
- Стартовая локация: Santo Domingo (рабочий район)
- Знакомство с независимыми
- Первый NPC: Независимый торговец или фиксер
- Бонус к репутации с независимыми

---

### 3.2. По фракции

**Arasaka:**
- Стартовая локация: Downtown (корпоративный район)
- Первый NPC: Представитель Arasaka
- Первый квест: Корпоративное задание
- Стартовая репутация с Arasaka: +10

**Militech:**
- Стартовая локация: Downtown или Watson
- Первый NPC: Представитель Militech
- Первый квест: Военное задание
- Стартовая репутация с Militech: +10

**Valentinos:**
- Стартовая локация: Watson или Heywood
- Первый NPC: Член банды Valentinos
- Первый квест: Бандитское задание
- Стартовая репутация с Valentinos: +10

**Maelstrom:**
- Стартовая локация: Watson (опасный район)
- Первый NPC: Член банды Maelstrom
- Первый квест: Опасное задание
- Стартовая репутация с Maelstrom: +10 (но фракция враждебна к новичкам)

**NCPD:**
- Стартовая локация: Downtown (полицейский участок)
- Первый NPC: Офицер NCPD
- Первый квест: Полицейское задание
- Стартовая репутация с NCPD: +10

---

### 3.3. По классу

**Solo:**
- Первый квест: Боевое задание
- Стартовое снаряжение: Оружие и броня
- Фокус на боевых навыках

**Netrunner:**
- Первый квест: Хакерское задание
- Стартовое снаряжение: Кибердека и базовое оружие
- Фокус на хакерских навыках

**Techie:**
- Первый квест: Техническое задание
- Стартовое снаряжение: Инструменты и базовое оружие
- Фокус на технических навыках

## 4. Примеры уникальных стартов

### 4.1. Street Kid + Arasaka + Solo

**ID:** `start-street-kid-arasaka-solo`

**Описание:**
Уличный ребенок, который попал в корпорацию Arasaka. Знает улицы, но теперь работает на корпорацию.

**Стартовая локация:** Watson (уличный квартал, но рядом с корпоративным офисом Arasaka)

**Первый NPC:** Марко "Фикс" Санчес (фиксер, связанный с Arasaka)

**Первый квест:** "Корпоративное задание для уличного бойца"

**Вступительный диалог:**
"Чомбата, я Марко. Слышал, что Arasaka взяла тебя на работу. Это редкость для уличного ребенка. Но ты знаешь улицы, а они нуждаются в ком-то, кто знает город. Есть задание для тебя."

---

### 4.2. Corpo + Valentinos + Netrunner

**ID:** `start-corpo-valentinos-netrunner`

**Описание:**
Бывший корпоративный работник, который присоединился к банде Valentinos. Знает технологии, но теперь работает для банды.

**Стартовая локация:** Watson (территория Valentinos)

**Первый NPC:** Хосе "Тигр" Рамирес (лидер банды Valentinos)

**Первый квест:** "Хакерское задание для банды"

**Вступительный диалог:**
"Привет, чомбата. Я Хосе, лидер Valentinos. Слышал, что ты был корпоративным работником, но теперь хочешь работать для нас. Хорошо. У нас есть задание, которое требует твоих хакерских навыков."

---

### 4.3. Nomad + NCPD + Techie

**ID:** `start-nomad-ncpd-techie`

**Описание:**
Номад, который решил работать на NCPD. Знает технику, но теперь работает для полиции.

**Стартовая локация:** Downtown (полицейский участок)

**Первый NPC:** Сара Миллер (офицер NCPD)

**Первый квест:** "Техническое задание для полиции"

**Вступительный диалог:**
"Привет. Я офицер Миллер. Слышал, что ты номад, который решил работать для NCPD. Это редкость. Но нам нужны технические специалисты, и ты идеально подходишь. Есть задание для тебя."
