## Описание

Сводка всего контента, созданного для MVP текстовой версии NECPGAME. Это обзорный документ, который показывает, какой контент доступен для первого запуска игры.

## 1. JSON данные (mvp-data-json/)

### 1.1. Локации (locations.json)
- **Количество:** 5 локаций
- **Город:** Night City
- **Районы:** Downtown, Watson, Westbrook, Santo Domingo, Heywood
- **Содержание:** ID, названия, описания, опасность, связи, типы

### 1.2. NPC (npcs.json)
- **Количество:** 10 NPC
- **Распределение:** 3 в Downtown, 3 в Watson, 2 в Santo Domingo, 1 в Westbrook, 1 в Heywood
- **Типы:** Торговцы (3), Квестодатели (3), Риппердок (1), Лидер банды (1), Враг (1), Элитный торговец (1)
- **Содержание:** ID, имена, описания, диалоги, приветствия

### 1.3. Предметы (items.json)
- **Количество:** 15 предметов
- **Категории:** Оружие (5), Броня (4), Медикаменты (2), Ресурсы (2), Квестовые предметы (2)
- **Содержание:** ID, названия, описания, характеристики, требования, цены

### 1.4. Квесты (quests.json)
- **Количество:** 5 квестов
- **Уровни:** 1, 1, 2, 3, 5
- **Типы:** Побочные квесты
- **Содержание:** ID, названия, описания, цели, награды, требования

## 2. Тексты и описания (mvp-content/)

### 2.1. Детальные тексты (detailed-texts.md)
- **Локации:** Детальные описания для всех 5 локаций
- **Диалоги:** Детальные диалоги для всех 10 NPC
- **Действия:** Описания действий в локациях
- **События:** Описания событий (перемещение, взаимодействие, бой)

### 2.2. Описания предметов (item-descriptions.md)
- **Оружие:** Детальные описания всех 5 видов оружия
- **Броня:** Детальные описания всех 4 видов брони
- **Медикаменты:** Детальные описания всех 2 видов медикаментов
- **Ресурсы:** Детальные описания всех 2 видов ресурсов
- **Характеристики:** Детальные характеристики для каждого предмета

### 2.3. Описания действий (actions-descriptions.json)
- **Осмотр:** Детальные описания для всех 5 локаций
- **Отдых:** Описание отдыха с восстановлением здоровья и энергии
- **Использование:** Описания использования объектов
- **Хакерство:** Описания хакерства (для будущего)
- **Перемещение:** Описания путей между локациями

### 2.4. Диалоговые деревья (dialogue-trees.json)
- **NPC:** Диалоговые деревья для 4 ключевых NPC
- **Ветвления:** Ветвления диалогов с вариантами ответов
- **Действия:** Действия в диалогах (торговля, квесты)

### 2.5. События (events.json)
- **Случайные события:** 5 случайных событий (патруль NCPD, конфликт банд, предложение фиксера, найденный предмет, корпоративная встреча)
- **События локаций:** События при исследовании локаций
- **Временные события:** События в зависимости от времени суток (день/ночь)

### 2.6. Детали локаций (location-details.json)
- **Точки интереса:** Детальные точки интереса для всех 5 локаций
- **Действия:** Действия для каждой точки интереса
- **Временные вариации:** Описания локаций в зависимости от времени суток

### 2.7. Нарративы боя (combat-narratives.json)
- **Начало боя:** Описания начала боя для разных типов врагов
- **Действия:** Описания действий в бою (атака, защита, использование предметов, побег)
- **Действия врага:** Описания действий врага
- **Конец боя:** Описания победы, поражения, побега

### 3.6. Бой

- **Действия:** Атака, защита, использование предметов, побег
