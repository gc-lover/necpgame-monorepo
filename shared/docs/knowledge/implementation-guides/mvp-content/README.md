# Контент для MVP

**Статус:** draft  
**Версия:** 1.0.0  
**Дата создания:** 2025-11-04  

---

## Описание

Контент для MVP текстовой версии NECPGAME. Включает тексты, диалоги, события, описания и другие элементы, которые будут видны игроку.

---

## Структура

### JSON данные
- `mvp-data-json/locations.json` - локации (5 районов Night City)
- `mvp-data-json/npcs.json` - NPC (10 персонажей с диалогами)
- `mvp-data-json/items.json` - предметы (15 предметов)
- `mvp-data-json/quests.json` - квесты (5 квестов)

### Тексты и описания
- `detailed-texts.md` - детальные тексты для локаций, диалогов и действий
- `item-descriptions.md` - детальные описания предметов
- `actions-descriptions.json` - описания действий в локациях
- `dialogue-trees.json` - диалоговые деревья для NPC
- `events.json` - случайные события и события в локациях

---

## Использование

Эти файлы можно использовать для:
1. Загрузки данных в БД через миграции
2. Отображения текстов в игровом интерфейсе
3. Генерации диалогов и событий
4. Тестирования игрового процесса
