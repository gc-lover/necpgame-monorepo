## 1. Оружие

### 1.1. Пистолет "Liberty"

**Краткое описание:**
```
Базовый пистолет для новичков. Надежный и простой в использовании.
```

**Детальное описание:**
```
Пистолет "Liberty" - это классический выбор для начинающих в Night City. 
Простой, надежный, без излишеств. Он не самый мощный, но достаточный для 
защиты на улицах. Производитель - Militech, гарантия качества. 
Подходит для ближнего и среднего боя. Хорошая скорострельность, но 
небольшой урон. Идеален для новичков, которые только начинают свой путь 
в Night City.
```

**Характеристики:**
- Урон: 20-30
- Скорострельность: Средняя
- Дальность: Ближняя-Средняя
- Надежность: Высокая
- Требования: Уровень 1, Ловкость 3
