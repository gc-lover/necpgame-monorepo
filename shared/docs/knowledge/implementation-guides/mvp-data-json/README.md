## Описание

JSON файлы с конкретными данными для MVP текстовой версии NECPGAME. Эти данные можно загрузить в БД для первого запуска игры.

**Файлы:**
- `locations.json` - локации (5 районов Night City)
- `npcs.json` - NPC (10 персонажей с диалогами)
- `items.json` - предметы (15 предметов)
- `quests.json` - квесты (5 квестов)
- `unique-starts.json` - уникальные старты для всех комбинаций (45 комбинаций, структура)
- `unique-starts-structure.md` - описание структуры данных для уникальных стартов

---

## Использование

Эти JSON файлы можно использовать для:
1. Загрузки данных в БД через миграции
2. Создания API endpoints для получения справочных данных
3. Тестирования игрового процесса
4. Демонстрации MVP

---

## Структура данных

Все файлы используют UUID или строковые идентификаторы для связи между данными.

**Связи:**
- NPC → Location (npc.locationId → location.id)
- Quest → NPC (quest.giverNpcId → npc.id)
- Quest → Location (quest.objectives[].target.id → location.id)
- Quest → Item (quest.objectives[].target.id → item.id)
