## Общее описание

План и требования для MVP (Minimum Viable Product) текстовой веб-версии NECPGAME. Определяет минимальный набор функций, необходимый для первого запуска игры в текстовом формате.

**Цель MVP:** Создать рабочую текстовую версию игры с минимальным функционалом для проверки основных механик и игрового процесса.

## 3. Структура MVP

### 3.1. Поток игры (User Flow)

```
1. Регистрация/Вход
   ↓
2. Выбор персонажа (или создание)
   ↓
3. Главный экран игры
   ↓
4. Просмотр локации
   ↓
5. Взаимодействие с NPC / Принятие квеста
   ↓
6. Выполнение квеста
   ↓
7. Завершение квеста / Получение награды
   ↓
8. Повтор шагов 4-7 или выход
```

---

### 3.2. Минимальный набор данных для MVP

**1. Локации:**
- Минимум 3-5 локаций в одном городе (например, Night City)
- Текстовые описания локаций
- Связи между локациями

**2. NPC:**
- Минимум 5-10 NPC
- Базовые диалоги
- Квестовые NPC

**3. Предметы:**
- Минимум 10-15 базовых предметов
- Оружие (2-3 типа)
- Броня (2-3 типа)
- Медикаменты (1-2 типа)
- Ресурсы (базовые)

**4. Квесты:**
- Минимум 3-5 простых квестов
- Квесты на доставку
- Квесты на поиск
- Квесты на убийство (если есть бой)

**5. Персонажи:**
- Базовые классы (минимум 3)
- Базовые характеристики
- Простая система прокачки

## 5. API Endpoints для MVP

### 5.1. Аутентификация

```
POST /api/v1/auth/register
POST /api/v1/auth/login
POST /api/v1/auth/logout
GET  /api/v1/auth/me
```

### 5.2. Персонажи

```
GET    /api/v1/characters
POST   /api/v1/characters
GET    /api/v1/characters/{id}
DELETE /api/v1/characters/{id}
GET    /api/v1/characters/{id}/stats
GET    /api/v1/characters/classes
GET    /api/v1/characters/origins
```

### 5.3. Локации

```
GET /api/v1/locations
GET /api/v1/locations/{id}
GET /api/v1/locations/current
POST /api/v1/locations/move
GET /api/v1/locations/{id}/npcs
GET /api/v1/locations/{id}/actions
```

### 5.4. Инвентарь

```
GET  /api/v1/inventory
GET  /api/v1/inventory/{itemId}
POST /api/v1/inventory/equip
POST /api/v1/inventory/unequip
POST /api/v1/inventory/use
```

### 5.5. Квесты

```
GET  /api/v1/quests
GET  /api/v1/quests/{id}
POST /api/v1/quests/{id}/accept
POST /api/v1/quests/{id}/complete
POST /api/v1/quests/{id}/abandon
```

### 5.6. NPC

```
GET /api/v1/npcs
GET /api/v1/npcs/{id}
GET /api/v1/npcs/{id}/dialogue
POST /api/v1/npcs/{id}/interact
```

### 5.7. Бой (опционально)

```
GET  /api/v1/combat/status
POST /api/v1/combat/start
POST /api/v1/combat/action
POST /api/v1/combat/end
```

### 5.8. Торговля (опционально)

```
GET  /api/v1/trade/shops
GET  /api/v1/trade/shops/{id}/items
POST /api/v1/trade/buy
POST /api/v1/trade/sell
```

## 10. Вопросы для проработки

### Q1: Минимальный набор функций

**Вопрос:** Что именно должно работать в MVP?

**Варианты:**

**A. Минимальный набор:**
- Только аутентификация, персонаж, локации, квесты
- Без боя, торговли, социальных функций
- Быстрее разработка, меньше функций

**B. Расширенный набор:**
- Минимальный набор + простой бой
- Минимальный набор + базовая торговля
- Больше функций, дольше разработка

**C. Гибридный подход:**
- Минимальный набор обязателен
- Бой и торговля опциональны, добавляются по готовности
- Гибкость в разработке

**D. Собственный вариант** (опишите)

---

### Q2: Уровень детализации

**Вопрос:** Насколько детализированными должны быть функции MVP?

**Варианты:**

**A. Минимальная детализация:**
- Базовые функции без сложных механик
- Простые квесты, простой бой
- Быстрее разработка

**B. Средняя детализация:**
- Базовые функции с некоторыми механиками
- Квесты с выбором, простой бой с тактикой
- Баланс между функциональностью и скоростью

**C. Полная детализация:**
- Все функции с полными механиками
- Сложные квесты, детальный бой
- Дольше разработка, но больше функциональности

**D. Собственный вариант** (опишите)

---

### Q3: Минимальный набор данных

**Вопрос:** Сколько данных нужно для MVP?

**Варианты:**

**A. Минимум данных:**
- 3 локации, 5 NPC, 10 предметов, 3 квеста
- Быстрое заполнение данных
- Меньше контента

- 5-7 локаций, 10-15 NPC, 20-30 предметов, 5-7 квестов
- Баланс между контентом и временем

- 10+ локаций, 20+ NPC, 50+ предметов, 10+ квестов
