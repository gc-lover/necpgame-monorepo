## Общее

Во время сетевых войн 2010–2011 NetWatch оставил в городской инфраструктуре множество следов. Фиксер просит извлечь даташард с сигнатурами из корпоративного узла.

- ID: `quest-class-netrunner-2011-netwatch-signal`
- Дал: `npc-marco-fix`
- Уровень: 2
- Локация: Downtown

---

## Цели

1. Принять задание у Марко
2. Добраться до корпоративного узла (Downtown)
3. Извлечь даташард (импровизация: электроника как носитель)
4. Вернуться к Марко
