## Общее

После разрушений 2023 начался ремонт городской инфраструктуры. Анна из Santo Domingo просит помочь восстановить узел энергосети.

- ID: `quest-class-techie-2025-repair-grid`
- Дал: `npc-anna-petrova`
- Уровень: 2
- Локация: Santo Domingo

---

## Цели

1. Принять задание у Анны Петровой
2. Переместиться в Santo Domingo
3. Собрать электронику (2 шт.)
4. Вернуться к Анне

- Деньги: 280 eddies
