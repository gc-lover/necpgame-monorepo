## 1. Синопсис

2035: переговоры о Хартии Свободного города (Найт-Сити). Офицер NCPD Сара Миллер просит обеспечить порядок и помочь с консультацией у корпоративного посредника Элизабет Чен. Игрок балансирует между безопасностью и корпоративными интересами.

- ID: `quest-main-2035-free-city-charter`
- Тип: main
- Дает: `npc-sarah-miller`
- Консультант: `npc-elizabeth-chen`
- Локация: Westbrook (`loc-westbrook-001`)
- Репутация: потенциальный прирост у NCPD

---

## 2. Цели и ветки

1) Принять задание у Сары → добраться до Westbrook → поговорить с Элизабет → вернуться к Саре.

Социальная проверка (COOL DC 14) при беседе с корп-сопровождающими:
- Успех: предотвращает потасовку (без боя), +репутация NCPD +8, XP без штрафов.
- Провал: начинается бой (3 противника), XP стандарт, репутация как у базы.

Альтернатива INT/TECH DC 13:
- Выявить скрытую камеру и саботаж микрофонов (снижение риска эскалации: если будет бой, один противник «ошеломлен» в 1-й раунд).

## 4. Бой (если провал соц-проверки)

- Инициатива стандартная.

- Награды боя стандартные (вне квестовых, через систему лута).

## 5. Награды

- 420 XP, 480 eddies.
- Репутация: `ncpd +8` (при успехе социалки), и `ncpd +5` (если был бой).

## 8. Тест-кейсы

1) Успех COOL: переговоры без боя, репутация +8, квест завершен.
2) Провал COOL, без INT/TECH: бой на 3 врага, квест завершен.
3) Провал COOL, успех INT/TECH: бой на 3 врага, один «ошеломлен» в 1-й раунд.
4) Возврат к Саре без выполнения промежуточных шагов — отказ завершения.
