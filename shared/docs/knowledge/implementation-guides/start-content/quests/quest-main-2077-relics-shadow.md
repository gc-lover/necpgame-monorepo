# [АРХИВ] 2077 — Тень Реликта (Relic's Shadow)

> ⚠️ **Shooter pivot:** содержит устаревшие проверки и параметры «кубиков». Используйте shooter-шаблон `QUEST-TEMPLATE-SHOOTER.md` и новый документ `combat-shooter-core.md` для актуальных заданий.

## 1. Синопсис
2077: в подполье ходят слухи о новой версии Relic и таинственных носителях. Виктор Вектор и Элизабет Чен просят выяснить правду: получить образцы телеметрии, перехватить корпоративные переговоры и понять, насколько технология готова к развёртыванию. Силовой путь возможен, но рискован — корпорации устраняют свидетелей.

- ID: `quest-main-2077-relics-shadow`
- Тип: `main`
- Дает: `npc-victor-vector`
- Уровень: 9
- Локации: Westbrook (`loc-westbrook-001`), Downtown (`loc-downtown-001`), Watson (`loc-watson-001`)

## 2. Цели и ветвление
Основной путь:
1) Встретиться с Виктором → получить модуль записи телеметрии.
2) Перехватить переговоры курьера в Westbrook (соц/скрытность/бой).
3) Снять телеметрию с подозреваемого узла в Downtown.
4) Доставить данные Элизабет Чен.

Ветвления:
- Социальная проверка (COOL 17): убедить курьера слить часть информации → минус 2 боя, +репутация.
- Скрытность (STEALTH 18): украсть чип у курьера незаметно → пропуск боя, +XP.
- Бой: 5 противников, шанс подкрепления при тревоге.

Бонусы по классам:
- `netrunner`: INT/NETRUNNING COMBAT — облегчённый перехват шифра.
- `solo`: REF/INTIMIDATION — запугать связного и разойтись без выстрелов.
- `techie`: TECH — ускоренная установка сенсоров и +компоненты.

- ## 3. Геймдизайн-детали (Shooter формат)
- Инициатива: `reaction_time = 0.62 × REF`, элита использует `combat_sync` (−0.04 сек).
- Попадание: shooter metrics против `defenseRating 215–225` (охранники), `defenseRating 235` (элита).
- Крит: headshot ×2.0; при `accuracy < 55%` возникает риск jam (потеря реакции).
- Состояния: «Подавление», «Кровотечение», «Ослепление» (гранаты, −25% visibility).
- Соц-проверки: shooter thresholds 0.60–0.68 с учетом COOL/INT/TECH/STEALTH/INTIMIDATION.
- Ситуационные модификаторы: ночь (+1 STEALTH), дождь (−1 дальний бой), толпа (+1 COOL).

## 6. Награды и баланс
- База: 720 XP, 950 eddies, `item-relic-data` ×1.
- Соц/скрытность успех: −враги, +50–60 XP.
- TECH/INT успех: +компоненты ×1, +50 XP.
- Поражение: перенос к началу ветки, штраф −10% денег.

## 10. Тест-кейсы
1) Успех COOL 17 → без боя.
2) Успех STEALTH 18 → пропуск боя.
3) Провал стелса → погоня ATHLETICS 15; ветка боя.
4) TECH 16 успех → +компоненты; повтор TECH 18 при провале.
5) INT 16 успех → +XP.
6) Поражение → респаун у Виктора, −10% денег.
