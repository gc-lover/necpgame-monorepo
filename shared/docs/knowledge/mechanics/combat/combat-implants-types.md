## 1. Типы имплантов для боя — ✅ Решение

### 1.1 Детализация системы имплантов — ✅ Решение
- **Базовый набор обязательных имплантов:**
  - Боевые: имплант точности, имплант урона, имплант скорострельности, имплант критического урона
  - Тактические: базовые киберглаза (сканирование), базовая кибердека
  - Защитные: киберкожа (броня), имплант восстановления
  - Двигательные: киберноги (скорость, прыжки)
  - OS: базовая кибердека (для всех классов)

- **Расширяемый набор:**
  - Авторские импланты для MMORPG механик (социальные, экономические)
  - Пользовательские импланты через систему брендов (`equipment-matrix.md`)
  - Уникальные импланты из лора Cyberpunk (Сандевистан, продвинутая кибердека и т.д.)

### 1.2 Применение имплантов в бою — ✅ Решение

**Влияние на статы:** Импланты напрямую изменяют характеристики (точность +10%, урон +5%, скорость +15%)

**Влияние на способности:** Импланты дают новые способности или модифицируют существующие (Q, E, R из VALORANT стиля)
- Боевые импланты: способности для стрельбы (автонаведение, взрывные пули, усиленный урон)
- Тактические импланты: хакерские способности через кибердеку, стелс-способности
- Защитные импланты: щиты, восстановление, иммунитеты
- Двигательные импланты: мобильные способности (рывок, двойной прыжок, паркур-усиления)
- OS: сигнатурные способности (Сандевистан = замедление времени, кибердека = хакерство)

**Синергии:**
- Синергии между имплантами (боевые + двигательные = мобильный боевой билд)
- Синергии с экипировкой (`equipment-matrix.md`): импланты одного бренда с оружием дают сет-бонусы
- Синергии с системой способностей: импланты усиливают способности из прокачки

### 1.3 Связь имплантов с другими системами — ✅ Решение

**Боевая система:**
- Влияние на стрельбу (`combat-shooting.md`): точность, урон, скорострельность
- Влияние на способности (`combat-abilities.md`): дают новые способности, модифицируют существующие
- Влияние на паркур (`combat-freerun.md`): киберноги дают паркур-усиления
- Влияние на стелс (`combat-stealth.md`): тактические импланты улучшают скрытность
- Влияние на хакерство (`combat-hacking-types.md`): кибердека для хакерских способностей

**Экономика:**
- Торговля имплантами (`economy-trading.md`): рынок игроков, аукционы
- Крафт имплантов (`economy-crafting.md`): создание имплантов из компонентов
- Ремонт и обслуживание: экономика содержания имплантов

**Прокачка:**
- Связь с классами (`progression/classes-overview.md`): разные классы имеют доступ к разным имплантам
- Усиление навыков: импланты усиливают навыки прокачки

**Социальные механики:**
- Влияние на репутацию: некоторые импланты требуют репутацию с фракциями
- Доступ к NPC: определенные импланты дают доступ к специфическим NPC/услугам
- Визуальная идентичность: импланты показывают принадлежность к фракциям/стилю

### 1.4 Примеры конкретных имплантов из лора Cyberpunk — ✅ Решение

**Ядро из лора Cyberpunk:**

**OS (Операционные системы):**
- Сандевистан (Sandevistan) - замедление времени, боевая OS
- Кибердека (Cyberdeck) - хакерские способности, хакерская OS
- Берсерк (Berserk) - усиление боевых характеристик, боевая OS
- Горилла Армс (Gorilla Arms) - усиление ближнего боя, боевой имплант

**Киберглаза (Optics):**
- Kiroshi Optics - улучшенное зрение, сканирование, обнаружение врагов
- Arasaka Optics - сетевые системы, интеграция с хакерством
- Militech Optics - боевое зрение, автонаведение

**Киберруки:**
- Gorilla Arms - ближний бой, разрушение укрытий
- Mantis Blades - клинки для ближнего боя
- Projectile Launch System - запуск снарядов
- Monowire - энергетическое оружие

**Киберноги:**
- Reinforced Tendons - двойной прыжок, паркур
- Fortified Ankles - увеличенная высота прыжка
- Lynx Paws - улучшенная скрытность, тихие шаги

**Нервная система:**
- Reflex Tuner - замедление времени при низком здоровье
- Kerenzikov - замедление времени при уклонении

**Интегральная система:**
- Subdermal Armor - броня под кожей
- Second Heart - восстановление здоровья при смерти
- Biomonitor - мониторинг состояния

**Другие:**
- Cyberdeck различных брендов (Arasaka, Zetatech, NetWatch и т.д.)
- Титан (Titanium Bones) - усиленный скелет
- Бионические легкие - улучшенная выносливость

**Авторские импланты для MMORPG:**
- Социальные импланты: улучшение репутации с фракциями, доступ к NPC
- Экономические импланты: улучшение торговли, крафта
- Коммуникационные импланты: расширенные социальные функции

**Пользовательские импланты:**
- Через систему брендов (`equipment-matrix.md`)
- Модерация и баланс через систему
