## 3. Визуальное проявление имплантов — ✅ Решение

### 3.1 Варианты визуализации имплантов — ✅ Решение

**Комбинация (выбор игрока + влияние типа импланта):**
- **Выбор игрока:** Игрок может выбирать видимость имплантов (полная видимость / частичная / скрыть)
- **Влияние типа импланта:**
  - Внешние импланты всегда видны (киберруки, киберглаза, киберноги) - физически заменяют части тела
  - Внутренние импланты можно скрыть (опционально) через косметические моды (скрывающая краска, одежда)
- **Визуальные эффекты:** Визуальные эффекты при использовании всегда видны (для обратной связи)
- **Интеграция:** Система кастомизации персонажей

### 3.2 Влияние имплантов на кастомизацию персонажа — ✅ Решение

**Полная интеграция с кастомизацией (визуальные изменения + косметика + стили):**
- **Визуальные изменения:** Импланты визуально изменяют персонажа (киберруки, киберглаза, киберноги меняют соответствующие части тела)
- **Косметические опции:**
  - Цвет кибервара (хром, золото, черный, цветные варианты)
  - Стиль (минимализм, грубость, эстетика, технологичный)
  - Световые эффекты (свечение, мерцание, подсветка)
- **Стили по брендам:**
  - Arasaka = минимализм, чистота линий, элегантность
  - Militech = грубость, надежность, военный стиль
  - Zetatech = эстетика, технологичность, инновации
  - Biotechnica = органичность, интеграция, естественность
- **Интеграция:** `equipment-matrix.md` (бренды и их визуальный стиль), система кастомизации персонажей

### 3.3 Видимость имплантов у других игроков (тактика) — ✅ Решение

**Комбинация (видимость зависит от типа + навыки обнаружения + опциональная скрытность):**
- **Внешние импланты:** Всегда видны у других игроков (киберруки, киберглаза, киберноги) - дают тактическую информацию о билде
- **Внутренние импланты:** Скрыты по умолчанию, но могут быть обнаружены через навыки/импланты сканирования
- **Опциональная скрытность:** Игрок может настроить видимость внешних имплантов через косметику (частично скрыть через одежду/краску)
- **Тактическая информация:** Видимые импланты дают информацию о билде противника (можно подготовиться к бою)
- **Обнаружение:** Навыки сканирования (`combat-stealth.md`) и импланты сканирования могут обнаруживать скрытые внутренние импланты
- **Интеграция:** `combat-stealth.md` (навыки обнаружения), PvP механики (`combat-pvp-pve.md`), социальные механики

### 3.4 Визуальные эффекты при использовании имплантов — ✅ Решение

**Полная система визуальных эффектов (активные + перегрев + энергия + состояния):**

**Активные способности:**
- Уникальные визуальные эффекты для каждого типа импланта
- Сандевистан = замедление времени, визуальный эффект движения
- Кибердека = визуализация хакерства, сетевые эффекты
- Боевые импланты = вспышки, эффекты урона
- Двигательные импланты = следы движения, эффекты скорости

**Перегрев:**
- Тепловое свечение (зависит от уровня перегрева: слабое → сильное)
- Пар, искажения воздуха
- Мерцание при критическом перегреве
- Интеграция с `combat-implants-limits.md` (система перегрева)

**Энергопотребление:**
- Свечение (зависит от энергопотребления)
- Мерцание при низкой энергии
- Световые эффекты при использовании способностей
- Интеграция с `combat-implants-limits.md` (энергетика)

**Состояния:**
- Повреждение: искры, сбои, мерцание
- Перегрузка: яркое свечение, нестабильность
- Киберпсихоз: визуальные эффекты стадий (искажения, мерцание, нестабильность)
- Интеграция с `combat-cyberpsychosis.md` (визуальные эффекты стадий)

**Тактическая информация:** Визуальные эффекты дают информацию о состоянии противника (перегрев, повреждение) - можно использовать в тактике
