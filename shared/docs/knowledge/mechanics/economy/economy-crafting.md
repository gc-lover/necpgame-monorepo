### 2. Как работает крафт?

**Варианты:**

**Вариант А: Простой крафт (MINECRAFT стиль)**
- Крафт через верстак с ингредиентами
- Рецепты известны всем
- *Плюсы:* Простота и доступность
- *Минусы:* Меньше глубины

**Вариант Б: Профессии (WOW стиль)**
- Профессии для крафта (оружейник, кузнец и т.д.)
- Нужно прокачивать профессии
- Рецепты получаются при прокачке
- *Плюсы:* Развитие персонажа
- *Минусы:* Может ограничивать игроков

**Вариант В: Сложное производство (EVE Online стиль)**
- Производство с компонентами и временем
- Производство на станциях/базах
- Сложные рецепты
- *Плюсы:* Глубина, реализм
- *Минусы:* Может быть сложно

**Вариант Г: Гибридная система (комбинация)**
- Базовый крафт доступен всем
- Продвинутый крафт требует профессий/навыков
- Сложное производство для редких предметов
- *Плюсы:* Максимальная гибкость
- *Минусы:* Сложнее реализовать

---

### 3. Источники рецептов

**Варианты:**

**Вариант А: Рецепты известны всем (стандартные)**

**Вариант Б: Рецепты через прокачку профессий**

- *Минусы:* Может ограничивать

**Вариант В: Рецепты как лут/находки**
- Рецепты можно найти/купить
- Редкие рецепты как ценность
- *Плюсы:* Глубина, ценность
- *Минусы:* Может быть сложно получить

## Решения

### 4.1 Типы крафта

**✅ Решение:** Все предложенные + типы крафта из лора Cyberpunk
- **Оружие и броня:** Традиционные боевые предметы
- **Импланты и модификации:** Кибервар, импланты для тела
- **Предметы и инструменты:** Утилиты и инструменты
- **Ресурсы и компоненты:** Материалы для производства
- **Типы крафта из лора Cyberpunk:**
  - **Кибервар (Cyberware):** Импланты, модификации тела, нейро-интерфейсы
  - **Хакерские программы (Cyberdecks):** Программы для хакерства, вирусы, защиты
  - **Модификации оружия:** Модификации для оружия (урон, точность, эффекты)
  - **Модификации имплантов:** Улучшение существующих имплантов
  - **Данные и информация:** Создание данных, программ, баз данных
  - **Электроника:** Создание устройств, чипов, процессоров
  - **Биомодификации:** Биологические модификации (из CYBERPUNK 2077)

### 4.2 Как работает крафт?

**✅ Решение:** Вариант Г - Гибридная система + механики из лора Cyberpunk
- Базовый крафт доступен всем (простые предметы)
- Продвинутый крафт требует профессий/навыков (оружие, импланты)
- Сложное производство для редких предметов (кибервар высокого класса)
- **Механики из лора Cyberpunk:**
  - Крафт требует специальных инструментов (верстаки, рабочие столы, кибердеки)
  - Крафт имплантов требует медицинских/хирургических навыков
  - Крафт хакерских программ требует навыков программирования/хакерства
  - Крафт может требовать доступа к специальным ресурсам/данным
  - Крафт некоторых предметов требует связей с фракциями/корпорациями
- Максимальная гибкость и реализм

### 4.3 Источники рецептов

**✅ Решение:** Вариант Г - Комбинация методов + источники из лора Cyberpunk
- Базовые рецепты известны всем (стандартные предметы)
- Продвинутые рецепты через прокачку профессий (развитие навыков)
- Редкие рецепты как лут/находки (ценные знания)
- **Источники из лора Cyberpunk:**
  - **Взлом сетей:** Рецепты можно получить через взлом корпоративных сетей
  - **Взлом роботов/устройств:** Данные о технологиях из взломанных устройств
  - **Взлом данных:** Рецепты в украденных/взломанных базах данных
  - **Корпоративные секреты:** Рецепты как часть корпоративных секретов
  - **Кража технологий:** Рецепты через кражу технологий у корпораций/фракций
  - **Информационные брокеры:** Покупка рецептов у информационных брокеров
  - **Взлом имплантов:** Данные о киберваре из взломанных имплантов
- Глубина и реализм в стиле киберпанк
