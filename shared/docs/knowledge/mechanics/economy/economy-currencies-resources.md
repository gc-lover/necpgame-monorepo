## Вопросы

### 1. Типы валют

**Варианты:**

**Вариант А: Одна валюта (простота)**
- Одна основная валюта для всего
- *Плюсы:* Простота и доступность
- *Минусы:* Меньше разнообразия

**Вариант Б: Несколько валют (разные типы)**
- Основная валюта (золото, евродоллары)
- Валюта PvP (бонусы за PvP)
- Валюта PvE (бонусы за рейды/контент)
- Валюта фракций (влияние на фракции)
- *Плюсы:* Глубина, разнообразие
- *Минусы:* Может быть сложно управлять

**Вариант В: Региональные валюты (разные валюты для регионов)**
- Разные валюты в разных регионах/странах
- Обменные курсы между валютами
- *Плюсы:* Реализм, возможности для торговли
- *Минусы:* Сложнее для игроков

### 3. Как добываются ресурсы?

**Варианты:**

**Вариант А: Добыча в экстракт-зонах (TARKOV стиль)**
- Выход в зоны для добычи ресурсов
- Риск потери при смерти
- *Плюсы:* Риск/награда, хардкорность
- *Минусы:* Может быть сложно

**Вариант Б: Добыча в открытом мире (WOW стиль)**
- Ресурсы добываются везде в открытом мире
- Нет риска потери
- *Плюсы:* Доступность для всех
- *Минусы:* Меньше напряжения

**Вариант В: Производство игроками (EVE Online стиль)**
- Игроки производят ресурсы через крафт/производство
- Добыча исходных материалов
- *Плюсы:* Глубина экономики
- *Минусы:* Может быть сложно

**Вариант Г: Комбинация всех способов**
- Добыча в экстракт-зонах (редкие ресурсы)
- Добыча в открытом мире (базовые ресурсы)
- Производство игроками (обработанные ресурсы)
- *Плюсы:* Максимальная гибкость
- *Минусы:* Сложнее реализовать

## Решения

### 2.1 Типы валют

**✅ Решение:** Вариант В - Региональные валюты + биржа валют
- Разные валюты в разных регионах/странах
- Обменные курсы между валютами
- Биржа валют для обмена между игроками
- Реализм и возможности для торговли валютой

### 2.2 Типы ресурсов

**✅ Решение:** Все предложенные + товары из лора Cyberpunk
- **Базовые ресурсы:** Руды, материалы
- **Редкие ресурсы:** Специальные материалы
- **Ресурсы для крафта:** Компоненты
- **Ресурсы для имплантов:** Кибервар материалы
- **Ресурсы для фракций:** Влияние на фракции
- **Товары из лора Cyberpunk:**
  - **Карты (Data Cards):** Данные, информация, данные для хакерства
  - **Информация (Info):** Секреты, контакты, данные о фракциях/корпорациях
  - **Кибердеки и данные:** Данные для нетраннеров, программы для хакерства
  - **Контракты и заказы:** Торгуемые контракты на задания
  - **Лицензии и права:** Торгуемые лицензии на технологии/права
  - **Персональные данные:** Данные игроков/NPC (в лоре - важный товар)
  - **Чипы и программы:** Софт, вирусы, защиты для кибердеков

### 2.3 Как добываются ресурсы?

**✅ Решение:** Вариант Г - Комбинация всех способов
- Добыча в экстракт-зонах (редкие ресурсы, высокий риск)
- Добыча в открытом мире (базовые ресурсы, низкий риск)
- Производство игроками (обработанные ресурсы, требует навыков)
- Торговля информацией и данными (киберпанк товары)
- Максимальная гибкость и разнообразие способов получения ресурсов
