## Концепция экономической системы

### Важность экономики

**Почему важна:**
- Глобальная экономика - один из основных столпов игры
- MMORPG требует глубокой экономической системы
- Экономика влияет на все аспекты игры (торговля, крафт, социальные структуры)
- Живой мир требует живую экономику (как в KENSHI и EVE Online)
- Экономические взаимодействия между игроками, фракциями, корпорациями

**Источники вдохновения:**
- **EVE Online** - глобальная экономика, политика, торговля, рынок игроков
- **RIMWORLD** - глобальная экономика, торговля, взаимодействия фракций
- **KENSHI** - живой мир, экономика развивается сама по себе
- **WOW** - система торговли и аукционов
- **MINECRAFT** - система крафта и торговли
- **CYBERPUNK 2077** - экономика в киберпанк сеттинге

## Компоненты экономической системы

### 1. Тип экономической системы
- **См. документ:** `economy-type.md`
- Кто управляет экономикой (NPC, игроки, комбинация)
- Работа рынков (глобальный, региональный, фракционный)
- Определение цен (фиксированные, динамические, гибрид)

### 2. Валюты и ресурсы
- **См. документ:** `economy-currencies-resources.md`
- Типы валют (региональные валюты + биржа валют)
- Типы ресурсов (базовые, редкие, для крафта, из лора Cyberpunk)
- Добыча ресурсов (экстракт-зоны, открытый мир, производство)

### 3. Торговля
- **См. документ:** `economy-trading.md` - общая концепция торговли
- **См. документ:** `economy-auction-house.md` - ⭐ детальные механики аукцион дома (MVP)
- **См. документ:** `economy-player-market.md` - ⭐ детальные механики рынка игроков с ордерами (MVP)
- Типы торговли (рынок игроков, аукцион дом, прямая торговля)
- Влияние социальных структур на торговлю
- Торговые маршруты (локальные, региональные, фракционные)

### 4. Крафт
- **См. документ:** `economy-crafting.md`
- Типы крафта (оружие, импланты, кибервар, данные)
- Механики крафта (базовый, продвинутый, сложное производство)
- Источники рецептов (базовые, прокачка, лут, взломы)

### 5. Влияние экономики на игровой мир
- **См. документ:** `economy-world-impact.md`
- Влияние на социальные структуры (фракции, корпорации, банды)
- Влияние на игровой мир (регионы, города, события)
- Живая экономика (игроки + события)

### 6. Монетизация и создание контента
- **См. документ:** `economy-monetization.md`
- Создание контента игроками (скины, программы, данные)
- Монетизация контента (игровая валюта + реальные деньги)
- Инструменты для создания контента (TODO)
