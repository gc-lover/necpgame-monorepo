## Контекст

**EVE Online торговля:**
- Торговля между игроками через рынок
- Торговые маршруты между регионами
- Влияние фракций на торговлю
- Спекуляция и торговые операции

**WOW торговля:**
- Аукцион дом для торговли
- Торговля между игроками напрямую
- NPC торговцы для базовых товаров

**RIMWORLD торговля:**
- Торговля с другими фракциями
- Влияние отношений на торговлю
- Динамические цены в зависимости от отношений

## Вопросы

### 1. Типы торговли

**Варианты:**

**Вариант А: Рынок игроков (EVE Online стиль)**
- Игроки создают ордера на покупку/продажу
- Рыночная система с поиском товаров
- *Плюсы:* Глубина, свобода
- *Минусы:* Может быть сложно для новичков

**Вариант Б: Аукцион дом (WOW стиль)**
- Аукцион дом для продажи товаров
- Простая система торговли
- *Плюсы:* Простота и доступность
- *Минусы:* Меньше гибкости

**Вариант В: Прямая торговля между игроками**
- Торговля напрямую между игроками
- Обмен товарами
- *Плюсы:* Непосредственное взаимодействие
- *Минусы:* Может быть неудобно

**Вариант Г: Комбинация всех типов**
- Рынок игроков для масштабной торговли
- Аукцион дом для быстрой торговли
- Прямая торговля для особых случаев
- *Плюсы:* Максимальная гибкость
- *Минусы:* Сложнее реализовать

---

- Доступ к фракционным рынкам по репутации
- Торговые бонусы для союзников
- Ограничения на торговлю с врагами

### 3. Торговые маршруты

**Варианты:**

**Вариант А: Нет торговых маршрутов (торговля только локально)**
- Торговля только в одном месте
- *Плюсы:* Простота
- *Минусы:* Меньше глубины

**Вариант Б: Торговые маршруты между регионами (EVE Online стиль)**
- Торговля между разными регионами
- Разные цены в разных регионах
- Возможность заработать на перевозке товаров
- *Плюсы:* Глубина, реализм
- *Минусы:* Может быть сложно

**Вариант В: Торговые маршруты через фракции (RIMWORLD стиль)**
- Торговля с другими фракциями
- Влияние отношений на торговлю
- *Плюсы:* Связь с социальными механиками
- *Минусы:* Зависимость от социальных механик

## Решения

### 3.1 Типы торговли

**✅ Решение:** Вариант Г - Комбинация всех типов
- Рынок игроков для масштабной торговли (EVE Online стиль)
- Аукцион дом для быстрой торговли (WOW стиль)
- Прямая торговля для особых случаев (личные сделки)
- Максимальная гибкость для разных ситуаций

### 3.2 Влияние социальных структур на торговлю

**✅ Решение:** Все предложенные механики
- Влияние репутации на цены (скидки/надбавки)
- Доступ к фракционным рынкам по репутации
- Торговые бонусы для союзников
- Ограничения на торговлю с врагами
- Торговые войны между фракциями
- Глубокая связь между социальными механиками и экономикой

### 3.3 Торговые маршруты

**✅ Решение:** Гибрид между всеми вариантами
- Локальная торговля (в одном месте) - для базовых товаров
- Торговые маршруты между регионами - для региональных товаров
- Торговые маршруты через фракции - для фракционных товаров
- Комбинация всех вариантов для максимальной гибкости
- Возможность заработать на перевозке товаров между регионами
