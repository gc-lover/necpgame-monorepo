## Система классов и подклассов

**Решение (2025-11-03):** Гибридная система классов
- **Подклассы для всех классов:** Каждый класс имеет 2-3 подкласса (специализации)
- **"Свой путь":** Игрок может выбрать вариант "Свой путь" и собрать свой класс свободно
- **Ограничения "Своего пути":** При выборе "Свой путь" недоступны уникальные классовые способности
- **Баланс:** Больше гибкости для "Своего пути", но уникальные способности только у классов

**Структура классов:**
- **10 основных классов** из лора Cyberpunk
- **3 авторских класса** (Политик, Торговец, Учитель)
- **Подклассы** для каждого класса (2-3 подкласса)
- **"Свой путь"** - возможность собрать класс свободно без уникальных способностей
- **Классовые способности** - уникальные для каждого класса

## Система классовых способностей

### Механика классовых способностей

**Решение (2025-11-03):** Комбинация активных и пассивных способностей
- **Активные способности:** Мощные действия, требующие активации (кулдауны/ресурсы)
- **Пассивные способности:** Постоянные бонусы к характеристикам/навыкам
- **Мощность способностей:** Достаточно сильные, чтобы соответствовать лору Cyberpunk
- **Соответствие лору:** Все способности основаны на лоре Cyberpunk 2020/Red

**Ограничения:**
- **"Свой путь"** не получает уникальные классовые способности (только общие навыки)
- **Классовые способности** доступны только при выборе конкретного класса
- **Подклассы** могут модифицировать классовые способности или добавлять свои
