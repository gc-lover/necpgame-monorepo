## Контекст и принципы

**Используемые классы:**
- **10 основных классов** из лора Cyberpunk (Solo, Netrunner, Fixer, Rockerboy, Media, Nomad, Corpo, Lawman, Medtech, Techie)
- **3 авторских класса** (Politician, Trader, Teacher)
- **Подклассы** для каждого класса (2-3 подкласса)
- **"Свой путь"** - возможность собрать класс свободно без уникальных способностей

**Ключевые решения:**
- Использовать все 10 классов из лора Cyberpunk
- Гибридная система классов с подклассами
- Система классовых способностей (активные + пассивные)
- Гибридная система связи с прокачкой (стартовые бонусы + доступ к веткам + общие навыки)
- 7 категорий общих навыков (35+ навыков)
