## 3. Активные способности

### 3.1. "Ускоренное обучение" (Active Ability)

**Описание:** Временно ускоряет прокачку навыков для ученика

**Механика:**
- Наставник активирует способность для ученика
- Ученик получает временный бонус к прокачке навыка (+50-100% опыта)
- Длительность: 15-30 минут
- Кулдаун: 1-2 часа

**Особенности:**
- Работает только если наставник и ученик находятся вместе
- Можно использовать на нескольких учениках одновременно (зависит от уровня)
- Эффект увеличивается с уровнем наставника

### 3.2. "Создание контента" (Active Ability)

**Описание:** Создание учебных материалов для обучения других игроков

**Механика:**
- Наставник создает учебные материалы (квесты, гайды, видео, интерактивные уроки)
- Другие игроки могут использовать учебный контент для обучения
- Наставник получает вознаграждение за использование контента
- Качество контента влияет на эффективность обучения

**Типы контента:**
- **Текстовые гайды** - подробные инструкции по навыкам
- **Видеоуроки** - демонстрация применения навыков
- **Интерактивные квесты** - практические задания для обучения
- **Учебные карты** - специальные зоны для обучения

**Особенности:**
- Контент может быть монетизирован (игровая валюта, реальные деньги)
- Контент может быть размещен в игровом магазине или системе обучения
- Качество контента влияет на рейтинг и популярность
- Контент может быть оценен другими игроками

### 3.3. "Мастер-класс" (Active Ability)

**Описание:** Массовое обучение группы игроков определенным навыкам

**Механика:**
- Наставник проводит мастер-класс для группы игроков
- Участники мастер-класса получают бонусы к прокачке навыка
- Мастер-класс требует подготовки и ресурсов
- Мастер-класс может быть публичным или приватным

**Особенности:**
- Максимальное количество участников зависит от уровня наставника
- Эффективность обучения снижается при большом количестве участников
- Мастер-класс может требовать специального места или оборудования
- Мастер-класс может быть записан и использован как учебный контент

**Механика:**
- Все навыки прокачиваются быстрее для Teacher класса (+10-20%)
- Особенно эффективно для навыков, связанных с обучением
- Эффект увеличивается с уровнем класса
