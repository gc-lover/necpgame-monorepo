## 1. Типы наставничества

### 1.1. Игрок (Teacher класс) → Другой игрок

**Концепция:** Игрок с классом Teacher может обучать других игроков

**Механика:**
- Игрок с классом Teacher может стать наставником для другого игрока
- Наставник обучает ученика навыкам и знаниям
- Ученик получает бонусы к прокачке навыков при обучении у наставника
- Наставник получает вознаграждение за обучение (опыт, репутация, ресурсы)

**Подклассы Teacher и их особенности:**

**Наставник (Mentor):**
- Фокус: Обучение новичков, помощь в освоении игры
- Особенности: Базовые навыки, обучение основам игры
- Бонусы: Максимальные бонусы к обучению новичков

**Профессор (Professor):**
- Фокус: Углубленное обучение, исследования, создание учебного контента
- Особенности: Продвинутые навыки, создание контента, исследования
- Бонусы: Создание учебных материалов, доступ к продвинутым ресурсам

**Тренер (Trainer):**
- Фокус: Обучение боевым/техническим навыкам, улучшение навыков других игроков
- Особенности: Боевые и технические навыки, физическая подготовка
- Бонусы: Максимальные бонусы к обучению боевым навыкам

**Условия для начала наставничества:**
- У наставника должен быть класс Teacher (любой подкласс)
- Ученик может быть любого класса
- Наставник должен иметь достаточный уровень навыка для обучения
- Ученик должен принять предложение наставничества

**Ограничения:**
- Один наставник может обучать ограниченное количество учеников одновременно
- Ученик может иметь только одного наставника одновременно (для конкретного навыка)
- Наставник должен иметь более высокий уровень навыка, чем ученик

### 1.2. Игрок (Teacher класс) → NPC

**Концепция:** Игрок с классом Teacher может обучать NPC

**Механика:**
- Игрок с классом Teacher может обучать NPC различным навыкам
- Обучение NPC улучшает отношения с NPC
- Обученный NPC может помогать игроку в операциях
- Обученный NPC может получать уникальные возможности

**Особенности:**
- Обучение NPC требует много времени и ресурсов
- Обученный NPC может стать более эффективным помощником
- Обучение NPC может открыть доступ к специальному контенту

### 1.3. NPC → Игрок

**Концепция:** NPC могут обучать игроков различным навыкам

**Механика:**
- NPC могут стать наставниками для игроков
- NPC обучают игроков навыкам в зависимости от своей специализации
- Обучение у NPC требует выполнения заданий и взаимодействия
- Обучение у NPC улучшает отношения с NPC

**Типы NPC-наставников:**
- **Боевые наставники** - обучение боевым навыкам (Solo, Lawman NPC)
- **Технические наставники** - обучение техническим навыкам (Netrunner, Techie NPC)
- **Социальные наставники** - обучение социальным навыкам (Fixer, Media NPC)
- **Классовые наставники** - обучение навыкам конкретного класса (NPC соответствующего класса)

**Условия для начала обучения у NPC:**
- Достаточная репутация с NPC
- Выполнение специальных квестов для доступа к обучению
- Доступность NPC для обучения (не все NPC могут обучать)
- Игрок должен соответствовать требованиям для обучения

**Особенности обучения у NPC:**
- Обучение у NPC обычно дешевле, чем у игроков
- NPC могут обучать только определенные навыки (в зависимости от специализации)
- Обучение у NPC может требовать выполнения специальных заданий
- Обучение у NPC может быть ограничено по времени или количеству учеников

### 1.4. NPC → NPC

**Концепция:** NPC могут обучать друг друга (опционально, для живого мира)

**Механика:**
- NPC могут обучать друг друга навыкам
- Это делает мир более живым и динамичным
- Обучение NPC друг друга может влиять на игровой мир
- Это может быть фоновой механикой для создания динамики мира
