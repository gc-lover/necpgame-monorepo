# Фреймворк мировых событий — Эпохи и механики (D&D/MMORPG)

**Статус:** draft  
**Версия:** 0.1.0  
**Дата:** 2025-11-05

## Цель

Унифицировать логику мировых событий в разных исторических периодах лора (эпохах) и их влияние на:
- D&D-проверки (DC, модификаторы, преимущества/помехи)
- Поведение фракций (AI-слайдеры агрессии/экономики/социального давления)
- Экономику (ресурсные множители, доступность рынков)
- Технологии (импланты, сеть, киберпространство) и доступные роли
- PvE/PvP интенсивность и типы событий (рейды, диверсии, дипломатия)

## Связанные правила
- Проверки D&D: `02-gameplay/combat/combat-dnd-core.md`
- Проверки в квестах/диалогах: `04-narrative/quest-dnd-checks.md`
- Интеграция с боем/имплантами: `02-gameplay/combat/combat-dnd-integration-shooter.md`
- Применение в механиках: `02-gameplay/combat/combat-dnd-mechanics-integration.md`

---

## Общая модель эпохи

Для каждой эпохи определяются:
- Сводка эпохи и доминирующие темы
- DC-скейлинг для категорий проверок (Соц / Тех/Хак / Исследование / Действия / Боевые микро-проверки)
- AI-слайдеры фракций (агрессия, экспансия, дипломатия, шпионаж)
- Экономические множители (ресурсный дефицит/избыток, волатильность цен)
- Технологические доступы (уровень имплантов, сложность сетей, наличие киберпространства)
- Генераторы событий (таблицы d100) и хуки для квестов

Шкала DC (ориентиры):
- Лёгкая: 10 | Средняя: 15 | Сложная: 20 | Очень сложная: 25 | Экстремальная: 30+

---

## AI-слайдеры фракций (0..5)
- Агрессия: 0 низкая → 5 тотальная война
- Экспансия: 0 стабильность → 5 активный захват
- Дипломатия: 0 отсутствует → 5 активные союзы
- Шпионаж/Саботаж: 0 редко → 5 постоянные операции

---

## Экономические множители (×)
- Доступность ресурсов: 0.5..1.5
- Волатильность цен: низкая/средняя/высокая
- Налоги/пошлины/корп-контроль: -2..+2 к Торговле

---

## Технологические уровни
- Импланты: базовые / развивающиеся / продвинутые / топовые
- Сеть/Киберпространство: локальная / региональная / глобальная / изолированная (после катастроф)
- NetWatch/безопасность: низкая/средняя/высокая
