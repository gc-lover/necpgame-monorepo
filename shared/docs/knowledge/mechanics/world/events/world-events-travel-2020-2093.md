## Формула вероятности события

```
P(event) = baseChance * (1 + levelModifier) * (1 + reputationModifier) * (1 + timeModifier)

baseChance = 0.10-0.25 (зависит от события)
levelModifier = (playerLevel - eventLevel) / 10 (max 0.5)
reputationModifier = reputationBonus / 100 (max 0.3)
timeModifier = timeSinceLastEvent / cooldown (max 0.2)
```
